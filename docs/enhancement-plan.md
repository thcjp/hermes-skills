## 增强流程

1. 选择高优先级free skill
2. 复制到 `differentiated-skills/` 目录
3. 增加专业功能（深度分析、高级模板、企业集成等）
4. 设置 `edition: paid` 和 `pricing_tier: L3-专业级` 或 `L4-企业级`
5. 更新数据库 `skill_type: paid`, `is_paid: 1`
6. 关联 `source_slug` 和 `free_slug`
7. 执行L1-L8质量审计
8. 上传到SkillHub (付费) 和 GitHub origin (私有备份)
