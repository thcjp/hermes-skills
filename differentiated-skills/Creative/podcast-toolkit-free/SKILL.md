# 播客创作工具包 - 免费版

## 概述

播客创作工具包(免费版)为个人创作者提供从选题到发布的单节目创作全流程指引。涵盖节目定位、选题规划、脚本撰写、录制检查与节目笔记生成,帮助独立创作者快速起步并保持稳定输出。

免费版聚焦单节目创作,专业版(`podcast-toolkit-pro`)在此基础上提供多节目管理、增长分析、变现策略与 AI 生成等高级能力。

## 使用场景

### 场景一:从零启动个人播客

规划播客定位与首期内容。

```bash
# 创建播客项目目录
mkdir -p ~/podcasts/my-show/{brand,episodes,guests.md,analytics.md}

# 生成首期大纲
cat > ~/podcasts/my-show/episodes/001/outline.md << 'EOF'
# 第 001 期:为什么我开始做播客

## 节目信息
- 形式:单人独白
- 时长:15-20 分钟
- 目标听众:对内容创作感兴趣的职场人

## 大纲
1. 开场钩子(0-30秒):分享一个让我决定做播客的瞬间
2. 自我介绍(30秒-2分钟):我是谁,为什么这个话题重要
3. 主体内容(2-15分钟):
   - 我观察到的三个现象
   - 每个现象背后的思考
   - 我打算如何用播客探索
4. 结尾(15-20分钟):下期预告,呼吁订阅

## 转场设计
- 现象一 -> 现象二:"但这还不是最让我意外的..."
- 现象二 -> 现象三:"如果说前两个是表象,那第三个就是本质..."
EOF
```

### 场景二:采访类节目准备

为采访嘉宾准备问题清单。

```bash
# 嘉宾信息与问题准备
cat > ~/podcasts/my-show/guests.md << 'EOF'
# 嘉宾管理

## 嘉宾:张三(产品经理)
- 背景:5 年 toB 产品经验,现任某 SaaS 公司产品总监
- 话题:toB 产品的增长策略
- 联系方式:微信 zhangsan_pm

## 采访问题清单
1. 开场:简单介绍你目前负责的产品
2. 核心:toB 产品增长与 toC 有什么本质区别?
3. 深挖:分享一个你做过的最艰难的产品决策
4. 实操:你日常如何收集用户反馈?
5. 展望:未来 3 年 toB 产品趋势
6. 结尾:推荐一本对你影响最大的书

## 备注
- 嘉宾偏好深度对话,避免快问快答
- 录制前 15 分钟闲聊热身
EOF
```

```bash
## 时间轴
- 00:00 开场:决定做播客的瞬间
```

## 快速开始

### 1. 初始化播客项目

```bash
# 标准目录结构
mkdir -p ~/podcasts/<show-name>/
cd ~/podcasts/<show-name>/

mkdir -p brand/          # 封面、片头片尾、音乐
mkdir -p episodes/       # 每集一个子目录
mkdir -p episodes/001/   # 首期
touch guests.md          # 嘉宾管理
touch analytics.md       # 数据追踪
```

### 2. 确定节目定位

回答三个核心问题:
- **形式**:单人独白 / 双人对谈 / 嘉宾访谈 / 叙事类
- **垂直领域**:不是"商业",而是"独立 SaaS 创业者"
- **更新频率**:周更 / 双周更 / 季更(一致性 > 频率)

### 3. 创作首期内容

```bash
# 进入首期目录
cd episodes/001/

# 创建大纲
touch outline.md

# 录制后创建脚本
touch transcript.md

# 生成节目笔记
touch show-notes.md
```


## 最佳实践

1. **前 30 秒决定留存**
   - 开场直接抛出价值主张
   - 预告本期最精彩的内容片段
   - 避免冗长的自我介绍

2. **一致性胜过完美**
   - 按时发布比内容完美更重要
   - 逐步迭代改进,不要追求一步到位
   - 建立 3-5 期的库存缓冲

3. **切片是增长引擎**
   - 每期至少提取 3-5 个社媒切片
   - 切片聚焦最有价值的 30-60 秒
   - 配合文字摘要发布

4. **垂直领域优先**
   - 占领小众受众比追逐大众更有效
   - 明确的目标听众带来更高粘性
   - 口碑传播在垂直领域更高效

5. **项目结构规范**
   - 统一目录结构便于长期管理
   - 每集独立目录,避免文件混乱
   - 维护嘉宾与数据文件,持续积累

## 常见问题

### Q1: 播客形式该如何选择?

- **单人独白**:适合观点输出,门槛低,适合起步
- **双人对谈**:需要搭档,互动感强
- **嘉宾访谈**:需要人脉资源,内容深度高
- **叙事类**:制作成本高,故事性强

建议从单人独白起步,积累经验后扩展。

### Q2: 更新频率如何确定?

- 周更:适合已建立节奏的创作者
- 双周更:平衡质量与频率的推荐选择
- 季更:适合深度叙事类,每季 8-12 集

关键是**一致性**:选一个能长期坚持的频率。

### Q3: 免费版与专业版的区别?

免费版聚焦单节目创作全流程;专业版支持多节目管理、增长分析、变现策略与 AI 辅助生成。当你的播客进入增长期或需要商业化时,建议升级。

### Q4: 是否需要视频播客?

视频是可选但有力的补充。YouTube 播客搜索增长迅速,视频版本可扩大触达。建议音频优先,条件允许时增加视频。

### Q5: 节目笔记必须包含时间戳吗?

强烈建议。时间戳帮助听众快速定位感兴趣的内容,提升用户体验,也有利于 SEO。免费版提供时间戳模板。
