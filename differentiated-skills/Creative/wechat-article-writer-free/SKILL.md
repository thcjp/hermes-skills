## 使用场景

### 场景一：从话题生成文章

用户给出一个话题，自动生成完整的公众号文章。

```text
用户指令：帮我写一篇关于"时间管理技巧"的公众号文章，面向职场新人，2000字左右。

生成流程：
1. 分析话题与目标读者
2. 选择合适的文章结构（方法论型）
3. 生成标题："职场新人必看：5个时间管理技巧让你效率翻倍"
4. 生成摘要与正文
5. 输出完整的 draft.md
```

```python
# 示例
writing_config = {
    "topic": "时间管理技巧",
    "target_reader": "职场新人",
    "tone": "亲切专业",
    "writing_style": "方法论",
    "target_word_count": 2000,
    "default_structure": ["观点型"],
}

# Agent根据配置生成文章初稿
# 输出文件：draft.md
```

## 最佳实践

1. **明确读者画像**：写作前明确目标读者，内容深度与用词贴合读者水平
2. **选择合适结构**：根据内容类型选择结构模板，观点文用观点型，教程用方法论型
3. **先提纲后正文**：提供提纲比仅给话题能获得更精准的文章结构
4. **分步优化**：先生成初稿，再进行润色改写，逐步优化质量

6. **保持一致调性**：同一账号系列文章保持统一的调性与风格

## 常见问题

### Q1：生成的文章字数不够怎么办？

在配置中增加 `target_word_count`，或在指令中明确要求字数。也可使用续写功能扩展内容。

### Q2：文章风格不符合预期怎么办？

调整 `tone` 和 `writing_style` 配置。提供更具体的调性描述，如"幽默但不浮夸"比单纯的"幽默"更精准。

### Q3：免费版支持哪些文章类型？

免费版支持观点文、教程文、盘点文、故事型四种基础结构模板，覆盖公众号常见内容类型。

### Q4：如何进行文章改写？

直接告诉Agent需要改写的部分和改写方向，如"把第三段改写得更生动一些"或"续写一个案例分析"。

### Q5：免费版能否直接发布到公众号？

免费版仅生成文章草稿（draft.md），不包含发布功能。如需直接发布到公众号草稿箱，建议升级至PRO版本。
