# 创意总监免费版

## 概述

创意总监免费版是一款面向个人用户的轻量级创意辅助工具。扮演资深创意总监角色，提供创意咨询、头脑风暴、方案评估、创意简报生成等核心能力。帮助个人创作者、自由职业者、学生等快速获取专业创意建议，激发创意灵感.
## 核心能力

| 能力 | 说明 |
|---|---|
| 创意咨询 | 回答创意相关问题，提供专业建议与方向 |
| 头脑风暴 | 根据主题生成多个差异化创意方向 |
| 方案评估 | 对创意方案进行多维度评分与优化建议 |
| 创意简报 | 快速生成项目创意简报文档 |
| 品牌建议 | 提供品牌调性与视觉方向基础建议 |
**技术实现要点**：核心能力基于`input_params`参数与`output_format`配置实现,支持创建/查询/修改/删除等操作模式,通过`config_options`进行运行时配置.
### 核心功能执行
用`input_params`参数进行配置.
**输入**: 用户提供核心功能执行所需的指令和必要参数.
**处理**: 解析核心功能执行的输入参数,完成核心逻辑,返回结构化响应.
**输出**: 返回核心功能执行的响应数据,包含状态码、结果和日志.
- 执行此能力时使用`input_params`参数,支持创建/查询/导出操作

### 参数配置与调用
用`config_options`参数进行配置.
**输入**: 用户提供参数配置与调用所需的指令和必要参数.
**处理**: 解析参数配置与调用的输入参数,完成核心逻辑,返回结构化响应.
**输出**: 返回参数配置与调用的响应数据,包含状态码、结果和日志.
- 执行此能力时使用`config_options`参数,支持修改/重置/导入操作

### 结果处理与输出
用`output_format`参数进行配置.
**输入**: 用户提供结果处理与输出所需的指令和必要参数.

**输出**: 返回结果处理与输出的响应数据,包含状态码、结果和日志.

## 使用场景

### 场景一：创意头脑风暴

用户需要为一个项目构思创意方向，通过头脑风暴获取灵感.
```text
用户指令：我需要为一个咖啡品牌做推广，请帮我头脑风暴5个创意方向.
# ...
创意总监输出：
方向1：【日常仪式感】
- 核心理念：将喝咖啡变成每日仪式
- 执行方向：系列短视频展示不同人群的咖啡时刻
- 视觉风格：温暖、慢节奏、生活化
# ...
方向2：【咖啡地图】
- 核心理念：城市咖啡文化探索
- 执行方向：互动H5地图+线下打卡活动
- 视觉风格：手绘插画、城市探索感
# ...
...（共5个方向）
```

### 场景二：创意方案评估

用户已有创意方案，需要专业评估与优化建议.
```text
用户指令：请评估以下创意方案：
"通过短视频挑战赛推广运动饮料，用户拍摄运动后喝饮料的视频"
# ...
创意总监评估：
【整体评分】72/100
# ...
【各维度评分】
- 创意性：70分（挑战赛形式较常见，缺乏独特记忆点）
- 可执行性：85分（短视频挑战赛执行难度低）
- 传播性：75分（有分享动机，但需强化奖励机制）
- 品牌关联度：65分（运动场景与产品关联明确）
# ...
【优化建议】
1. 增加独特记忆点：设计专属手势或口号
2. 强化品牌关联：在挑战中融入产品特色卖点
3. 提升传播动力：设置阶梯式奖励机制
```

### 场景三：创意简报生成

快速为项目生成结构化的创意简报.
```text
用户指令：帮我生成一个新品上市的创意简报.
项目：智能水杯上市
目标：年轻上班族
# ...
创意总监输出创意简报：
【项目背景】
智能水杯新品上市，目标用户为25-35岁年轻上班族...
# ...
【创意目标】
1. 建立品牌认知度
2. 传达"智能提醒喝水"的核心卖点
3. 引发社交媒体讨论
# ...
【目标受众】
25-35岁都市白领，关注健康但工作忙碌...
# ...
【创意方向】
方向A：健康焦虑解药
方向B：办公室养生神器
# ...
【调性关键词】
科技感、健康、轻松、时尚
# ...
【交付物】
社交媒体内容3条、短视频1支
```

## 最佳实践

1. **需求明确**：提供清晰的项目背景与目标，创意方向更精准
2. **多方向探索**：不要满足于第一个方向，多探索几个不同角度
3. **逐步深入**：先获取多个方向，再选择最有潜力的深入展开
4. **评估优化**：对初步方案进行评估，根据反馈迭代优化
5. **受众导向**：始终从目标受众角度思考创意的吸引力
6. **预算匹配**：创意方向应与预算范围匹配，避免不可执行的方案

## 常见问题

### Q1：头脑风暴的方向不够创新怎么办？

尝试提供更多背景信息，如竞品分析、目标受众痛点等。也可以指定创意方向类型（如"请从互动驱动角度构思"）.
### Q2：方案评估的评分标准是什么？

免费版使用创意性、可执行性、传播性、品牌关联度四个维度，各维度权重不同，综合得出总分.
### Q3：免费版能做完整的品牌策划吗？

免费版提供品牌基础建议，但不包含完整品牌体系设计。如需完整品牌策划，建议升级至PRO版本.
### Q4：创意简报可以直接用于提案吗？

免费版生成的创意简报可作为提案基础框架，建议根据实际项目需求进一步细化.
### Q5：可以处理哪些行业的创意需求？

免费版适用于大多数行业的创意咨询，包括快消、科技、教育、餐饮等常见领域.
