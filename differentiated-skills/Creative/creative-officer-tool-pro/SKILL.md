# 创意总监专业版

## 概述

创意总监专业版是企业级创意管理平台，在免费版基础上提供完整品牌体系构建、营销战役全流程规划、创意资产管理、团队协作工作流等专业能力。适用于企业品牌部、广告代理商、品牌升级项目等高阶创意管理场景.
### 免费版与专业版对比

| 能力 | 免费版 | 专业版 |
|---|---|---|
| 创意咨询 | 基础咨询 | 深度咨询+策略建议 |
| 头脑风暴 | 5个方向 | 10个方向+差异化分析 |
| 方案评估 | 4维度评分 | 8维度+ROI预测+风险 |
| 创意简报 | 基础简报 | 完整战役简报+执行计划 |
| 品牌体系 | 基础建议 | 完整品牌指南 |
| 营销战役 | 不支持 | 全流程规划 |
| 创意资产 | 不支持 | 资产库+版本管理 |
| 团队协作 | 不支持 | 审批+反馈+追踪 |
| 竞品情报 | 不支持 | 创意策略分析 |
| 趋势洞察 | 不支持 | 行业趋势追踪 |
| 跨渠道适配 | 不支持 | 一核多渠输出 |

## 使用场景

### 场景一：品牌升级全案

企业进行品牌升级，需要完整的品牌体系构建.
```text
用户指令：我们公司要从传统形象升级为科技感品牌，请提供完整的品牌升级方案.
# ...
创意总监输出：
【品牌升级策略】
升级方向：传统可靠 → 科技创新
核心挑战：保持客户信任感的同时注入科技活力
# ...
【品牌定位更新】
- 品牌使命：用科技赋能传统行业
- 品牌个性：创新、专业、前瞻
- 品牌承诺：技术驱动，价值可信
# ...
【视觉系统升级】
- 标志：保留核心识别，简化线条，增加动态效果
- 色彩：从沉稳蓝调升级为科技蓝+活力橙
- 字体：从衬线体切换为无衬线几何体
- 图形：引入数据可视化风格辅助图形
# ...
【语调系统更新】
- 声音特征：自信、前瞻、平易近人
- 语言风格：专业但不晦涩，创新但不浮夸
- 禁用词：过时、传统、老牌（强调新锐感）
# ...
【迁移计划】
阶段1：核心视觉系统更新（1个月）
阶段2：数字渠道全面迁移（2个月）
阶段3：线下物料逐步替换（3个月）
```

## 最佳实践

1. **体系化思维**：品牌建设应系统化规划，各模块保持一致性
2. **战役节奏**：营销战役遵循"预热→引爆→持续→长尾"节奏
3. **数据驱动**：创意评估加入ROI预测，用数据支撑决策
4. **差异化定位**：通过竞品分析找到市场空白，建立差异化优势
5. **跨渠道一致**：一个创意核心多渠道适配，保持品牌一致性
6. **版本管理**：创意方案迭代时做好版本追踪，便于回溯
7. **协作效率**：建立清晰的审批工作流，减少沟通成本

## 常见问题

### Q1：品牌体系构建需要多长时间？

完整品牌体系构建通常需要4-8周，取决于品牌复杂度与项目范围.
### Q2：营销战役方案可以直接执行吗？

专业版输出完整的战役框架与执行计划，但具体创意制作需要创意团队执行.
### Q3：竞品分析的情报来源是什么？

竞品分析基于公开可获取的营销案例与市场信息，非内部机密数据.
### Q4：团队协作功能如何使用？

通过创意审批工作流管理方案版本，支持多角色协作与反馈循环.
### Q5：与免费版的评估模型兼容吗？

兼容。专业版在免费版4维度基础上扩展为8维度，免费版的评估结果可作为专业版的输入.

### 可用性分类

- **分类**: MD（纯Markdown指令，核心功能无需exec命令行执行能力）
- **说明**: 基于Markdown的AI Skill，通过自然语言指令驱动Agent扮演企业级创意总监角色，提供品牌体系构建、营销战役规划、创意资产管理等专业服务。与免费版完全兼容，可直接复用免费版的创意咨询、头脑风暴与方案评估结果.
## 错误处理

| 错误场景 | 原因 | 处理方式 |
|:------:|--------|:-------|
| 配置错误 | 参数缺失或格式错误 | 检查依赖说明中的配置要求 |
| 运行时错误 | 运行环境不满足 | 确认运行环境符合依赖说明 |
| 网络错误 | 连接超时或不可达 | 执行ping命令测试网络连通性,检查防火墙和代理设置连接后执行ping命令测试网络连通性,检查防火墙和代理设置连接后重新执行命令，参考国内替代方案 |

## 已知限制

- 需LLM支持,无LLM环境不可用
- 复杂业务场景建议结合人工经验判断
- 执行效率受模型能力与网络环境影响
