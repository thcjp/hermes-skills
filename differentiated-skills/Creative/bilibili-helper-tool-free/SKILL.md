# B站创作助手免费版

## 概述

B站创作助手免费版是一款面向个人UP主的轻量级内容创作辅助工具。通过自然语言指令，快速生成符合B站平台调性的视频标题、简介与标签推荐。输出纯文本内容，可直接复制使用到B站投稿页面。

## 核心能力

| 能力 | 说明 |
| --- | --- |
| 标题生成 | 根据主题生成5个B站风格爆款标题 |
| 简介模板 | 自动生成视频简介结构与内容 |
| 标签推荐 | 根据视频主题推荐合适的标签组合 |
| 平台适配 | 内容风格符合B站社区调性 |
| 纯文本输出 | 所有输出为纯文本，直接可用 |

## 使用场景

### 场景一：生成视频标题

UP主准备投稿视频，需要吸引人的标题。

```bash
# 生成5个B站爆款标题
bili.sh title "Python入门教程"

# 输出示例：
# 1. 【零基础】Python入门到精通，一集就够了！
# 2. 学了3年Python才知道，入门应该这样学...
# 3. Python新手必看：30分钟掌握核心语法
# 4. 别再瞎学了！Python入门最正确的打开方式
# 5. 【保姆级教程】Python零基础入门指南
```

### 场景二：生成视频简介与标签

为视频生成完整的简介内容和配套标签。

```bash
# 生成视频简介+标签
bili.sh desc "Python入门教程"

# 输出示例：
# 【简介】
# 本视频是Python零基础入门教程，从安装环境开始，
# 带你逐步掌握Python核心语法。适合没有任何编程基础的同学。
#
# 【标签】
# Python, 编程入门, 零基础, 教程, 编程
```

### 场景三：标签推荐

根据视频主题获取标签推荐。

```bash
# 标签推荐
bili.sh tags "游戏实况"

# 输出示例：
# 推荐标签：游戏实况, 单机游戏, 游戏解说, 攻略, 通关
# 热门标签：游戏, 游戏区, 实况, 高能, 翻车
```

## 配置示例

### 标题生成策略

免费版标题生成遵循以下策略：

| 策略 | 说明 | 示例 |
| --- | --- | --- |
| 悬念型 | 制造好奇心 | "学了3年才知道..." |
| 数字型 | 具体数字吸引 | "30分钟掌握..." |
| 痛点型 | 直击用户痛点 | "别再瞎学了！" |
| 保姆型 | 强调详细易懂 | "【保姆级教程】" |
| 对比型 | 突出优势对比 | "从零到精通" |

### 标签推荐规则

| 标签类型 | 说明 | 数量建议 |
| --- | --- | --- |
| 核心标签 | 直接描述内容 | 2-3个 |
| 分类标签 | B站分区标签 | 1-2个 |
| 热门标签 | 当前热门话题 | 1-2个 |
| 长尾标签 | 精准搜索词 | 2-3个 |

## 最佳实践

1. **主题明确**：提供清晰的视频主题，标题更精准
2. **多选策略**：从5个标题中选择最合适的，或组合使用
3. **标签组合**：核心标签+热门标签+长尾标签的组合效果最佳
4. **简介结构**：简介应包含内容概述、适合人群、更新计划
5. **平台调性**：标题风格应符合B站社区文化，避免过度标题党
6. **字数控制**：标题建议20字以内，简介建议100-200字

## 常见问题

### Q1：生成的标题不够吸引人怎么办？

尝试提供更具体的主题描述，如"Python面向对象编程"比单纯的"Python"效果更好。

### Q2：标签推荐不够精准怎么办？

在主题中加入更多关键词，如"Minecraft生存实况"比"游戏"能获得更精准的标签。

### Q3：免费版能生成口播脚本吗？

免费版不包含口播脚本生成功能。如需此功能，请升级至PRO版本。

### Q4：输出内容可以直接用于B站吗？

是的，所有输出为纯文本格式，可直接复制到B站投稿页面使用。

### Q5：免费版有使用次数限制吗？

免费版无固定次数限制，但建议合理使用，避免短时间内大量请求。
