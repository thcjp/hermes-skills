# 文本人性化工具（免费版）

## 概述

本工具识别并去除 AI 生成文本的 24 类典型痕迹（夸大意义、推销腔、AI 高频词、三连排比、破折号滥用等），在保留原意与目标语气的前提下注入个人语气与节奏变化。适合单段/单篇文本润色.

## 使用场景

### 场景一：去除夸大意义与推销腔

```text

  确保用户高效达成目标。它不仅是一次更新，更是生产力革命.

改写后:
  软件更新增加了批量处理、快捷键和离线模式。早期测试者反馈不错，
  多数人表示任务完成更快了.
```

```text
          交织、关键、景观、关键、展示、挂毯、见证、强调、
```

## 最佳实践

- **只删不够**：删掉 AI 套话只是一半，还得注入语气.
- **节奏要变**：短句与长句交替，别每句一样长.
- **允许复杂情绪**：真实人类有矛盾感受，「既惊艳又不安」胜过「很惊艳」.
- **具体胜过聪明**：用具体细节替代模糊赞美.
- **朗读校验**：读出声别扭的地方就是还要改的地方.

## 改写质量自检

- **朗读不别扭**：读出声顺滑，无拗口套话.
- **像人说话**：有口语、有犹豫、有情绪.
- **信息没丢**：核心事实与目标语气保留.
- **无三连排比**：拆掉「A、B、C」式排比.
- **节奏有变化**：短长句交替，不是每句等长.
## 常见 AI 套话替换表

| AI 套话 | 自然替换 |
|:----:|:----:|
| 此外 | 另外 / 还 |
| 深入探讨 | 聊聊 / 看看 |
| 至关重要 | 很关键 / 离不开 |
| 值得注意的是 | 顺便说 / 注意 |
| 在当今社会 | 现在 / 如今 |
| 综上所述 | 所以 / 总之 |
