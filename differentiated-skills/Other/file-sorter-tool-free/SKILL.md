# 视觉文件整理工具（免费版）

## 概述

视觉文件整理工具免费版作为本地数字管家，自动清理用户杂乱的下载文件夹或桌面。完全模拟人类「双击打开 → 查阅内容 → 总结命名 → 拖拽归档」的物理工作流，利用视觉模型识别文件内容并智能重命名归档.
## 核心能力

| 能力 | 说明 |
|---|---|
| 文件遍历 | 自动遍历目标文件夹，过滤系统隐藏文件与临时文件 |
| 视觉识别 | 打开文件并截屏识别内容，提取关键元数据 |
| 智能重命名 | 按规则生成新文件名（财务/工作/图片/未分类） |
| 分类归档 | 移动至对应子文件夹（财务账单/工作文档/图片素材/其他） |
| 安全红线 | 禁止删除、扩展名保护、隐私隔离 |

## 使用场景

### 场景一：下载文件夹定期清理

用户希望整理杂乱的下载文件夹.
```bash
# 整理下载文件夹
file-sorter organize --target ~/Downloads
# ...
# 示例
# 📂 扫描: ~/Downloads
# 📊 发现 28 个待处理文件
# 🔍 视觉识别中...
#
# ✅ 已整理 25 个文件
#   - 财务账单: 8 个
#   - 工作文档: 12 个
#   - 图片素材: 5 个
#
# ⚠️ 未分类: 3 个（密码保护或无法识别）
```

### 场景二：桌面文件归档

用户希望将桌面文件按内容归档.
```bash
# 整理桌面
file-sorter organize --target ~/Desktop --destination ~/Documents/Sorted/
# ...
# 重命名规则示例：
# 发票.pdf → [2026-07]_发票_某某科技.pdf
# 合同.docx → [2026-07-15]_租房合同.docx
# IMG_001.jpg → [海边日落风景].jpg
```

### 场景三：发票与收据的归类

用户希望将发票与收据统一归类至财务目录.
```bash
# 仅整理财务类文件
file-sorter organize --target ~/Downloads --filter "financial" --destination ~/Documents/Sorted/财务账单/
# ...
# 输出
# 📊 财务文件整理报告
# 已识别: 12 张发票/收据
# 已重命名并归档: 12 个
# 按月份分组: 2026-05 (3张), 2026-06 (5张), 2026-07 (4张)
```

## 安全红线

1. **禁止删除**：仅拥有重命名和移动权限，绝对不允许执行 `rm` 或移入回收站.
2. **扩展名保护**：严禁修改文件后缀名，即使认为是伪装文件.
3. **隐私隔离**：视觉识别依赖本地或默认视觉模型，不将截图发送至未授权的第三方 API.
## 最佳实践

* 整理前先用 `--dry-run` 预览，确认重命名与归档方案.
* 财务类文件按月份分组，便于报销与对账.
* 工作文档按日期命名，便于时间线追溯.
* 无法识别的文件保留原名并加「未分类_」前缀.
* 定期整理（建议每周一次），避免文件堆积.
* 视觉识别依赖本地模型，确保隐私不外泄.

## 常见问题

**Q：免费版支持批量处理多少文件？**
A：免费版建议单次不超过 50 个文件。如需大批量处理与自定义规则，请考虑 PRO 版本.
**Q：免费版支持自定义分类规则吗？**
A：免费版使用固定四分类（财务/工作/图片/未分类）。如需自定义分类与规则，请使用 PRO 版本.
**Q：视觉识别会泄露文件内容吗？**
A：不会。免费版依赖本地或默认视觉模型，不将截图发送至未授权的第三方 API.
**Q：可以恢复原始文件名吗？**
A：免费版不提供操作历史与回滚。建议整理前用 `--dry-run` 预览。如需历史与回滚，请使用 PRO 版本.
**Q：支持哪些文件类型？**
A：支持图片、PDF、Office 文档等可通过视觉识别的文件。加密文件会标记为「未分类_」.
