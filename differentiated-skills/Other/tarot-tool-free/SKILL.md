# 塔罗反思工具（免费版）

## 概述

塔罗在这里是一面镜子，而非预言。本工具用于意义构建、情绪锚定与温和的自我反思，遵循「在场优先、非临床、非预测」的核心立场。免费版面向个人用户，提供大阿尔卡那牌的单张与三张牌阵抽牌能力，帮助你用象征性语言与自己的处境对话。

## 核心能力

| 能力 | 说明 |
|:-----|:-----|
| 单张牌阵 | 快速锚定当前情绪或处境，输出关键词与反思语 |
| 三张牌阵 | 情境 / 张力 / 下一步 三段式结构化反思 |
| 语言校准 | 内置温和措辞库，避免「你必须」「一定会」等绝对化表述 |
| 正逆位机制 | 约 35% 概率出现逆位，丰富解读维度 |
| 安全边界 | 识别自我伤害信号时自动切换至安全优先模式 |

## 使用场景

### 场景一：早晨的快速锚定

用户希望以一个象征开始新的一天。

```text
用户：帮我抽一张牌，作为今天的锚定。
助手：
  牌面：VII 战车（正位）
  关键词：动力、意志、方向、勇气
  反思：也许今天不需要同时打开太多扇门。选定一个方向，带着善意前进，节奏由你自己掌控。
  邀请提问：今天，哪一件小事最值得你投入注意力？
```

### 场景二：面临选择时的结构化反思

用户在两个选项之间犹豫，希望借助三张牌阵梳理。

```text
用户：我在考虑是否换工作，帮我抽三张牌。
助手：
  【情境】II 女祭司（正位）
  关键词：直觉、内在知晓、沉默、神秘
  反思：也许你已经感知到一些尚未说出口的答案。

  【张力】XII 倒吊人（正位）
  关键词：暂停、新视角、放下、耐心
  反思：这段悬停并非浪费时间，换一个角度看，信息会更清晰。

  【下一步】XVII 星星（正位）
  关键词：希望、更新、安静的信念、指引
  反思：无论选择哪条路，先保留一丝对自己温柔的期待。

  整合：三个画面共同指向——先倾听内在，再让暂停成为澄清，最后以希望收尾。
  邀请提问：如果暂停三天再做决定，你最担心错过什么？
```

### 场景三：情绪低落时的陪伴对话

用户感到疲惫，希望获得一段不被评判的对话。

```text
用户：今天很累，想抽张牌看看。
助手：
  牌面：IX 隐士（正位）
  关键词：独处、反思、内在指引、智慧
  反思：独处可以是疗愈，也可能是封闭——关键在于你是否在独处中听见自己。
  邀请提问：此刻，你是想被理解，还是想被安静地陪伴？
```

## 最佳实践

* 将塔罗视为对话的起点，而非决策的终点。
* 抽牌后先停留在感受上，再进入分析。
* 如果牌面让你不舒服，可以要求重新抽或换一个牌阵。
* 避免在一天内反复抽同一问题，给象征一些沉淀时间。

## 常见问题

**Q：免费版支持小阿尔卡那牌吗？**
A：免费版仅包含 22 张大阿尔卡那牌，聚焦核心象征。如需完整 78 张牌阵，请考虑 PRO 版本。

**Q：解读结果会保存吗？**
A：免费版不提供会话持久化，解读仅存在于当前对话中。如需历史记录归档，请使用 PRO 版本。

**Q：可以在多人场景下使用吗？**
A：免费版面向个人用户，不建议用于公开占卜或商业场景。

**Q：如果抽到「死神」或「塔」这类牌怎么办？**
A：这些牌在象征层面代表转变与揭示，并非灾祸。免费版会以温和的方式呈现其成长含义。
