# 图表生成器(免费版)

一个面向命令行场景的轻量级数据可视化工具，通过单一脚本即可生成十余种图表形态。无需安装Python科学计算栈或前端图表库，开箱即用。

## 概述

本工具通过`scripts/chart.sh`脚本接收用户输入的数据，输出可在终端直接显示的ASCII图表，或可保存为文件供后续分享的HTML/SVG格式。设计目标：

- **零依赖**：仅依赖Bash环境，不安装任何第三方运行时
- **双输出**：终端ASCII图表方便命令行查看，HTML文件方便分享嵌入
- **多形态**：覆盖柱状、折线、饼图、表格、迷你趋势、热力图、进度条、SVG矢量等十余种类型
- **即开即用**：一条命令直接出图，无需配置文件、无需初始化

## 使用场景

### 场景1：日报中展示部门预算对比

运维同学需要在每日站会前快速对比四个部门的预算消耗，使用终端柱状图即可在不打开浏览器的情况下完成展示：

```bash
chart.sh bar "销售:85,市场:62,研发:93,运维:41" --title "部门预算(万)"
```

### 场景2：监控数据的迷你趋势图

将过去8小时的CPU负载序列压缩为一行sparkline，嵌入到监控脚本输出中：

```bash
chart.sh sparkline "3,7,2,8,5,9,1,6"
```

输出：`▃▆▁█▄█▁▅`，一眼即可看出峰值在第4个采样点。

### 场景3：周报中嵌入SVG折线图

将本月每日活跃用户数据生成SVG文件，嵌入到周报文档中：

```bash
chart.sh svg-line "月度活跃" "1日:1200,8日:1850,15日:2120,22日:2480,29日:2630"
```

### 场景4：快速对比三组数据的占比

季度市场份额对比，使用饼图直观呈现：

```bash
chart.sh pie "苹果:35,三星:25,华为:20,其他:20" --title "Q3市场份额"
```

## 最佳实践

1. **标签长度统一**：保持柱状图标签字符数一致，对齐效果更好
2. **数值范围归一**：超过100的数值会自动按比例缩放，建议在标题中标注单位
3. **sparkline采样点控制**：8-12个采样点视觉效果最佳，过多会拥挤
4. **SVG输出路径用绝对路径**：在自动化脚本中调用时，使用绝对路径避免相对路径歧义
5. **表格数据校验**：使用管道符分隔行时，确保每行列数一致
6. **饼图数值总和**：饼图数值会自动归一化，但建议直接输入百分比便于校对

## 常见问题

### Q1：执行脚本报"command not found"？

A：脚本路径未在PATH中，请使用绝对路径或相对路径调用，例如`./scripts/chart.sh bar ...`。

### Q2：中文标签显示错位？

A：终端字体未启用等宽显示，建议使用等宽字体如"DejaVu Sans Mono"或"Source Code Pro"。

### Q3：HTML文件打开后样式错乱？

A：浏览器缓存了旧版本，强制刷新（Ctrl+Shift+R）即可。

### Q4：能否在Windows的PowerShell中直接运行？

A：需要安装Git Bash或WSL环境，原生PowerShell不支持bash脚本语法。

### Q5：如何把多张图表合并到一个HTML文件？

A：使用`chart.sh dashboard "标题"`生成仪表盘骨架，再分别生成子图表插入。
