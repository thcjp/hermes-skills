# 生活事件追踪 - 免费版

## 概述

生活事件追踪工具免费版帮助个人用户记录与回顾生活中的重要事件。提供三种记录模式,从简单的要点记录到详细的故事叙述,灵活适配不同场景。所有数据存储在本地,保护隐私.
## 核心能力

### 1. 三种记录模式

- 简单模式:一句话记录事件
- 详细模式:结构化记录(时间/地点/人物/感受)
- 故事模式:自由叙述完整故事

**输入**: 用户提供三种记录模式所需的指令和必要参数.
**处理**: 解析三种记录模式的输入参数,完成核心逻辑,返回结构化响应.
**输出**: 返回三种记录模式的响应数据,包含状态码、结果和日志.

## 输入格式
| 参数名 | 类型 | 必填 | 说明 |
|---|---|---|---|
| input | string | 是 | 生活事件追踪-免费版处理的输入数据或指令 |
| options | object | 否 | 附加配置选项,如模式选择、格式偏好等 |
| callback_url | string | 否 | 异步处理完成后的回调通知URL |

```bash
# 记录事件(详细模式)
python3 tardis.py add \
  --mode detailed \
  --category "工作" \
  --title "项目上线" \
  --date "2025-01-15" \
  --location "公司" \
  --people "团队全员" \
  --feeling "兴奋且疲惫" \
  --details "经过3个月开发,核心项目终于上线。团队通宵部署,早上8点正式发布。"
# ...
# 输出:
# 事件已记录
# ID: evt-2025-0015
# 标题: 项目上线
# 日期: 2025-01-15
# 类别: 工作
```

### 场景二:简单模式快速记录

用简单模式快速记录日常事件.
```bash
# 快速记录
python3 tardis.py add \
  --mode simple \
  --category "生活" \
  --title "学会了做红烧肉" \
  --tags "烹饪,新技能"
# ...
# 批量快速记录
python3 tardis.py batch-add \
  --file events.txt \
  --mode simple
# ...
# events.txt 格式:
# 2025-01-15|生活|学会了做红烧肉|烹饪,新技能
# 2025-01-16|健康|跑步5公里|运动
# 2025-01-17|学习|读完《深入理解计算机系统》|阅读
```

### 场景三:时间线回顾

按时间线回顾过去的事件.

## 最佳实践

1. **及时记录**:事件发生后尽快记录,避免遗忘细节
2. **简单模式优先**:日常用简单模式快速记录,重要事件再用详细模式
3. **善用标签**:标签是检索的关键,养成添加标签的习惯
4. **定期回顾**:每周/每月回顾时间线,总结与反思
5. **数据备份:定期导出数据,防止数据丢失
6. **隐私保护:本地存储确保隐私,不要将敏感记录同步到云端

## 常见问题

### Q: 数据存储在哪里?

A: 免费版数据存储在本地文件系统(默认 `~/tardis-data/`)。使用 JSON 格式,便于备份与迁移。不会上传到任何云端.
### Q: 可以在手机上使用吗?

A: 免费版是命令行工具,需要通过终端使用。可以通过 SSH 连接电脑使用,或使用 PRO 版的 Web 界面在手机浏览器访问.
### Q: 数据多了会变慢吗?

A: JSON 格式在数千条记录时仍能快速搜索。超过 1 万条建议按年分文件存储。免费版支持按年归档旧数据.
### Q: 可以导入其他日记应用的数据吗?

A: 免费版支持导入 CSV 格式数据。将其他应用的数据导出为 CSV,使用 `tardis.py import --format csv --file data.csv` 导入.
