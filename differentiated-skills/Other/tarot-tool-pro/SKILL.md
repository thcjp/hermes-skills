# 塔罗反思工具（专业版）

## 概述

专业版在免费版的「在场优先、非临床、非预测」立场之上，扩展为面向专业占卜师与内容创作者的完整塔罗解读平台。覆盖完整 78 张韦特塔罗牌，内置 8 种牌阵模板，支持会话归档、批量解读与自定义牌意词典，同时与免费版的输出格式保持向后兼容。

## 核心能力

| 能力 | 免费版 | 专业版 |
|:-----|:-------|:-------|
| 牌组范围 | 22 张大阿尔卡那 | 完整 78 张（大阿尔卡那 + 小阿尔卡那四花色） |
| 牌阵模板 | 单张、三张 | 单张、三张、凯尔特十字、关系阵、年运阵、抉择阵、月相阵、自定义 |
| 正逆位 | 约 35% 概率 | 可配置概率（0%-100%） |
| 会话归档 | 不支持 | 本地 JSON 归档 + 全文检索 |
| 批量解读 | 不支持 | 批量主题解读与导出 |
| 自定义词典 | 不支持 | 自定义牌意覆盖默认解读 |
| 多用户档案 | 不支持 | 按客户隔离的解读历史 |
| 输出格式 | 纯文本 | 纯文本 + JSON + Markdown 报告 |

## 使用场景

### 场景一：专业占卜师的客户服务

为固定客户提供结构化解读并归档。

```bash
# 为客户「林小姐」执行凯尔特十字牌阵并归档
tarot-pro draw --spread celtic-cross --client "林小姐" --archive

# 示例
# 牌阵: 凯尔特十字（10张）
# 客户: 林小姐
# 归档ID: 2026-07-18-lin-001
```

```text
解读报告（节选）：
1. 现状 — VIII 力量（正位）
   关键词：温柔、韧性、耐心、慈悲
   解读：你正在用柔软的方式承载一段需要耐心的关系。

2. 挑战 — XV 恶魔（逆位）
   关键词：释放、觉察、夺回力量
   解读：某段依附正在被你看见，觉察本身就在松动链条。

...（共10个位置）
```

### 场景二：内容创作者的塔罗栏目批量生产

为月度塔罗栏目一次性生成 12 个星座的解读草稿。

```bash
# 批量为 12 星座生成月度运势解读
tarot-pro batch --template monthly-horoscope --count 12 --format markdown --output ./monthly-report/

# 批量解读会为每个星座抽取一张牌并生成 200-300 字解读
# 输出目录：./monthly-report/2026-07/
```

### 场景三：团队工作坊的多人牌阵协作

在工作坊中为多位参与者同时抽牌并生成汇总报告。

```bash
# 为工作坊的 6 位参与者抽取「月相阵」并生成汇总
tarot-pro workshop --spread moon-phases --participants 6 --report summary

# 输出：每位参与者的牌阵 + 团队整体能量图谱
```

  default_format: markdown

  isolation: true

## 常见问题

**Q：专业版与免费版的输出格式兼容吗？**
A：兼容。专业版的纯文本输出与免费版格式一致，免费版用户可无缝升级。专业版额外支持 JSON 与 Markdown 报告格式。

**Q：会话归档数据存储在哪里？**
A：所有归档数据存储在本地 `~/tarot-pro/archives` 目录，不上传至任何第三方服务器。可通过配置文件自定义路径。

**Q：批量解读有数量上限吗？**
A：单次批量建议不超过 50 条以保证质量。如需更大批量，建议分批执行并启用并发控制。

**Q：自定义牌意词典如何编写？**
A：编辑 `~/tarot-pro/dictionary.yaml`，按牌名与正逆位覆盖关键词与反思语即可。未覆盖的牌面会回退到默认解读。

**Q：专业版支持多语言解读吗？**
A：默认中文输出，可通过配置 `language: ja` 或 `language: en` 切换。小阿尔卡那牌意已内置中日英三语对照。
