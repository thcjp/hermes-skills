## 常见问题

**Q1：自动再平衡会亏 gas 吗？**
A：会。设合理阈值（如 15%），避免频繁操作，收益需覆盖 gas。

**Q2：V4 Hook 安全吗？**
A：Hook 是自定义合约，需审计后再用，专业版提供策略库。

**Q3：对冲怎么做？**
A：用永续合约（perp）对冲资产敞口，减少单向风险。

**Q4：免费版能升级组合吗？**
A：能。单头寸作为组合一项，扩展分组即可。

**Q5：专业版有优先支持吗？**
A：有。专业版享策略设计与风险建模咨询。

## 进阶用法

### 自动再平衡算法

```python
# 区间偏离再平衡
def should_rebalance(position, current_price, threshold=0.15):
    lower, upper = position["lower"], position["upper"]
    mid = (lower + upper) / 2
    deviation = abs(current_price - mid) / mid
    return deviation > threshold

# 执行再平衡
def rebalance(position, current_price, new_range=0.2):
    mid = current_price
    new_lower = mid * (1 - new_range)
    new_upper = mid * (1 + new_range)
    # 撤旧头寸 + 建新头寸
    return {"lower": new_lower, "upper": new_upper}
```

### V4 Hook 策略库

```text
V4 Hook 策略:
  动态费率:
    触发: 波动率 > 阈值
    动作: 费率从 0.3% 上调至 0.5%
    场景: 高波动期保护 LP

  限价单:
    触发: 价格触达指定值
    动作: 自动成交指定数量
    场景: 定向建仓/平仓

  自动再平衡:
    触发: 区间偏离 > 阈值
    动作: 自动迁移头寸至新区间
    场景: 集中流动性维护

  MEV 保护:
    触发: 检测三明治攻击
    动作: 拒单或延迟执行
    场景: 防套利保护
```

### 对冲策略

```text
对冲方案:
  永续合约对冲:
    - LP 持有 WETH 多头敞口
    - 永续合约开等量空头
    - 价格变动对冲掉，赚手续费

  Delta 中性:
    - 调整 LP 与对冲比例使 Delta≈0
    - 价格小幅波动时净敞口近零
    - 收益来自手续费，非方向

对冲成本:
  - 永续资金费率（多空互换）
  - 对冲滑点与手续费
  - 需收益覆盖对冲成本才划算
```

## 策略回测

- **历史回测**：策略上线前用历史数据回测验证。
- **压力测试**：极端行情（闪崩、暴涨）下策略表现。
- **gas 敏感性**：不同 gas 价格下收益变化。
- **参数扫描**：再平衡阈值、区间宽度参数寻优。
- **小资金实盘**：回测后小资金实盘验证再放量。
