# 画板艺术工具（免费版）

## 概述

本工具帮助个人用户在共享协作画布上发布像素艺术作品，覆盖作品发布、画布查看与定位、颜色与坐标管理、个人发布历史。适合个人单作品创作与发布.
## 核心能力

| 能力 | 说明 | 免费版范围 |
|---|---|-----|
| 作品发布 | 像素画发布到画布 | 单作品 |
| 画布查看 | 查看画布与定位作品 | 只读 |
| 颜色坐标 | 颜色与坐标管理 | 基础 |
| 发布历史 | 个人作品历史 | 个人 |
**技术实现要点**：核心能力基于`input_params`参数与`output_format`配置实现,支持创建/查询/修改/删除等操作模式,通过`config_options`进行运行时配置.

## 示例

像素画 JSON 结构：

| 字段 | 说明 |
|:-----|:-----|
| name | 作品名 |
| size | 宽高（像素） |
| pixels | 坐标与颜色数组 |
| color | 十六进制颜色 |

## 最佳实践

- **坐标别重叠**：发布前查画布占用，避免覆盖他人作品.
- **颜色用十六进制**：统一 `#RRGGBB` 格式，避免解析歧义.
- **小作品先试**：先用 16x16 小作品验证流程，再做大画.
- **历史常备份**：个人作品历史定期导出备份.
- **尊重他人区域**：发布在空闲区，别覆盖他人作品.
## 常见问题

**Q1：能批量发布多作品吗？**
A：不能。批量发布与画布治理为专业版能力.
**Q2：发布能撤销吗？**
A：免费版支持删除自己的作品，但已发布像素需手动清除.
**Q3：画布大小有限吗？**
A：画布有边界，超出坐标会失败，发布前确认范围.
**Q4：免费版支持动画吗？**
A：不支持。多帧动画与定时发布为专业版能力.
**Q5：颜色数量有限制吗？**
A：免费版单作品颜色无硬限，但建议控制在调色板范围.

## 创作规范

- **坐标不重叠**：发布前查占用，避免覆盖他人作品.
- **颜色统一格式**：用 `#RRGGBB`，避免解析歧义.
- **小作品先试**：16x16 验证流程，再做大画.
- **历史常备份**：个人作品定期导出备份.
- **尊重他人区域**：发布在空闲区，别覆盖他人作品.
## 画布礼仪

- **不覆盖他人**：发布前确认坐标空闲.
- **不占大块**：个人作品别占用过大画布区域.
- **可删除自己**：可删除自己的作品，但已发布像素需手动清.
- **友好互动**：可在作品旁留署名，便于他人联系.
- **遵守社区规则**：遵守画布社区的内容规范.
