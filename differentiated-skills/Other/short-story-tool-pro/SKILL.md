# 短篇网文写作 - 专业版

## 概述

短篇网文写作工具专业版是企业级短篇创作平台,在免费版创作能力之上扩展批量产出、题材模板库、投稿平台管理与读者数据分析。适合内容工作室批量产出、专业作者高效创作与投稿策略优化.
专业版兼容免费版创作流程,支持平滑升级.
## 核心能力

### 1. 批量产出

一次生成多篇短篇,支持排期发布,满足高频内容需求.
**输入**: 用户提供批量产出所需的指令和必要参数.
**处理**: 解析批量产出的输入参数,完成核心逻辑,返回结构化响应.
**输出**: 返回批量产出的响应数据,包含状态码、结果和日志.
- 执行此能力时使用`input_params`参数,支持创建/查询/导出操作
### 2. 题材模板库

内置 50+ 题材模板,覆盖都市/悬疑/科幻/言情/奇幻等类型,快速启动创作.

## 使用场景

### 场景一:批量产出月度内容

为内容平台批量产出一个月的短篇内容.
用户可通过自然语言指令触发此场景，工具将自动执行相应操作并返回结构化结果.
## 输入格式
| 参数名 | 类型 | 必填 | 说明 |
|---|---|---|---|
| input | string | 是 | 短篇网文写作-专业版处理的输入数据或指令 |
| options | object | 否 | 附加配置选项,如模式选择、格式偏好等 |
| callback_url | string | 否 | 异步处理完成后的回调通知URL |

```bash
# 批量生成 20 篇短篇
./short-pro batch-create \
  --count 20 \
  --themes "都市情感,职场故事,悬疑推理,科幻想象" \
  --words 3000-5000 \
  --style-variation true \
  --output "monthly/"
# ...
# 输出:
# === 批量产出报告 ===
# 生成: 20 篇
# 总字数: 82,340 字
# 平均字数: 4,117 字

# #02 《代码之死》 职场故事 3,890字 原创
# #03 《第七封信》 悬疑推理 4,560字 原创
# ...
#
# 查重检测: 全部通过 (原创度 > 95%)
# 排期建议: 每日发布 1 篇
```

## 常见问题

### Q: 批量生成的短篇会不会雷同?

A: 专业版启用 `style-variation` 参数,每篇使用不同的风格、视角与结构。同时内置查重检测,确保每篇原创度高于 95%。建议批量生成后人工审阅,对相似度高的重新生成.
### Q: 投稿平台数据如何获取?

A: 1) 手动录入投稿记录;2) 支持与部分平台的 API 对接,自动同步状态;3) 读者数据需要从平台后台导出后导入分析.
### Q: AI 润色会改变我的写作风格吗?

A: 启用 `preserve-style` 参数后,润色保留作者原有风格,只优化节奏、语言与情感表达。可通过对比原文与润色稿,选择性采纳修改建议.
### Q: 系列短篇如何管理一致性?

A: 专业版提供系列管理功能:1) 共享人物卡与世界观设定;2) 自动检查时间线一致性;3) 追踪前文伏笔的回收;4) 生成系列大纲,确保各篇有联系又独立.
