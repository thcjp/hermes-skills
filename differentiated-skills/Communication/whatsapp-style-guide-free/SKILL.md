## 核心能力

### 1. 格式语法规范

WhatsApp支持以下文本格式,全部使用单层符号包裹:

#### 允许的格式模式

| 格式 | 语法 | 示例 | 效果说明 |
|:-----|:-----|:-----|:---------|
| 加粗 | `*文本*` | `*重要*` | 文字加粗显示 |
| 斜体 | `_文本_` | `_提示_` | 文字倾斜显示 |
| 删除线 | `~文本~` | `~旧价~` | 文字带删除线 |
| 等宽体 | `` `文本` `` | `` `代码` `` | 等宽字体显示 |
| 项目列表 | `* 文本` | `* 项目` | 圆点列表项 |
| 编号列表 | `1. 文本` | `1. 步骤` | 编号列表项 |
| 引用 | `> 文本` | `> 引用` | 引用块缩进 |

#### 格式使用要点

```text
┌─────────────────────────────────────────────────────┐
│  WhatsApp格式核心原则                                │
├─────────────────────────────────────────────────────┤
│  1. 所有格式符号均为单层(非双写)                     │
│  2. 符号与文本之间不留空格                            │
│  3. 每种格式独立使用,避免嵌套                        │
│  4. 列表符号后需有一个空格                            │
│  5. 引用符号后需有一个空格                            │
└─────────────────────────────────────────────────────┘
```

### 2. 禁止模式清单

以下Markdown语法在WhatsApp中不被支持,使用后会导致原始符号暴露:

| 禁止模式 | 问题描述 | 替代方案 |
|:---------|:---------|:---------|
| `**文本**` 双星号 | WhatsApp不识别双星号,星号会原样显示 | 改用 `*文本*` 单星号 |
| `# 标题` | 不支持Markdown标题语法 | 用 `*大写文本*` 代替 |
| `## 二级标题` | 同上 | 用 `*加粗文本*` 代替 |
| `\|表\|格\|` | 不支持Markdown表格 | 用列表格式代替 |
| `---` 水平线 | 不支持水平线语法 | 用 `__________` 下划线代替 |
| `` ``` `` 代码块 | 不支持多行代码块 | 逐行使用行内等宽体 |
| `[链接](url)` | 不支持Markdown链接 | 直接粘贴URL |
| `![图片](url)` | 不支持Markdown图片 | 直接发送图片文件 |

## 最佳实践

### 1. 格式符号不留空格

格式符号必须紧贴文本,中间不留空格,否则格式不会生效。

```text
正确: *重要通知*
错误: * 重要通知 *(星号和文字之间有空格)
```

### 2. 避免格式嵌套

WhatsApp对嵌套格式支持不稳定,建议每种格式独立使用。

```text
不推荐: *_加粗斜体_*(可能无法正确显示)
推荐: 选择一种格式使用即可
```

### 3. 列表符号统一

同一消息中的列表使用统一的符号风格。

```text
推荐(统一使用星号):
* 第一项
* 第二项
* 第三项

不推荐(混用符号):
* 第一项
- 第二项
* 第三项
```

### 4. 技术内容使用等宽体

订单号、验证码、代码等技术内容使用等宽体,便于识别和复制。

```text
推荐: 验证码: `836491`
不推荐: 验证码: 836491(普通字体,不易区分)
```

## 常见问题

### Q1: 为什么加粗文字两边显示了星号?

**A:** WhatsApp使用单星号 `*` 表示加粗,而非Markdown的双星号 `**`。请将 `**文本**` 改为 `*文本*`。另外,确保星号与文本之间没有空格。

### Q2: WhatsApp支持Markdown表格吗?

**A:** 不支持。WhatsApp不识别表格语法,请将表格数据转换为列表格式展示。

### Q3: 可以在一条消息中使用多种格式吗?

**A:** 可以。不同段落可以使用不同格式,只要每段遵循对应语法即可。但不建议在同一段文字中嵌套使用多种格式。

### Q4: 如何在WhatsApp中显示代码?

**A:** WhatsApp仅支持行内等宽体,使用单个反引号 `` ` `` 包裹。不支持多行代码块。多行代码请逐行使用行内等宽体。

### Q5: WhatsApp消息有长度限制吗?

**A:** WhatsApp文本消息最多支持4096个字符。超过此长度的消息会被截断或分多条发送。

### Q6: 免费版指南包含边界场景说明吗?

**A:** 免费版聚焦核心语法规范和常见模式。如需边界场景处理(如特殊字符转义、emoji与格式混用等)和高级排版模式,请升级至PRO版。

### Q7: 如何升级到PRO版?

**A:** PRO版与免费版完全兼容,升级后获得完整规范文档、边界场景处理、多语言指南和交互式格式测试等高级功能。直接安装PRO版Skill即可完成升级。
