# 照片配文工具 - 专业版
## 概述
---
照片配文工具专业版是一款面向专业摄影师与内容创作者的全平台照片配文生成解决方案。在完全兼容免费版 Instagram、X(Twitter)、Facebook 三大平台配文能力的基础上,专业版解锁了 Flickr、Glass、Tumblr、Bluesky、Threads、500px、Reddit、VSCO、Substack、Pinterest 共 13 个平台的配文生成能力。

无论是同时在 13 个平台发布摄影作品、批量处理一组系列照片的配文、维护统一的品牌语气风格,还是获取照片编辑分析建议,专业版都能高效完成,帮助创作者在多平台建立专业形象。

## 最佳实践
### 1. 全平台发布保持内容差异化
同一张照片在不同平台应有差异化表达,避免跨平台复制粘贴:

| 平台类型 | 表达策略 |
|:---------|:---------|
| 视觉优先(Instagram/VSCO/Glass) | 简短、留白、让图片说话 |
| 故事优先(Tumblr/Substack/Facebook) | 叙事感、背景故事、个人反思 |
| 技术优先(500px/Flickr/Reddit) | 器材、参数、拍摄条件 |
| 对话优先(X/Bluesky/Threads) | 口语化、即兴感、一句话抓人 |
| 搜索优先(Pinterest) | SEO关键词、描述性、可被发现 |

### 2. 批量配文保持系列一致性
处理系列作品时,统一以下要素:
- 器材描述格式一致
- 地点命名风格统一
- 语气基调保持一致
- 标签策略统一

### 3. 品牌配文严格遵循禁用词
品牌场景下,确保配文不出现禁用词,保持品牌调性一致:

```text
- "captured" -> 改用 "shot" / "took" / "made"
- "golden hour doing its thing" -> 具体描述光线行为
- "timeless" / "fleeting moment" -> 删除或替换
- "There's something about..." -> 直接进入正题
```

### 4. 编辑分析辅助提升作品质量
每次生成配文时,编辑分析章节会提供照片技术层面的改进建议,帮助摄影师持续提升:

```text
编辑分析关注维度:
- 构图:画面平衡、引导线、前景运用
- 曝光:高光/阴影细节、动态范围
- 色彩:色调统一性、色彩搭配
- 焦点:主体清晰度、景深控制
```

### 5. Pinterest 配文注重 SEO
Pinterest 是发现引擎,配文需融入可搜索关键词:

```text
[主题/氛围] — [地点]  或  [风格] [主题], [地点]

"Desert Road at Dusk, Amboy California"
"Film Photography, Mojave Desert Landscape"
```

### 6. Reddit 配文保持真实感
Reddit 社区反感营销腔与自我推销,配文需真实、谦虚:

```text
[地点或主题] [器材方括号]

"Bombay Beach [Canon EOS 1V, Tri-X 400]"
```

## 常见问题
### Q1: 13 个平台的配文会同时生成吗?
**A**: 是的。提供照片信息后,工具会一次性返回所有 13 个平台的配文(或指定的平台子集),每个平台独立呈现,无需多次请求。

### Q2: 批量配文最多支持多少张照片?
**A**: 专业版无硬性上限,但建议单次批量不超过 20 张,确保配文质量。超过 20 张建议分批处理。

### Q3: 品牌风格配置如何保存与复用?
**A**: 品牌风格配置以 YAML 文件形式保存,可在不同项目中复用。支持同时维护多个品牌配置,按需切换:

```text
请使用 brand_a.yaml 的品牌风格为这张照片生成配文。

请使用 brand_b.yaml 的品牌风格为这张照片生成配文。
```

### Q4: 编辑分析可以单独使用吗?
**A**: 编辑分析默认随配文一起生成(作为独立章节)。如只需编辑分析不需配文,可在请求中说明:"请只提供编辑分析,不需要配文。"

### Q5: 双语配文如何呈现?
**A**: 专业版支持中英双语配文。每个平台会同时输出中文与英文两个版本,便于跨地区发布:

```text
Instagram (中文):
公路尽头是橘红色的天...

Instagram (English):
The road ends where the sky turns orange...
```

### Q6: 免费版用户升级后使用方式有变化吗?
**A**: 完全兼容。专业版沿用免费版的输入方式与配文格式,升级后直接获得全部 13 个平台与高级功能的访问权限,原有使用习惯无需改变。

### Q7: 平台格式规范会随平台更新调整吗?
**A**: 会。各平台的字数限制、标签策略、格式规范会随平台更新动态调整。专业版会定期同步最新规范,确保配文始终符合平台原生风格。

- 编辑分析功能依赖 `photo-edit-analysis` 技能,同样无需额外 API Key

- **分类**: MD(纯 Markdown 指令,无需 exec 命令行执行能力)
- **说明**: 基于自然语言的 AI Skill,通过指令驱动 Agent 为照片生成 13 个社交平台适配的配文内容。专业版完全兼容免费版三大平台配文能力,额外解锁 Flickr、Glass、Tumblr、Bluesky、Threads、500px、Reddit、VSCO、Substack、Pinterest 共 10 个专业平台,并提供批量配文生成、品牌风格定制与照片编辑分析能力,适合专业摄影师与内容创作者的规模化内容生产。
