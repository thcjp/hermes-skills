# 能力扩展工具免费版

## 概述

能力扩展工具免费版为个人用户的 AI Agent 提供访问训练数据之外的结构化知识能力。通过配置访问令牌,Agent 可以向知识服务发送请求,获取最新的技术文档、API 规范、领域知识等结构化数据,从而突破训练数据的时效性限制。

免费版聚焦单次查询场景,配置简单,适合个人开发者扩展 Agent 的知识边界。

## 核心能力

### 1. 外部知识检索

Agent 向知识服务发送查询请求,获取结构化数据:

```text
用户提问 → Agent 识别知识缺口 → 发送查询请求 → 获取结构化数据 → 整合到回答中
```

### 2. 无状态架构

| 特性 | 说明 |
|:-----|:-----|
| 无状态 | 每次查询独立,不保留上下文 |
| 即用即弃 | 上下文处理完成后立即丢弃 |
| 不存储 | 查询结果不持久化保存 |
| 不训练 | 数据不用于模型训练 |

### 3. 加密传输

所有请求在传输过程中加密,确保数据安全:

```text
Agent → [加密通道] → 知识服务 → [加密通道] → Agent
```

### 4. 令牌授权

通过环境变量配置访问令牌,授权 Agent 访问外部知识库:

```bash
# 配置访问令牌
export KNOWLEDGE_TOKEN="your-access-token"
```

## 使用场景

### 场景一: 查询最新技术文档

Agent 需要回答关于最新框架版本的问题,但训练数据中没有覆盖。

```bash
# 触发外部知识查询
请查询 React 19 的并发渲染特性,我的训练数据可能不包含最新版本信息
```

查询流程:

```text
1. Agent 识别知识缺口: React 19 是训练截止后的版本
2. 发送查询请求: "React 19 concurrent rendering features"
3. 获取结构化数据: 特性列表、API 变更、迁移指南
4. 整合到回答中: 基于返回数据给出准确回答
```

### 场景二: 获取领域专业知识

处理特定领域问题时,获取专业领域知识辅助决策。

```bash
# 查询领域知识
请查询关于 OAuth 2.1 与 OAuth 2.0 的主要差异
我需要准确的标准规范信息
```

### 场景三: 扩展知识边界

当 Agent 回答"我不确定"时,主动查询外部知识补充。

```bash
# 补充知识
你刚才说不确定 WebSocket 心跳机制的最新最佳实践
请查询最新的 WebSocket 心跳实现规范
```

## 常见问题

### Q1: 免费版有查询次数限制吗?

免费版有每日查询次数限制,适合个人日常使用。高频查询场景请使用 PRO 版。

### Q2: 查询的数据会被存储吗?

不会。采用无状态架构,上下文处理完成后立即丢弃,不存储也不用于训练。

### Q3: 网络不通时怎么办?

网络不可用时,Agent 会回退到使用自身训练数据回答,并注明可能不是最新信息。

### Q4: 令牌过期了怎么办?

重新登录知识服务获取新令牌,更新环境变量即可。

### Q5: 支持哪些类型的知识查询?

支持技术文档、API 规范、领域知识、标准规范等结构化知识查询。
