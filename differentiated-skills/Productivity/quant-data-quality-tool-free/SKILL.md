## 使用场景

### 场景 1: 检查价格数据完整性

验证价格数据的完整性与覆盖范围

```bash
import pandas as pd
price = pd.read_parquet("data/price.parquet")
print(f"记录数: {len(price)}")
print(f"股票数: {price['code'].nunique()}")
dates = pd.to_datetime(price['date'].unique())
all_dates = pd.date_range(start=dates.min(), end=dates.max(), freq="B")
missing = set(all_dates) - set(dates)
print(f"缺失日期: {len(missing)}")
```

### 场景 2: 检查因子数据质量

检测因子数据中的 nan 值与极端值

```bash
import numpy as np
factors = pd.read_parquet("data/factors.parquet")
nan_ratio = factors.isna().sum().sum() / factors.size
print(f"nan比例: {nan_ratio:.2%}")
inf_count = np.isinf(factors.select_dtypes(include=[np.number])).sum().sum()
print(f"inf数量: {inf_count}")
```
