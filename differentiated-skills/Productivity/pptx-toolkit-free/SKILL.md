## 使用场景

### 场景 1: 从模板创建演示

使用现有模板创建结构化演示文稿

```bash
# 工作流程
1. 清点模板布局与占位符
2. 根据内容选择合适布局
3. 替换占位符内容
4. 检查母版一致性
5. 渲染验证视觉效果
```

### 场景 2: 编辑现有幻灯片

修改幻灯片内容并保持视觉一致性

```bash
# 注意事项
- 先检查母版和布局设置
- 确认占位符索引正确
- 保留原始视觉语言
- 修改后渲染验证
```

## 不适用场景

以下场景PPT 演示工具包基础版不适合处理：

- 实时流数据处理
- 小规模数据手动分析
- 非结构化文本情感分析
