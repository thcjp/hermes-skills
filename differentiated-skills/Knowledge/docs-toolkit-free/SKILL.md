### 基础用法

```python
# 文档决策树导航
decision_tree = {
    "如何设置": "查看 providers/ 或 start/ 目录",
    "为什么不工作": "查看 troubleshooting 故障排查",
    "如何配置": "查看 gateway/ 或 concepts/ 目录",
    "这是什么": "查看 concepts/ 概念文档",
    "如何自动化": "查看 automation/ 目录",
}
question = "如何设置定时任务"
for keyword, action in decision_tree.items():
    if keyword in question:
        print(f"导航建议: {action}")
        break
```

### 执行结果

执行上述代码后，将根据输入参数返回结构化结果。免费版支持单次操作，适合处理单个文件或任务。

## 常见问题

### Q1: 处理速度较慢怎么办？

A: 免费版为单线程处理，对于大文件建议分批处理或升级到专业版获得并行处理能力。

### Q2: 支持的文件格式有哪些？

A: 支持标准格式输入输出，常见格式包括JSON、YAML、Markdown等。

### Q3: 如何获取API Key？

A: 本Skill基于Markdown指令，无需额外API Key。如需外部服务API，请参考对应服务的注册流程。

### Q4: 免费版有使用限制吗？

A: 免费版支持单次操作，适合个人日常使用。如需批量处理或高级功能，建议升级到专业版。
