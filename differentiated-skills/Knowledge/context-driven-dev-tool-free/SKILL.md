### 基础用法

```python
# 上下文文档结构验证
import os

context_docs = ["product.md", "tech-stack.md", "workflow.md", "tracks.md"]
context_dir = "./context"

for doc in context_docs:
    path = os.path.join(context_dir, doc)
    if os.path.exists(path):
        print(f"[OK] {doc} 已存在")
    else:
        print(f"[缺失] {doc} 需要创建")
```
