### 基础用法

```python
# PDF工具包基础
from pypdf import PdfReader, PdfWriter
# ...
# 提取文本
reader = PdfReader("input.pdf")
text = reader.pages[0].extract_text()
print(text[:200])
# ...
# 加密PDF
writer = PdfWriter()
for page in reader.pages:
    writer.add_page(page)
writer.encrypt("password123")
with open("encrypted.pdf", "wb") as f:
    writer.write(f)
print("PDF已加密")
```
