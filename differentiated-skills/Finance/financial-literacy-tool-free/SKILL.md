## 使用场景

### 场景一：学习基础概念

用户输入："什么是复利？"

```text
复利是指利息加入本金后再次计算利息的方式，即"利滚利".
# ...
【举例】
本金10000元，年利率5%，投资10年：
- 单利：10000 + 10000×5%×10 = 15000元
- 复利：10000 × (1+5%)^10 ≈ 16289元
# ...
复利的威力在于时间。爱因斯坦称复利为"世界第八大奇迹".
# ...
【小贴士】
越早开始投资，复利效应越显著.
每月定投1000元，年化8%，30年后约150万元.
```

### 场景二：理财入门咨询

用户输入："刚工作，月薪8000，怎么理财？"

```text
建议遵循"50-30-20"法则：
- 50% 必需支出：4000元（房租/吃饭/交通）
- 30% 生活改善：2400元（娱乐/学习/社交）
- 20% 储蓄投资：1600元（应急金+定投）
# ...
【第一步】建立应急金
存够3-6个月生活费（约15000-25000元）放入货币基金.
# ...
【第二步】开始定投
每月拿出500-1000元定投指数基金，长期持有.
# ...
【第三步】配置保险
优先配置医疗险和意外险，年保费几百元起.
```

### 场景三：风险防范学习

用户输入："怎么识别金融诈骗？"

```text
金融诈骗常见特征：
1. 承诺高收益低风险（年化超过15%需警惕）
2. 要求快速转账或保密
3. 无正规金融牌照
4. 拉人头返利模式
5. 虚拟货币/外汇平台
# ...
【防骗口诀】
收益越高风险越大，
天上不会掉馅饼，
不懂的不投，
先查牌照再投资.
```

## 常见问题(补充)

### Q1：免费版支持个性化理财方案吗？

免费版提供通用理财建议，不支持根据个人情况定制方案。如需个性化理财规划，建议升级PRO版.
### Q2：知识内容是否权威？

内容参考标准金融教材与监管机构科普资料。但金融知识更新较快，建议以最新法规和产品说明为准.
### Q3：学完能考取金融证书吗？

免费版为科普性质，不覆盖专业认证考试内容。如需备考基金/证券从业资格等，建议升级PRO版获取专业课程.
### Q4：内容适合什么人群？

适合金融零基础的成年人、大学生和初入职场的年轻人。已有金融背景的用户可能觉得内容偏基础.
