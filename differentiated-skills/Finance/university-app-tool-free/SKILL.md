# 命理咨询入门工具（免费版）

## 概述

本工具为个人用户提供传统命理八字分析能力。基于出生时间排出八字命盘，分析五行强弱、性格特质与流年运势。适合对传统命理文化感兴趣的个人用户，用于自我认知与人生规划参考。

注：本工具内容仅供文化娱乐参考，不构成任何决策建议。

## 最佳实践

1. **时间准确**：出生时间尽量精确到时辰（2小时为一时辰）
2. **农历转换**：工具自动处理公历转农历，无需手动转换
3. **综合判断**：命理分析需综合四柱，不可单看一柱
4. **理性对待**：命理为传统文化，仅供参考娱乐

| 实践要点 | 说明 |
| --- | --- |
| 时辰确认 | 早子时(00-01)与晚子时(23-24)需区分 |
| 性别影响 | 大运排法因性别而异 |
| 文化性质 | 命理为传统文化，非科学预测 |
| 免责声明 | 结果仅供文化娱乐参考 |

## 常见问题

### Q1：命理分析准确吗？

命理学属于传统文化范畴，非科学预测方法。分析结果仅供文化娱乐参考，不应作为人生决策的依据。

### Q2：免费版支持合婚分析吗？

免费版不包含合婚（两人命盘匹配）分析。如需合婚分析，建议升级PRO版。

### Q3：出生时间不确定怎么办？

出生时间不确定会影响时柱准确性。建议尽量确认准确时间，或提供大致时间段进行范围分析。

### Q4：支持紫微斗数等其他命理吗？

免费版仅支持八字命理。如需紫微斗数、奇门遁甲等其他命理分析方法，建议升级PRO版。
