# 图表工坊免费版

一款本地优先的图表生成引擎，将数字转化为清晰的可视化图形，支持四种基础图表类型与智能推荐。

## 概述

在数据分析与汇报场景中，将数字转化为图表是最直接的表达方式。然而，大多数在线图表工具需要上传数据到云端，存在隐私泄露风险；专业可视化库学习成本高，不适合快速出图。本工具通过本地Python脚本生成图表，数据不离开本机，兼顾隐私与效率。

免费版聚焦于四种最常用的基础图表类型与智能推荐能力，适合个人开发者快速将数据可视化。

## 核心能力

### 四种基础图表类型

| 图表类型 | 适用场景 | 典型用途 |
|----------|----------|----------|
| `bar`（柱状图） | 类别比较 | 各产品销量对比 |
| `line`（折线图） | 趋势变化 | 月度增长趋势 |
| `pie`（饼图） | 占比构成 | 预算分配比例 |
| `scatter`（散点图） | 关系探索 | 价格与销量相关性 |

### 智能图表推荐
- 根据数据特征（类别数、时间维度、数值范围）自动建议最佳图表类型
- 输入标签与数值后，推荐引擎分析数据分布给出建议
- 支持多维度数据时推荐组合图表方案

### 本地生成与存储
- 所有图表在本地通过Python与matplotlib生成
- 数据存储在本地文件系统，不上传任何云端
- 图表历史记录持久化，可随时查看与复用
- 输出高质量PNG图片，适合嵌入报告与幻灯片

### 历史记录管理
- 自动记录每次生成的图表元数据（类型、标题、时间）
- 通过`list_charts.py`查看历史图表列表
- 图表文件按时间戳命名，避免覆盖

## 使用场景

### 场景一：报表配图快速生成
月度报告中需要展示各产品线销量对比。通过命令行输入产品名与销量数据，一键生成柱状图PNG，直接嵌入到报告文档中。

### 场景二：数据趋势快速验证
分析师拿到一组时序数据，希望快速判断趋势走向。通过折线图生成脚本输入数据，几秒内得到可视化结果，辅助决策。

### 场景三：占比关系直观展示
团队预算分配需要向成员公示。输入各部门预算数值，生成饼图，直观展示占比关系，提升沟通效率。

### 场景四：数据关系探索
研究者有一组价格与销量的配对数据，希望探索两者关系。通过散点图生成脚本输入数据，观察点的分布趋势，判断相关性。

## 快速开始

### 第一步：初始化存储

```bash

```

### 第二步：生成第一张图表



## 最佳实践

### 图表类型选择
- 比较不同类别的数值：用柱状图
- 展示时间序列的变化趋势：用折线图
- 表达各部分占整体的比例：用饼图
- 探索两个变量之间的关系：用散点图
- 不确定时先用`suggest_chart.py`获取建议

### 数据准备建议
- 标签与数值数量必须一致
- 数值使用英文逗号分隔，不要包含空格
- 标签过长时图表会自动旋转，建议控制在6个字符以内
- 饼图的数值会自动归一化为百分比

### 输出与使用建议
- 生成的PNG图片分辨率默认为300dpi，适合打印与投影
- 图片文件名包含时间戳，不会覆盖历史文件
- 定期清理`output/`目录，避免文件过多占用磁盘
- 嵌入到报告时建议保持图片比例，避免拉伸变形

### 隐私与安全
- 所有数据处理在本地完成，不上传任何云端
- 敏感数据可放心使用，无需担心泄露
- 图表文件存储在用户目录下，注意设置适当权限

## 常见问题

### Q1：提示找不到python3命令？
确保Python 3已安装并在PATH中。Windows下可能需要使用`python`而非`python3`。执行`python3 --version`验证。

### Q2：提示matplotlib未安装？
执行`pip3 install matplotlib`安装。若权限不足，使用`pip3 install --user matplotlib`安装到用户目录。

### Q3：中文标签显示为方框？
matplotlib默认字体不支持中文。在脚本中设置中文字体：`plt.rcParams['font.sans-serif'] = ['SimHei']`（Windows）或`['Arial Unicode MS']`（macOS）。

### Q4：数值中有小数可以吗？
可以。数值支持整数与小数，例如`--values "12.5,18.3,15.7"`。

### Q5：能否生成更多图表类型？
免费版支持四种基础类型。如需堆叠柱状图、面积图、雷达图、热力图等高级类型与自定义模板，请使用专业版。
