# 图表工坊免费版

一款本地优先的图表生成引擎，将数字转化为清晰的可视化图形，支持四种基础图表类型与智能推荐.
## 概述

在数据分析与汇报场景中，将数字转化为图表是最直接的表达方式。然而，大多数在线图表工具需要上传数据到云端，存在隐私泄露风险；专业可视化库学习成本高，不适合快速出图。本工具通过本地Python脚本生成图表，数据不离开本机，兼顾隐私与效率.
免费版聚焦于四种最常用的基础图表类型与智能推荐能力，适合个人开发者快速将数据可视化.

- 数值使用英文逗号分隔，不要包含空格
- 标签过长时图表会自动旋转，建议控制在6个字符以内
- 饼图的数值会自动归一化为百分比

### 输出与使用建议
- 生成的PNG图片分辨率默认为300dpi，适合打印与投影
- 图片文件名包含时间戳，不会覆盖历史文件
- 定期清理`output/`目录，避免文件过多占用磁盘
- 嵌入到报告时建议保持图片比例，避免拉伸变形

### 隐私与安全
- 所有数据处理在本地完成，不上传任何云端
- 敏感数据可放心使用，无需担心泄露

## 常见问题

### Q1：提示找不到python3命令？
确保Python 3已安装并在PATH中。Windows下可能需要使用`python`而非`python3`。执行`python3 --version`验证.
### 依赖详情
执行`pip3 install matplotlib`安装。若权限不足，使用`pip3 install --user matplotlib`安装到用户目录.
### Q3：中文标签显示为方框？
matplotlib默认字体不支持中文。在脚本中设置中文字体：`plt.rcParams['font.sans-serif'] = ['SimHei']`（Windows）或`['Arial Unicode MS']`（macOS）.
### Q4：数值中有小数可以吗？
可以。数值支持整数与小数，例如`--values "12.5,18.3,15.7"`.
### Q5：能否生成更多图表类型？
免费版支持四种基础类型。如需堆叠柱状图、面积图、雷达图、热力图等高级类型与自定义模板，请使用专业版.
