# Excel 专家（免费版）

本 Skill 把 Excel 当作"决策引擎"而非教程，先诊断问题类型再给出最稳健的工具路径。免费版覆盖 7 大类常见表格场景，输出可复制的公式、透视表配置与清洗步骤。

## 概述

表格工作的真正痛点不在于"功能不够"，而在于"工具选错"——用嵌套 IF 解决本该用查找表的问题，用 VBA 写本该透视表秒杀的任务，用脆弱的 VLOOKUP 硬编码列号导致插入列后全盘崩溃。本 Skill 的设计哲学是"诊断先行"：先识别问题属于哪一类，再选择最稳健的工具，最后输出可直接使用的方案与失败点提示。

## 使用场景

### 场景一：跨表查找（运营角色）
从价格表按产品编码取价，找不到的返回 0。Agent 诊断后输出 XLOOKUP 方案（兼容旧版给 INDEX-MATCH），并提示硬编码列号的失败点。

### 场景二：脏日期数据（数据分析师角色）
源表日期混合了 `2024.01.01` 与 `01/01/24`，Excel 无法排序。Agent 识别为归一化问题而非排序问题，输出统一解析为真实日期序列号的公式链。

### 场景三：月度汇总重建（财务角色）
每月手工重建按区域 × 产品线的汇总。Agent 选择透视表方案，给出具体的行 / 列 / 值 / 筛选字段，并提示何时加入计算字段。

### 场景四：跨平台兼容（开发者角色）
用户使用 Excel 365 但下游同事用旧版 Excel 与 Google Sheets。Agent 优先输出兼容方案，对 365 独有函数给出降级路径。

### 示例：跨表查找

**用户输入**：从"价格"表按产品编码取价，找不到返回 0。

**诊断**：关系查找 + 错误兜底 + 现代版本优先

**输出**：
```excel
=XLOOKUP(C2, 价格!C:C, 价格!A:A, 0)
```

**理由**：XLOOKUP 默认精确匹配，第 4 参数直接控制找不到时的返回值，比 IFERROR(VLOOKUP(...)) 更简洁。硬编码列号会导致插列后崩溃，XLOOKUP 的返回范围是完整列引用，可自动跟随。

**失败点**：旧版 Excel 不支持 XLOOKUP，需降级为 `=IFERROR(INDEX(价格!A:A, MATCH(C2, 价格!C:C, 0)), 0)`。

### 工具选择优先级

```text
1. 标准公式（XLOOKUP / SUMIFS / IFS / TEXT / EDATE）
   - 适用：可在单元格里干净解决的问题
2. 动态数组（FILTER / UNIQUE / SORT / SEQUENCE）
   - 适用：需要溢出或多行匹配结果
3. 透视表
   - 适用：关系型汇总、分组聚合、仪表盘
4. 清洗流程（TRIM/CLEAN/SUBSTITUTE 链）
   - 适用：源数据脏，先洗再做分析
5. VBA / 宏（最后手段）
   - 适用：重复多步、事件驱动、跨文件循环
```

### 黄金法则

- 公式能解决就别用 VBA
- 查找表能解决就别用嵌套 IF
- 需要行级返回值就别用透视表
- 数据脏就先清洗，别用分析逻辑掩盖
- 永远优先"稳健"而非"巧妙"

## 最佳实践

1. **诊断先于公式**：不要急着写公式，先判断问题类型，选择最稳健的工具。
2. **避免硬编码列号**：使用整列引用或命名范围，插列后不会崩。
3. **错误兜底**：所有查找公式都加上 `if_not_found` 或 `IFERROR`，避免 `#N/A` 蔓延。
4. **阈值集中化**：业务阈值写入查找表，不要散落在公式里。
5. **跨平台降级**：先确认目标平台，对 365 独有函数提供旧版降级方案。
6. **VBA 谨慎用**：仅用于重复多步、事件驱动、跨文件场景，简单计算别上 VBA。
7. **可视化克制**：默认折线（趋势）、柱状（对比），饼图只在占比场景使用。

## 常见问题

### Q1：XLOOKUP 与 VLOOKUP 怎么选？
A：现代 Excel 优先 XLOOKUP——返回范围可以是整列，插列后不崩，找不到直接控制返回值。旧版 Excel 用 INDEX-MATCH 组合替代。

### Q2：嵌套 IF 几层算太多？
A：超过 3 层就该用查找表或 IFS。嵌套 IF 难以维护，阈值散落，修改时容易遗漏分支。

### Q3：透视表 vs SUMIFS 怎么选？
A：需要交互式切片、动态更新选透视表；需要在公式中嵌入结果或返回数组选 SUMIFS。重建月度汇总优先透视表。

### Q4：源数据日期混乱怎么办？
A：先用 DATEVALUE 配合 SUBSTITUTE / REPLACE 统一格式，再转为真实日期序列号。分析前必须清洗，否则排序、分组、时间序列都会出错。

### Q5：Google Sheets 的公式能直接用在 Excel 吗？
A：大部分通用，但 QUERY、IMPORTRANGE、SPARKLINE 等是 Sheets 独有；动态数组在 Excel 365 中支持，旧版 Excel 需降级为 CSE 数组公式。
