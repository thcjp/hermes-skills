## 快速开始

### 基础搭建（<60秒）

配置定时任务与告警规则：

```json
{
  "name": "macro-sentinel-pro-daily",
  "schedule": {
    "kind": "cron",
    "expr": "0 22 * * *",
    "tz": "Asia/Shanghai"
  },
  "payload": {
    "kind": "agentTurn",
    "message": "执行专业版宏观数据监控，采集+深度解读+生成可视化图表+多渠道推送"
  }
}
```

### 标准搭建（<120秒）

配置告警阈值与推送渠道：

```json
{
  "skills": {
    "entries": {
      "macro-sentinel": {
        "config": {
          "alerts": {
            "CPI_year_over_year": {"threshold": 3.0, "direction": "above"},
            "PPI_year_over_year": {"threshold": -2.0, "direction": "below"},
            "GDP_quarterly": {"threshold": 4.0, "direction": "below"},
            "LPR_change": {"threshold": 0, "direction": "any_change"}
          },
          "push_channels": ["email", "wechat_work", "feishu", "dingtalk"],
          "chart_engine": "matplotlib",
          "report_format": ["pdf", "html"]
        }
      }
    }
  }
}
```

### 完整搭建（<300秒）

配置高频监控与专业报告：

```text
用户："创建完整宏观监控体系：每小时采集高频数据+每日深度简报+每周专业周报+阈值告警实时推送"
# ...
Agent执行：
1. 配置高频监控（每小时采集重要数据源）
2. 配置每日简报（22:00 采集+解读+推送）
3. 配置每周周报（每周日生成深度分析）
4. 配置告警规则（CPI>3%、GDP<4%、LPR变动等）
5. 配置多渠道推送（邮件+企业微信+飞书+钉钉）
6. 测试执行并验证
```

## 核心能力
### 一、指标深度解读（专业版独有）

| 功能 | 说明 | 示例 |
|:-----|:-----|:-----|
| 超预期分析 | 分析实际值与预期值差异及原因 | "CPI超预期0.3个百分点，主因食品价格上涨" |
| 低于预期解读 | 分析低于预期的原因与影响 | "GDP低于预期，受出口下滑拖累" |
| 市场影响预判 | 预判数据对市场的影响 | "通胀超预期，预计债市承压，股市结构性机会" |
| 政策影响分析 | 分析数据对政策制定的影响 | "CPI高位运行，央行短期内降息概率降低" |

```text
# 深度解读示例
💡 CPI同比3.2%（预期3.1%，前值3.0%）
# ...
【超预期分析】
超预期0.3个百分点，主要驱动因素：
- 食品价格环比上涨1.5%（猪肉价格反弹+鲜菜季节性上涨）
- 服务价格环比上涨0.4%（暑期出行需求旺盛）
- 核心CPI（剔除食品能源）同比1.8%，保持稳定
# ...
【市场影响预判】
- 债市：通胀超预期，长端利率可能上行5-10bp
- 股市：消费板块短期受益，高估值成长股承压
- 汇率：中美利差收窄，人民币汇率小幅承压
# ...
【政策影响分析】
- 央行：短期内降息概率降低，维持中性偏松
- 财政：民生补贴可能加码，稳定消费预期
```

**输入**: 用户提供一、指标深度解读（专业版独有）所需的指令和必要参数.
**处理**: 解析一、指标深度解读（专业版独有）的输入参数,完成核心逻辑,返回结构化响应.
**输出**: 返回一、指标深度解读（专业版独有）的响应数据,包含状态码、结果和日志.

## 使用场景

### 场景一：机构投资决策支持（基金经理）

**痛点**：管理百亿资金需实时掌握宏观数据，但信息源分散，难以快速形成决策依据.
**解决方案**：

```text
配置：
- 高频监控：每小时采集央行操作与重要新闻
- 告警规则：CPI>3%、GDP<4%、LPR变动实时告警
- 推送渠道：企业微信（白天）+邮件（晚间）
- 专业报告：每日简报+每周深度周报
# ...
每日晨会前收到：
1. 隔夜国际市场数据（欧美CPI/就业/利率）
2. 国内昨日政策动态
3. 今日数据发布预告
4. 告警事项汇总
5. 趋势图表
# ...
每周日收到周报：
1. 周度宏观数据汇总
2. 趋势分析与拐点判断
3. 下周重要事件预告
4. 资产配置建议
```

## 性能优化策略

### 数据采集优化

1. **增量采集**：仅采集过去24小时新增数据，避免全量重复
2. **并行采集**：多数据源并行访问，减少总耗时
3. **缓存策略**：历史数据本地缓存，减少重复请求
4. **失败重试**：网络异常自动重试3次，间隔递增

### 告警优化

1. **去重机制**：同一指标短时间内多次触发仅告警一次
2. **分级推送**：高级告警实时推送，低级告警汇总推送
3. **静默期**：非工作时间仅推送critical级别告警
4. **回执确认**：重要告警需确认接收，未确认升级推送

### 报告生成优化

1. **模板复用**：报告模板缓存，减少重复渲染
2. **图表预生成**：常用图表提前生成，按需引用
3. **异步生成**：大报告异步生成，不阻塞主流程
4. **增量更新**：周报/月报基于日报数据增量汇总

### 成本控制

- 高频监控仅采集关键指标，非关键指标用每日频率
- 可视化图表按需生成，非每日报告必含
- 多渠道推送按优先级选择，避免重复推送
- 历史数据归档压缩，减少存储占用

## FAQ

### Q1：免费版与专业版有什么区别？

免费版提供核心数据采集与整理（国际/国内/政策/资讯+科普解释+简报推送）。专业版新增指标深度解读、历史趋势对比、高频实时监控、多渠道推送、可视化图表、阈值告警、专业报告七大高级功能。此外提供6种角色场景指南、性能优化策略和多平台集成示例.
### Q2：告警如何配置？

通过JSON配置告警规则，包含指标名、阈值、方向（above/below/any_change）、严重级别（critical/high/medium/low）。告警触发后自动生成告警消息（含数据、影响分析、建议），推送至配置的渠道。支持告警去重、分级推送、静默期.
### Q3：可视化图表支持哪些类型？

支持折线图（趋势）、柱状图（对比）、热力图（相关性）、雷达图（多维评估）、散点图（相关性分析）等10+种图表类型。基于matplotlib渲染，可自定义颜色、字体、标注。图表可嵌入报告或单独输出PNG/SVG.
### Q4：高频监控会消耗大量资源吗？

专业版采用增量采集策略，仅采集过去1小时新增数据，避免全量重复。多数据源并行访问，配合本地缓存，每小时采集耗时约30秒。建议非关键指标用每日频率，仅关键指标用高频监控.
### Q5：多渠道推送如何配置？

每个渠道需配置对应的Token/Webhook：邮件需SMTP配置，企业微信需群机器人Webhook，飞书需应用Token，钉钉需群机器人Token。所有Token通过环境变量配置，不硬编码。支持按严重级别选择推送渠道.
### Q6：专业报告可以自定义模板吗？

支持自定义报告模板。预设日报/周报/月报/季报四种模板，每种可自定义章节结构、图表样式、分析深度。模板支持变量引用（如{{本月CPI}}、{{同比变化}}），自动填充采集数据.
### Q7：历史数据可以存储到数据库吗？

支持。专业版可将采集的宏观数据存入 `PostgreSQL`、MySQL、SQLite等数据库，便于历史回溯与深度分析。数据库连接配置存储在环境变量中，不硬编码.
### Q8：告警去重机制如何工作？

同一指标在短时间内（默认5分钟）多次触发阈值，仅推送第一次告警，后续合并为"持续告警"汇总推送。用户可配置去重时间窗口。高级别告警不受去重限制.
### Q9：数据源采集失败如何处理？

单个数据源失败时跳过并记录，不影响其他数据源采集。失败会在简报中标注。系统自动重试3次（间隔递增）。连续3天失败的数据源会触发维护告警，提醒更新采集规则.
### Q10：如何生成跨年度对比分析？

通过历史趋势对比功能，选择自定义时段（如2020-2025年），系统自动采集该时段所有数据，生成同比/环比对比、趋势分析、均值回归判断。支持多指标同时对比，输出趋势图与相关性矩阵.
### Q11：支持海外宏观数据吗？

支持。通过Trading Economics和FRED采集全球主要经济体的宏观数据：美国（GDP/CPI/就业/利率）、欧元区（GDP/CPI/PMI）、日本（GDP/CPI/利率）、英国（GDP/CPI/利率）等。支持中英文指标名映射.
### Q12：报告可以导出哪些格式？

支持PDF（正式报告）、HTML（网页浏览）、Excel（数据表）、PNG（图表）、Markdown（简报）五种格式。PDF报告含完整图表与排版，适合正式场合。HTML支持交互式图表.

## 专业版特性

本专业版相比免费版新增以下能力：

- **指标深度解读**：自动分析超预期/低于预期的原因、市场影响预判、政策影响分析，让数据变为决策依据
- **历史趋势对比**：支持同比/环比/自定义时段对比，识别趋势拐点，判断均值回归，提供历史视角
- **可视化图表**：自动生成趋势折线图、对比柱状图、相关性热力图等10+种图表，让数据一目了然
- **阈值告警**：自定义指标阈值与触发条件，突破时实时推送告警（含数据+影响+建议），响应速度分钟级
- **高频实时监控**：支持每小时采集频率，重要数据发布期实时跟踪，不遗漏关键信息
- **多渠道推送**：支持邮件、企业微信、飞书、钉钉四大渠道，按严重级别选择推送方式
- **专业报告**：自动生成日报/周报/月报/季报，含完整图表与深度分析，支持PDF/HTML/Excel多格式

此外，专业版还提供：
