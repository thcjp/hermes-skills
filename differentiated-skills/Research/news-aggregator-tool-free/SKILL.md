## 概述

新闻聚合工具免费版是一款自动聚合国内外社会、科技、军事新闻的工具。通过搜索多个新闻源、自动筛选过滤重复与不可靠内容、按类别整理要点，帮助用户高效获取结构化的新闻汇总，避免信息过载和来源偏差。

## 常见问题

### Q1：免费版支持哪些新闻类别？

免费版支持三个类别：国内科技新闻、国内军事新闻、国内社会新闻，覆盖 10 个核心信息源。

### Q2：免费版能获取国际新闻吗？

免费版不支持国际新闻源。如需 TechCrunch、Defense News 等国际媒体，请升级至 PRO 版本。

### Q3：新闻更新频率如何？

免费版在用户触发时实时搜索获取最新新闻，不支持后台自动更新。如需定时自动聚合，请升级至 PRO 版本。

### Q4：去重功能如何工作？

免费版通过标题相似度匹配（阈值 85%）进行去重，同一事件的多篇报道会被合并，保留信息最完整的版本。

### Q5：免费版与 PRO 版本的区别？

| 对比项 | 免费版 | PRO 版本 |
|:-------|:-------|:---------|
| 新闻类别 | 3类（科技/军事/社会） | 6类+自定义 |
| 信息源 | 10个国内源 | 30+国内外源 |
| 国际新闻 | 不支持 | 完整支持 |
| 定时聚合 | 不支持 | 支持自动聚合 |
| 导出格式 | Markdown | MD/PDF/Email |
| 自定义源 | 不支持 | 支持接入 |
| API 集成 | 不支持 | REST API |
