## 最佳实践

### 提问技巧

1. **明确具体**：问题越具体，搜索越精准
2. **包含关键词**：直接说出你想查的内容
3. **避免歧义**：确保问题指向单一主题
4. **适当限定**：可加入时间、地点等限定

```text
推荐提问：
- "查一下 2026 年中国 GDP 增长率"
- "搜索 Python 异步编程最新教程"
- "帮我查一下北京到上海的高铁时刻表"

不推荐提问：
- "帮我查一下"（过于模糊）
- "你能做什么"（非搜索意图）
- "写一首诗"（非搜索任务）
```

### 结果使用建议

- 优先查看「核心发现」，快速了解要点
- 查看「详细信息」获取来源链接
- 点击来源链接查看完整内容
- 参考「建议」获取实用提示

## 常见问题

### 搜索结果不准确

```text
可能原因：
- 问题过于模糊
- 关键词提取不准确
- 搜索引擎返回无关结果

解决方案：
- 更具体地描述你的问题
- 直接使用关键词而非完整句子
- 重新表述问题
```

### 搜索速度慢

```text
可能原因：
- 网络连接较慢
- 搜索引擎响应慢
- 结果筛选耗时

解决方案：
- 检查网络连接
- 简化查询内容
- 减少结果数量要求
```

### 结果数量少

```text
可能原因：
- 搜索主题过于小众
- 关键词过于具体
- 搜索引擎限制

解决方案：
- 使用更宽泛的关键词
- 尝试相关主题搜索
- 适当放宽搜索范围
```

### 无法搜索

```text
可能原因：
- 网络连接问题
- 搜索服务不可用
- 触发词未识别

解决方案：
- 检查网络连接
- 使用明确的搜索触发词
- 重试查询
```
