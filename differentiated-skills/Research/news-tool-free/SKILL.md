# 个性化新闻助手（免费版）

## 概述

个性化新闻助手免费版是一款能够学习用户兴趣偏好并提供定制化新闻简报的工具。通过构建用户兴趣画像，逐步了解用户关注的领域、偏好的输出格式和接收时间，提供越来越精准的新闻推荐服务.
所有用户数据（兴趣偏好、阅读历史）均存储在本地目录，不上传至任何外部服务，充分保护用户隐私。本版本适合个人用户日常新闻浏览和学习研究，如需更高级的多用户管理、企业推送等能力，可升级至 PRO 版本.
## 核心能力

### 能力概览

| 能力项 | 说明 |
|---|---|
| 兴趣画像 | 记录用户具体兴趣领域及比例分配 |
| 格式定制 | 支持要点列表、叙述段落、标题速览等格式 |
| 时间偏好 | 记录用户偏好接收简报的时间段 |
| 多源覆盖 | 重要事件交叉验证多个信息源 |
| 阅读学习 | 跟踪用户阅读行为，动态调整推荐 |
| 隐私保护 | 所有数据本地存储，不上传外部 |

新用户首次使用时，Agent 会引导设置兴趣偏好.
```text

1. 询问具体兴趣领域（非泛泛的分类）
2. 确认多兴趣的比例分配

```


## 常见问题

### Q1：我的数据存储在哪里？安全吗？

所有数据存储在本地 `~/news/` 目录下，包括：
- `memory.md` - 兴趣画像与偏好
- `history.md` - 简报历史记录
- `sources.md` - 可信信息源列表

本技能不会将任何用户数据发送至外部服务.
### Q2：每次简报包含多少条新闻？

免费版建议单次简报 5-7 条，避免信息过载。早晨简报尤为重要，保持精简.
### Q3：如何更新兴趣偏好？

随时告诉 Agent 你想调整偏好即可：

```text
我最近对量子计算也感兴趣，加到我的关注列表里，占比15%
```

### Q4：免费版支持自定义新闻源吗？

免费版不支持自定义新闻源接入，使用预设的可信信息源。如需自定义，请升级至 PRO 版本.
### Q5：免费版与 PRO 版本的区别？

| 对比项 | 免费版 | PRO 版本 |
|---:|---:|---:|
| 用户数 | 单用户 | 多用户/多租户 |
| 兴趣领域 | 最多5个 | 无限制 |
| 信息源 | 预设列表 | 自定义接入 |
| 定时推送 | 不支持 | 支持自动推送 |
| 数据分析 | 不支持 | 阅读行为分析 |
| 导出格式 | Markdown | MD/PDF/Email |
| 多语言 | 中文为主 | 多语言支持 |

## 依赖说明

### 运行环境

- **Agent 平台**: 支持SKILL.md的任意AI Agent（Claude Code / Cursor / Codex / Gemini CLI等）
- **操作系统**: Windows / macOS / Linux
- **网络连接**: 需要可访问互联网以获取新闻
- **本地存储**: 需要 `~/news/` 目录读写权限

| 依赖项 | 类型 | 是否必需 | 获取方式 |

| 本地文件系统 | 存储 | 必需 | 操作系统提供 |

免费版基于 Markdown 指令驱动，无需额外 API Key。所有新闻获取通过 Agent 内置搜索能力完成.
```bash
# 验证本地目录可写
```

### 可用性分类

- **分类**: MD+EXEC（纯 Markdown 指令，部分功能需要 exec 命令行执行能力）
- **说明**: 基于自然语言指令驱动 Agent 获取新闻并根据用户偏好生成个性化简报，数据本地存储
- **适用规模**: 个人用户、轻量级新闻浏览场景
- **隐私等级**: 高（所有用户数据本地存储，不上传外部）
- **升级路径**: 可无缝升级至 news-tool-pro 获取多用户管理与定时推送能力

## 错误处理

| 错误场景 | 原因 | 处理方式 |
|:------|------:|:------|
| 配置错误 | 参数缺失或格式错误 | 检查依赖说明中的配置要求 |
| 运行时错误 | 运行环境不满足 | 确认运行环境符合依赖说明 |
| 网络错误 | 连接超时或不可达 | 执行ping命令测试网络连通性,检查防火墙和代理设置连接后执行ping命令测试网络连通性,检查防火墙和代理设置连接后重新执行命令，参考国内替代方案 |

## 已知限制

- 本地运行，不支持多设备同步
- 搜索结果依赖第三方搜索引擎的可用性
- 免费版有搜索次数限制，高频使用可能被限流
- 无法访问需要登录或付费墙的内容
