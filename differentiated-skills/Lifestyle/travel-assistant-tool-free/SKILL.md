# 旅行规划助手免费版

你的私人旅行规划顾问,专门帮你把旅行准备做得滴水不漏。不只是整理行程,更会主动发现你没想到的细节——就像身边有一个跑过很多地方、经验丰富的朋友帮你把关。

## 概述

本工具面向个人旅行用户,提供完整的出行前规划支持。通过自然语言对话,自动整理旅行信息总览表,主动检查护照签证、天气、货币、插头等常见遗漏,并生成个性化行李清单。完全免费,适合单人或小团队出行的快速规划。

## 使用场景

### 场景一:出国旅行前的整体准备

```bash
# 示例
用户: 我打算7月15日到20日去泰国清迈,两个人,机票订好了,酒店还没定

助手回应:
- 生成清迈旅行总览表
- 提醒7月是雨季需要带雨具

```

### 场景二:周末短途旅行快速规划

```python
# 快速生成行李清单示例
def generate_packing_list(destination, days, season):
    """生成个性化行李清单"""
    essentials = ["身份证/护照", "银行卡", "现金", "手机充电器"]
    seasonal = {
        "summer": ["防晒霜", "太阳镜", "轻薄衣物"],
        "winter": ["保暖外套", "围巾", "手套"],
        "rainy": ["雨伞", "防水鞋套", "速干毛巾"]
    }
    return {
        "目的地": destination,
        "天数": days,
        "必带证件": essentials,
        "季节用品": seasonal.get(season, []),
        "电子设备": ["充电宝", "转换插头(如需要)"]
    }

# 使用示例
packing = generate_packing_list("清迈", 6, "rainy")
print(packing)
```

## 最佳实践

- **提前规划**:建议出发前至少2周开始使用,留足签证办理时间
- **信息完整**:尽量一次性提供完整信息,提升整理准确度
- **多次更新**:每完成一项待办就更新状态,助手会重新检查遗漏
- **结合天气**:出发前3天再次查询天气,调整行李清单
- **官方确认**:签证与入境政策以官方渠道为准,助手建议仅供参考

## 常见问题

### Q1: 免费版支持多目的地行程吗?

不支持。免费版聚焦单一目的地的完整规划,如需多目的地串联行程,请使用PRO版本。

### Q2: 天气查询的数据来源是什么?

免费版通过 `wttr.in` 与 `Open-Meteo` 等公开免费API获取天气数据,无需API Key。

```bash
# 单次天气查询示例
curl -s "wttr.in/Chiang+Mai?format=%l:+%c+%t+%h+%w&lang=zh"
```

### Q3: 生成的行李清单可以导出吗?

免费版仅在对话内查看,不支持文件导出。如需PDF或Markdown导出,请升级PRO版本。

### Q4: 签证政策更新及时吗?

助手基于通用知识提供建议,签证与入境政策变化较快,请出发前通过官方渠道确认最新要求。

### Q5: 可以离线使用吗?

基础行程整理可离线进行,但天气查询与文化注意事项需要联网获取最新信息。
