# 网盘同步专家

百度网盘文件管理工具，所有操作限制在 `/apps/bdpan/` 目录内。适配所有支持 SKILL.md 的 Agent 平台。

## 核心能力

### 1. 智能下载策略
根据文件大小自动选择下载策略。先用 `bdpan ls --json` 获取文件大小，≤50MB 直接下载（Bash timeout=300000/5 分钟），>50MB 使用 nohup 后台下载并轮询进度，解决大文件超时中断痛点。下载完成后执行完整性校验（远端大小 vs 本地大小比对）。

**输入**: 用户提供智能下载策略所需的指令和必要参数。
**处理**: 解析智能下载策略的输入参数,完成核心逻辑,返回结构化响应。
**输出**: 返回智能下载策略的响应数据,包含状态码、结果和日志。

- 执行此能力时使用`input_params`参数,支持创建/查询/导出操作
### 2. 完整文件管理操作
通过 bdpan CLI 支持上传（upload）、下载（download）、转存（transfer）、分享（share）、搜索（search）、移动（mv）、复制（cp）、重命名（rename）、创建文件夹（mkdir）、删除（rm）、列表查询（ls）、登录状态（whoami）全套操作，所有操作限制在 `/apps/bdpan/` 范围内。

**输入**: 用户提供完整文件管理操作所需的指令和必要参数。
**处理**: 解析完整文件管理操作的输入参数,完成核心逻辑,返回结构化响应。
**输出**: 返回完整文件管理操作的响应数据,包含状态码、结果和日志。

- 执行此能力时使用`input_params`参数,支持创建/查询/导出操作
### 3. 增量同步机制
仅上传变化的文件，减少带宽消耗。获取远端文件列表（含大小），对比本地文件大小，仅上传有变化的文件，跳过未变化文件。支持批量上传/下载模板（for 循环遍历目录）。

**输入**: 用户提供增量同步机制所需的指令和必要参数。
**处理**: 解析增量同步机制的输入参数,完成核心逻辑,返回结构化响应。
**输出**: 返回增量同步机制的响应数据,包含状态码、结果和日志。

- 执行此能力时使用`input_params`参数,支持创建/查询/导出操作
### 4. 分享链接转存与下载
支持分享链接转存（transfer，保存到网盘不下载）和分享链接下载（download，先转存再下载到本地），支持提取码单独传入（-p）和指定转存目录（-t）。

**输入**: 用户提供分享链接转存与下载所需的指令和必要参数。
**处理**: 解析分享链接转存与下载的输入参数,完成核心逻辑,返回结构化响应。
**输出**: 返回分享链接转存与下载的响应数据,包含状态码、结果和日志。

- 执行此能力时使用`input_params`参数,支持创建/查询/导出操作
### 5. Agent 记忆备份与恢复
通过 `memory-backup.sh` 脚本安全备份/恢复 Agent 记忆文件（AGENTS.md、SOUL.md、MEMORY.md 等 7 个文件 + memory/*.md），支持 backup/list/restore 操作，恢复前自动 Safety net 备份防止误操作。

**输入**: 用户提供Agent 记忆备份与恢复所需的指令和必要参数。
**处理**: 解析Agent 记忆备份与恢复的输入参数,完成核心逻辑,返回结构化响应。
**输出**: 返回Agent 记忆备份与恢复的响应数据,包含状态码、结果和日志。

## 常见问题

**Q1：大文件下载总是超时中断怎么办？**
A：使用智能下载策略。先用 `bdpan ls --json` 获取文件大小，大于 50MB 的文件使用 nohup 后台下载模式，Agent 轮询进度。这避免了 Bash 工具超时限制导致的中断。

**Q2：下载速度很慢怎么办？**
A：下载速度受百度网盘服务端限制。非会员用户可能遇到限速。建议：确认网络连接正常；使用后台下载模式避免超时；如需更高速度，考虑开通百度网盘会员。

**Q3：分享链接转存和下载有什么区别？**
A：转存（`bdpan transfer`）将分享文件保存到你的网盘，不下载到本地。下载分享链接（`bdpan download "分享链接"`）先转存再下载到本地。如只需保存到网盘用 transfer，如需本地文件用 download。

**Q4：记忆备份支持哪些环境？**
A：支持 Agent 记忆备份的运行环境。脚本会自动检测当前环境，不支持的环境会报错退出。操作对象必须是 Agent 记忆文件（AGENTS.md、SOUL.md、MEMORY.md 等）。

**Q5：恢复记忆会覆盖当前记忆吗？**
A：会，但有安全机制保护。恢复前会列出所有将被覆盖的文件清单，要求用户显式确认，并自动将当前记忆备份到 `.backup-before-restore/` 目录，防止误操作数据丢失。

## 已知限制(补充)

1. **仅支持百度网盘**：所有操作限定在百度网盘（pan.baidu.com），不支持 OneDrive、Google Drive、阿里云盘等其他云盘服务。
2. **操作范围限定 `/apps/bdpan/`**：所有远端操作限制在 `/apps/bdpan/` 目录内，无法访问网盘根目录或其他应用目录，这是安全约束不可覆盖。
3. **下载速度受服务端限制**：非会员用户可能遇到限速，本工具无法突破百度网盘服务端限速，仅能通过后台模式避免超时。
4. **分享接口为付费服务**：`bdpan share` 需在百度网盘开放平台购买服务，未开通则分享操作会失败。
5. **install.sh 不执行本地 SHA256 校验**：安装器从百度 CDN 下载执行，完整性依赖 HTTPS 传输保护，安全敏感场景建议先手动审查安装器内容或在沙箱中执行。
6. **写操作每次须重新确认**：即使当前对话已在进行百度网盘操作，写操作（upload/download/share/mv/cp/rename/mkdir/rm）每次都必须重新确认，不能凭历史上下文静默执行。
