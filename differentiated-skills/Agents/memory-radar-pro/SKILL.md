## 性能优化策略

### 扫描性能优化

1. **增量扫描**：仅扫描新增或修改的文件，避免全量重复扫描
2. **并行扫描**：多文件并行扫描，充分利用多核CPU
3. **分块处理**：大文件分块扫描，避免内存溢出
4. **缓存机制**：已扫描文件的哈希缓存，避免重复扫描

### LLM调用优化

1. **脱敏处理**：上传前脱敏敏感信息，仅保留威胁特征
2. **批量请求**：多个威胁特征批量发送，减少API调用次数
3. **模型路由**：简单威胁用本地模式，复杂威胁用LLM增强
4. **结果缓存**：相同内容重复扫描时使用缓存结果

### 告警优化

1. **告警抑制**：相同威胁不重复告警，避免告警疲劳
2. **级别过滤**：可配置最低告警级别，过滤低级别噪音
3. **聚合通知**：短时间内多个威胁聚合为一条通知
4. **静默时段**：可配置静默时段，非工作时间不告警

### 成本控制

- 优先使用本地模式：常见威胁用模式匹配，零API成本
- LLM增强按需启用：仅对高风险文件启用LLM扫描
- 增量扫描优先：日常监控使用增量模式，节省80%扫描时间
- 规则库精简：定期清理无效规则，提升扫描效率

## FAQ

### Q1：免费版与专业版有什么区别？

免费版提供本地模式扫描、单文件与全量扫描、基础威胁检测（模式匹配，准确率85%）、五级安全评级。专业版解锁LLM增强扫描（准确率95%+）、批量定时监控、自动隔离、威胁情报更新、合规报告生成、多维度深度检测、告警通道集成七大高级功能。此外提供多角色场景指南、性能优化策略和多平台集成示例。

### Q2：LLM增强扫描会上传我的记忆内容吗？

会，但仅上传脱敏后的内容。专业版在上传前会自动脱敏敏感信息（如API Key、密码、个人信息），仅保留威胁特征供LLM分析。脱敏规则可配置，确保符合组织的数据安全策略。如对数据安全有严格要求，可仅使用本地模式。

### Q3：自动隔离会影响Agent正常运行吗？

不会。自动隔离仅对检测到的威胁行进行脱敏处理（替换为隔离标记），不影响其他内容。隔离前自动备份原始文件，可随时恢复。建议对CRITICAL级别启用自动隔离，HIGH级别手动确认后隔离。

### Q4：威胁情报规则库如何更新？

规则库通过社区共享更新，可配置每日自动更新。更新内容包括最新的提示注入模式、恶意指令特征、凭据泄露模式等。组织也可添加自定义规则，检测特定于自身的威胁。规则库版本可追溯，便于审计。

### Q5：合规报告包含哪些内容？

合规报告包含：扫描时间与范围、检测到的威胁清单（含文件位置、行号、威胁级别、描述）、已采取的响应措施（隔离/告警/人工处理）、合规性评估结论、改进建议。支持PDF和HTML格式，可配置保留天数（默认90天，金融场景建议180天+）。

### Q6：批量定时监控如何配置？

通过`schedule-scan.sh`脚本创建cron定时任务，默认每日下午3点执行。可通过`--schedule`参数自定义cron表达式。监控范围可配置为全部记忆文件或指定目录。增量扫描模式仅扫描新增或修改的文件，节省扫描时间。

### Q7：告警通道支持哪些平台？

支持邮件通知、Webhook（兼容企业微信、钉钉、Slack、飞书等）、系统通知、日志文件四种告警通道。可配置最低告警级别（默认MEDIUM），过滤低级别噪音。支持告警抑制（相同威胁不重复告警）和聚合通知（短时间内多个威胁聚合为一条）。

### Q8：如何检测变体的提示注入攻击？

专业版LLM增强扫描通过语义理解检测变体提示注入：(1) Unicode替换检测（如全角字符替换）；(2) 分隔符混淆检测；(3) 语义意图分析（理解指令真实意图）；(4) 上下文关联检测（结合上下文判断威胁）。这些是本地模式无法检测的高级威胁。

### Q9：扫描会影响Agent性能吗？

不会。扫描在后台异步执行，不阻塞Agent正常运行。增量扫描模式仅扫描新增文件，对系统资源占用极小。LLM增强扫描采用分块处理，避免大文件导致内存溢出。建议在低峰期执行全量扫描。

### Q10：可以与现有安全工具集成吗？

可以。专业版支持与SIEM系统（Splunk、ELK等）、GRC平台、合规系统、告警平台集成。通过JSON格式输出和Webhook通知，可实现与大多数安全工具的对接。详细集成方式参见"多平台集成示例"章节。

### Q11：医疗/金融行业的特殊规则如何配置？

专业版支持自定义规则，可针对行业特性添加检测规则。例如医疗行业可添加患者信息泄露检测规则（PHI模式匹配），金融行业可添加交易信息泄露检测规则。规则使用正则表达式或LLM语义匹配，可灵活配置。
