# 卖货文案匠

## 使用场景

解决内容营销中"文案干瘪无转化"的核心痛点。支持以下场景:

1. **新品上市推广**: 提炼产品核心卖点,生成多平台种草文案
2. **直播带货预热**: 生成直播前引流文案和直播间促单话术
3. **电商详情页优化**: 为商品详情页生成高转化率描述文案
4. **社交媒体营销**: 为小红书/抖音/微信等平台生成差异化营销内容
5. **活动促销文案**: 结合紧迫感和社交证明,生成限时活动文案

---

## 不适用场景

以下场景卖货文案匠不适合处理：

- 非法营销手段
- 虚假宣传
- 恶意竞争


## 工作流

### 步骤1: 卖点提炼 (FAB法则)

接收内容主题和品牌信息,提炼3-5个核心卖点。

**FAB法则说明**:
- **Feature (特性)**: 产品具备的客观属性
- **Advantage (优势)**: 特性带来的竞争优势
- **Benefit (利益)**: 用户能获得的实际价值

输入参数: `主题` + `品牌信息`
输出: 3-5个FAB结构化卖点

### 步骤2: 情绪钩子生成

选择情绪钩子类型,生成营销文案框架。

**支持的6种情绪钩子类型**:

| 钩子类型 | 标识 | 适用场景 | 示例 |
|:---------|:-----|:---------|:-----|
| 痛点共鸣 | pain_resonance | 解决用户痛点 | "你是不是也经常加班写不出文案?" |
| 好奇悬念 | curiosity_suspense | 引发好奇心 | "这个方法让文案转化率翻了3倍..." |
| 紧迫感 | urgency | 限时促销 | "最后3天,错过再等一年" |
| 社交证明 | social_proof | 建立信任 | "10万+用户都在用的文案神器" |
| 对比反差 | contrast | 凸显优势 | "别人写文案要2小时,你只需要2分钟" |
| 权威背书 | authority | 专业可信 | "文案大师都在用的FAB公式" |

### 步骤3: 平台适配 + CTA注入

根据目标平台特性调整文案风格,并注入对应的CTA话术。

**平台适配规则**:

| 平台 | 文案风格 | 字数建议 | CTA建议 |
|:-----|:---------|:---------|:--------|
| 小红书 | 活泼种草,emoji丰富 | 300-500字 | 私信/关注 |
| 抖音 | 口语化,直接有力 | 50-100字 | 私信/下单 |
| 微信公众号 | 深度种草,逻辑清晰 | 800-1500字 | 关注/加群 |
| 微博 | 简洁观点,话题标签 | 140字以内 | 关注 |
| 电商平台 | 卖点突出,促转化 | 200-400字 | 下单 |

**CTA类型**:
- `follow` (关注): 引导用户关注账号
- `dm` (私信): 引导用户私信咨询
- `order` (下单): 引导用户直接下单购买
- `group` (加群): 引导用户加入社群

### 步骤4: 输出完整营销文案

整合卖点、情绪钩子、平台适配文案和CTA,输出完整结构化结果。

## 输入格式

```json
{
  "topic": "AI代写文案服务",
  "brand_info": "品牌名:XX, slogan:让AI帮你写",
  "platform": "xiaohongshu",
  "hook_type": "pain_resonance",
  "cta_type": "order"
}
```

| 字段 | 类型 | 必填 | 说明 |
|:-----|:-----|:----:|:-----|
| topic | string | 是 | 内容主题/产品名称 |
| brand_info | string | 否 | 品牌信息(品牌名+slogan) |
| platform | string | 是 | 目标平台(xiaohongshu/douyin/wechat/weibo/ecommerce) |
| hook_type | string | 否 | 情绪钩子类型,默认pain_resonance |
| cta_type | string | 否 | CTA类型,默认follow |

## 异常处理

| 异常场景 | 处理方式 | 降级策略 |
|:---------|:---------|:---------|
| 主题为空 | 返回success=false, error="主题不能为空" | 无法降级,必须提供主题 |
| 平台不支持 | 降级为通用文案,标注warning | 使用通用文案模板,不针对平台优化 |
| 品牌信息缺失 | 生成无品牌版本,标注warning | 省略品牌相关内容,聚焦产品卖点 |
| 钩子类型无效 | 默认使用pain_resonance | 标注warning,使用默认钩子类型 |
| CTA类型无效 | 默认使用follow | 标注warning,使用默认CTA类型 |
