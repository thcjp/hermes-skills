## 概述

本版本适合个人文档作者、学生及知识工作者,快速将文字描述转化为可视化图表,提升内容表达效果。

## 使用场景

### 场景一:文档配图生成

用户在编写技术文档时,需要一张系统架构图说明组件关系。

```bash
# 认证(首次使用)
anygen auth login --no-wait

# 生成架构图
anygen smart_draw "一个微服务架构图:前端通过API网关访问3个微服务(用户服务、订单服务、支付服务),每个服务连接独立的数据库,消息队列连接订单服务与支付服务"
```

### 场景二:学习笔记可视化

学生希望将一个算法流程转化为流程图,便于理解与复习。

```bash
# 生成算法流程图
anygen smart_draw "快速排序算法流程图:开始→选择基准值→分区(小于基准/大于基准)→递归排序→合并结果→结束"
```

### 场景三:项目流程说明

项目经理需要一张项目流程图说明开发流程。

```bash
# 生成项目流程图
anygen smart_draw "软件开发流程图:需求分析→设计→编码→测试→部署→维护,测试失败回到编码,维护完成后结束"
```

```bash
# - 组织结构图:描述层级与归属

# - 时序图:描述交互时序
```

## 最佳实践

1. **描述要具体**:明确节点名称与关系,如"用户服务连接用户数据库"而非"服务连数据库"
2. **关系方向清晰**:用"访问"、"连接"、"调用"等明确关系动词
3. **层级结构分明**:描述时按层级组织,先总后分,便于理解结构
4. **认证失败重试**:遇到认证错误先运行 `anygen auth login --no-wait`,浏览器完成授权后重试
5. **复杂图分步生成**:过于复杂的图可分多张生成,避免单图信息过载
6. **结果保存**:生成的图表会返回 URL 或文件路径,及时保存到本地

## 常见问题

### Q1:认证失败怎么办?
A:运行 `anygen auth login --no-wait`,在浏览器完成授权后重试。也可使用 `--api-key` 方式或设置 `ANYGEN_API_KEY` 环境变量。

### Q2:支持哪些图表类型?
A:支持流程图、架构图、组织结构图、思维导图、时序图等多种类型,通过自然语言描述自动识别。

### Q3:生成的图表是什么格式?
A:服务端渲染输出高质量图像,通常为 PNG 或 SVG 格式。具体格式取决于 AnyGen 服务返回。

### 已知限制
A:免费版有调用次数限制,适合个人日常使用。如需批量生成或团队协作,请使用 PRO 版。

### Q5:需要安装工作流技能吗?
A:首次使用需安装 `anygen-workflow-generate` 技能,通过 `anygen skill install` 命令安装。已安装则可直接使用。

### Q6:可以生成中文图表吗?
A:可以。自然语言描述支持中文,生成的图表中的文字也会使用中文。
