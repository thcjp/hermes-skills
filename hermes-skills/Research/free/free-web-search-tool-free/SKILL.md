# 免费网页搜索工具

## 概述

免费网页搜索工具是一款面向个人用户的轻量级联网搜索解决方案。通过 Playwright 驱动无头浏览器，智能路由 Bing 国内版与 DuckDuckGo，为中文用户提供精准、快速的搜索体验。无需任何 API Key 配置，安装依赖后即可立即使用。

## 最佳实践

### 搜索关键词优化

| 场景 | 推荐写法 | 不推荐写法 |
| --- | --- | --- |
| 时效新闻 | `2026年04月 科技新闻` | `最近有什么科技新闻` |
| 技术教程 | `Python 异步编程 2026` | `我想学 Python 异步编程` |
| 本地内容 | `广东东莞 今日天气` | `东莞今天天气怎么样啊` |
| 官方信息 | `华为云 ModelArts 文档` | `华为云那个 ModelArts 怎么看` |

### 关键词撰写原则

1. **简洁精准**：2-5 个核心词组合，避免长句、反问句
2. **限定明确**：需要时效性/地区内容时加上对应限定词
3. **格式正确**：中文关键词 + 数字/英文限定词
4. **避免口语化**：使用搜索语法而非自然语言提问

### 隐私保护建议

- 不需要正文抓取时，保持 `--full=0`，仅访问搜索引擎
- 定期清理浏览器上下文，避免 Cookie 累积
- 在公共网络环境使用时，注意搜索内容敏感性

## 常见问题

### 搜索返回空结果

```bash
# 检查网络连通性
ping cn.bing.com

# 尝试切换搜索引擎
python scripts/web_search.py "测试关键词" --region=intl
```

可能原因及解决方案：
- 网络限流：等待几分钟后重试
- 地区误判：使用 `--region` 强制指定引擎
- 关键词过长：精简到 2-5 个核心词

### 浏览器启动失败

```bash
# 重新安装 Playwright 依赖
pip install --upgrade playwright
playwright install chromium

# 验证安装
python -c "from playwright.sync_api import sync_playwright; print('OK')"
```

### 正文抓取失败

部分站点使用重度 JS 渲染或返回非 HTML 内容，免费版不做特殊处理。建议：
- 减少抓取数量，仅抓取 `--full=1` 或 `--full=2`
- 尝试更换搜索关键词，获取不同来源结果

### IP 探测失败

工具依次探测 4 个公开 IP 服务，若全部失败则默认使用 DuckDuckGo。可通过 `--region=cn` 或 `--region=intl` 手动覆盖。
