## 代码示例 ()

```` | 代码块 | 三反引号包裹 |
| `**粗体**` | 粗体 | `**重要**` |
| `*斜体*` | 斜体 | `*强调*` |
| `~~删除~~` | 删除线 | `~~废弃~~` |
| `- [ ] 任务` | 待办任务 | `- [ ] 待办` |
| `- [x] 任务` | 已完成任务 | `- [x] 完成` |
| `\|表格\|` | 表格 | 用竖线分隔 |
| `---` | 分隔线 | 三横线 |

**Obsidian特有语法**：

| 语法 | 说明 | 示例 |
|------|------|------|
| `[[笔记名]]` | wikilink（内部链接） | `[[会议记录]]` |
| `[[笔记名\|显示名]]` | 带别名的链接 | `[[项目A\|项目]]` |
| `[[笔记名#标题]]` | 链接到标题 | `[[笔记#概述]]` |
| `![[笔记名]]` | 嵌入笔记 | `![[每日总结]]` |
| `![[图片.png]]` | 嵌入图片 | `![[截图.png]]` |
| `%%注释%%` | 注释（不显示） | `%%这是注释%%` |
| `$公式$` | 行内数学公式 | `$E=mc^2$` |
| `$$公式$$` | 块级数学公式 | `$$\int_0^1 x dx$$` |

**配置优先级**（新手建议按顺序配置）：

| 优先级 | 配置项 | 路径 | 关键设置 |
|--------|--------|------|----------|
| 1 | 文件与链接 | 设置 → 文件与链接 | 自动更新内部链接：开启 |
| 2 | 编辑器 | 设置 → 编辑器 | 默认视图、字体大小 |
| 3 | 外观 | 设置 → 外观 | 主题、颜色模式 |
| 4 | 核心插件 | 设置 → 核心插件 | 启用基础插件 |
| 5 | 快捷键 | 设置 → 快捷键 | 自定义常用操作 |

**编辑器配置详解**：

| 配置项 | 推荐值 | 说明 |
|--------|--------|------|
| 默认视图 | 编辑视图 | 新手友好，所见即所得 |
| 严格换行 | 关闭 | 避免意外换行 |
| 显示行号 | 开启 | 便于定位 |
| 拼写检查 | 关闭 | 中文用户建议关闭 |
| 字体大小 | 16-18 | 根据屏幕调整 |
| 缩进 | 2空格 | 标准缩进 |
| 自动配对括号 | 开启 | 提升编辑效率 |
| 智能缩进 | 开启 | 自动对齐 |

**文件与链接配置详解**：

| 配置项 | 推荐值 | 说明 |
|--------|--------|------|
| 新笔记位置 | 当前文件夹 | 便于组织 |
| 链接格式 | 相对路径 | 便于vault迁移 |
| 默认新文件扩展名 | .md | 标准扩展名 |
| 自动更新内部链接 | 开启 | **重要！**移动笔记时自动更新链接 |
| 使用Wiki链接 | 开启 | Obsidian原生格式 |

```

## 代码示例 (text)

```text
步骤1：打开设置
- 快捷键：Ctrl+,（Windows）/ Cmd+,（macOS）
- 或点击左下角齿轮图标

步骤2：编辑器配置
路径：设置 → 编辑器

推荐配置：
- 默认视图：编辑视图（新手友好）
- 严格换行：关闭（避免意外换行）
- 显示行号：开启（便于定位）
- 拼写检查：关闭（中文用户）
- 字体大小：16-18（根据屏幕调整）
- 缩进：2空格

步骤3：文件与链接配置
路径：设置 → 文件与链接

推荐配置：
- 新笔记位置：当前文件夹（便于组织）
- 链接格式：相对路径（便于迁移）
- 默认新文件扩展名：.md
- 启用"自动更新内部链接"：开启（重要！）

步骤4：外观配置
路径：设置 → 外观

推荐配置：
- 主题：默认（新手建议）/ Minimal / Blue Topaz
- 基础颜色模式：浅色或深色（根据偏好）
- 字体：系统默认或安装等宽字体（如JetBrains Mono）
- 界面字体大小：16

步骤5：核心插件配置
路径：设置 → 核心插件

推荐启用：
- 文件列表（必须）
- 搜索（必须）
- 标签面板（推荐）
- 大纲（推荐）
- 页面预览（推荐）
- 日记（推荐）
- 模板（推荐）
- 星标（推荐）
```
