# 详细参考 - zh-file-organizer-pro

> 本文件从SKILL.md拆分而来，包含详细代码示例和扩展章节。

## 代码示例 (yaml)

```yaml
整理器:
  版本: "1.0"
  语言: 中文
  日志文件: ~/整理器日志.log

扫描:
  路径:
    - ~/下载/
    - ~/桌面/
    - ~/文档/
  定时: "0 9 * * *"
  增量: true
  排除: [".DS_Store", "Thumbs.db", "desktop.ini", "*.tmp", "~$*"]

去重:
  启用: true
  策略: 保留最新
  哈希算法: sha256
  最小文件: 1KB
  扫描路径: [~/文档/, ~/下载/]

内容感知:
  启用: true
  规则: rules/content-rules.yaml
  最低置信度: 0.8
  待确认目录: 待确认/

监控:
  启用: true
  路径: [~/下载/]
  自动整理: true
  防抖秒数: 5
  守护进程: true

报告:
  格式: [html, pdf]
  输出: ~/报告/整理器/
  保留天数: 90
  月度汇总: true

定时任务:
  - 名称: 每日整理
    cron: "0 9 * * *"
    路径: [~/下载, ~/桌面]
    操作: [整理, 去重]
  - 名称: 每周深度清理
    cron: "0 10 * * 0"
    路径: [~/文档/]
    操作: [整理, 去重, 内容分类]

团队:
  共享规则: rules/团队规则.yaml
  同步周期: "0 0 * * 0"
  审计: 每季度
```

## 代码示例 (yaml)

```yaml
内容感知规则:
  - 名称: 合同文档识别
    类型: 内容感知
    匹配:
      内容关键词: [合同, 协议, 甲方, 乙方, 签约, 违约]
      文件类型: [pdf, docx]
      最低置信度: 0.8
    归入: 文档/合同/

  - 名称: 发票财务识别
    类型: 内容感知
    匹配:
      内容关键词: [发票, 收据, 账单, 报销, 金额, 税额]
      文件类型: [pdf, jpg, png]
    归入: 文档/财务/

  - 名称: 项目文档识别
    类型: 上下文关联
    匹配:
      同目录关键词: [README, 需求文档, 设计文档]
      文件类型: [md, docx, pdf]
    归入: 文档/项目/

  - 名称: 证件识别
    类型: 内容感知
    匹配:
      内容关键词: [身份证, 护照, 驾驶证, 营业执照]
      文件类型: [jpg, png, pdf]
      最低置信度: 0.9
    归入: 证件/ (加密存储)
    加密: true
```
