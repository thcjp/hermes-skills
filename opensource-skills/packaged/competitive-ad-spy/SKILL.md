# 竞品广告侦察兵

从广告库提取并分析竞品广告。不仅看竞品投了什么广告,更分析为什么这样投,为你的投放提供情报。

## 适用场景

| 场景 | 输入 | 输出 |
|:-----|:-----|:-----|
| 竞品调研 | 竞品名称+目标品类 | 完整广告情报报告(数量/平台/时长/创意) |
| 创意灵感 | 自有产品定位+目标受众 | 从竞品广告提炼的创意方向与素材模板 |
| 差异化定位 | 自有卖点+3-5个竞品清单 | 卖点矩阵+差异化机会清单 |
| 投放优化 | 自有广告数据+竞品广告 | 对标分析报告+改进建议 |
| 新市场进入 | 目标市场+品类 | 当地广告生态概览+竞品投放策略 |

### 不适用于
- 实时广告投放出价与预算调整(非投放执行工具)
- 竞品内部销售数据/财务数据获取(广告库不提供)
- 竞品广告ROI/转化率还原(广告库不提供效果数据)
- 自动化广告创意生成(请使用copywriting-master)
- 广告素材设计制作(请使用设计工具)

## 核心能力

1. **多平台广告库情报提取**:覆盖Meta Ad Library、TikTok Creative Center、Google Ads Transparency,国内补充巨量引擎创意中心、腾讯广告创意库,支持手动输入广告素材截图/链接/文案
2. **视觉与文案深度分析**:钩子类型识别(前3秒)、色彩主调、构图模式、视频结构拆解、文案卖点与情感诉求分类
3. **受众与策略推断**:基于素材推断目标受众画像、信息策略(主打/弱化)、创意方向(UGC/专业制作)、投放节奏与变体测试力度
4. **差异化机会挖掘**:卖点矩阵对比、空白点识别、行动建议输出
5. **情报报告结构化输出**:统一目录结构(intel-report/ads.csv/creative-analysis/differentiation/assets)

## 使用流程

### Step 1: 竞品识别与范围界定
1. 用户提供品类或竞品名称清单
2. 分类:直接竞品(同品类/同价位/同受众)与间接竞品(解决同一问题的不同方案)
3. 确定广告库:海外用Meta/TikTok/Google Transparency,国内用巨量引擎创意中心/腾讯广告创意库
4. 筛选条件:活跃广告/投放时长/平台/地域

### Step 2: 广告数据提取
1. 用户手动输入或粘贴广告库URL/截图/文案
2. 提取字段:图片/视频/轮播、标题/正文/CTA、落地页URL、投放平台、开始日期、状态、变体数量
3. 受众信号(如可获取):定位地区、年龄/性别范围

### Step 3: 创意分析
1. **视觉分析**:视觉风格、色彩主调、构图模式、视频结构(钩子/展示/证明/CTA)
2. **文案分析**:钩子类型(问题/数据/故事/反常识)、核心卖点、情感诉求、CTA用词
3. **受众推断**:基于素材推断目标受众、痛点假设、语言风格匹配

### Step 4: 策略解读
1. 信息策略:主打卖点(USP)、弱化方向(可能弱点)、定位方式
2. 创意方向:UGC占比、专业制作占比、变体测试力度
3. 投放策略:多平台vs单平台、长期vs短期、变体频率

### Step 5: 情报报告生成
1. 竞品广告概览(数量/平台/时长)
2. 创意模式总结
3. 卖点矩阵(各竞品主打对比)
4. 差异化机会(竞品未覆盖空白)
5. 行动建议(基于情报的投放建议)

## 分析维度

| 维度 | 分析内容 |
|:-----|:---------|
| 钩子 | 前3秒/首句如何抓注意力 |
| 卖点 | 主打什么价值主张 |
| 证据 | 如何证明产品有效 |
| 情感 | 调动什么情绪 |
| CTA | 如何引导行动 |
| 受众 | 面向谁说话 |
| 频率 | 创意更新节奏 |
| 平台 | 哪个平台投入最多 |

## 错误处理

| 错误场景 | 原因 | 处理方式 |
|:---------|:-----|:---------|
| 广告库无法访问 | 海外平台网络限制 | 改用巨量引擎创意中心/腾讯广告创意库,或手动输入广告截图 |
| 广告数据缺失 | 竞品未在广告库展示/已下架 | 标注"未公开",从官方公众号/小红书/视频号补充素材 |
| 素材识别失败 | 截图分辨率低/非中文 | 提示用户重传高清原图,提供OCR降级方案 |
| 竞品信息不足 | 用户仅提供品牌名 | 主动询问品类、地域、时间范围,逐步引导补全 |
| 分析结果空泛 | 输入素材少于3条 | 提示用户补充至5-10条广告样本,标注结论置信度 |
| 平台覆盖不全 | 仅单一平台数据 | 标注平台局限性,建议跨平台验证 |

### 案例5: 投放优化对标分析

**输入**:
```

- 月预算: ¥50,000

- 平均CPC: ¥3.8（行业均值¥2.5）
- 转化率: 3.5%
- 主平台: 朋友圈(60%) + 抖音(40%)

竞品: 花西子、完美日记
品类: 国货美妆
目的: 找到CTR低、CPC高的原因
```

**LLM生成输出**:
```markdown

# 投放优化对标分析报告

## 一、核心指标对比
| 指标 | 自有 | 花西子 | 完美日记 | 行业均值 | 诊断 |
|:-----|:-----|:-------|:---------|:---------|:-----|
| CTR | 1.2% | 3.8% | 4.2% | 2.5% | 严重偏低 |
| CPC | ¥3.8 | ¥1.8 | ¥2.1 | ¥2.5 | 偏高52% |
| 转化率 | 3.5% | 5.8% | 6.2% | 4.0% | 偏低 |
| CPM | ¥45 | ¥38 | ¥42 | ¥40 | 略高 |

## 二、问题诊断

### 问题1: CTR严重偏低（1.2% vs 行业2.5%）
**根因分析**:
- **钩子弱**: 自有广告前3秒为产品展示，竞品为痛点场景
  - 花西子: "底妆卡粉？" → 痛点共鸣
  - 完美日记: "3秒换脸" → 反差冲击
  - 自有: "新品上市" → 平铺直叙
- **视觉同质化**: 自有广告80%为产品特写，竞品UGC占比60%+
- **受众过宽**: 定位18-50岁女性，竞品精准定位22-30岁

### 问题2: CPC偏高（¥3.8 vs 行业¥2.5）
**根因分析**:
- CTR低导致CPC高（CPC = CPM / 1000 / CTR）
- 竞价激烈时段投放（早8-10点），竞品避开高峰
- 人群包未优化，重复曝光率高（35%）

### 问题3: 转化率偏低（3.5% vs 行业4.0%）
**根因分析**:
- 落地页加载慢（3.2秒，竞品<2秒）
- 落地页与广告创意不一致（广告讲成分，落地页讲价格）
- 缺乏紧迫感元素（限时/限量）

### P1: 创意组合优化（预计CTR +0.8pp）
| 创意类型 | 当前占比 | 建议占比 | 依据 |
|:---------|:---------|:---------|:-----|
| 产品特写 | 80% | 30% | 竞品UGC更有效 |
| UGC场景 | 10% | 50% | 完美日记验证 |
| KOL背书 | 10% | 20% | 花西子验证 |

### P2: 受众精准化（预计CPC -30%）
- 收窄年龄: 18-50 → 22-30（对标竞品）
- 兴趣细分: 美妆兴趣 → "成分党"/"国货爱好者"
- 排除已购买用户（当前重复曝光35%）

### P3: 落地页优化（预计转化率 +1.5pp）
- 加载速度: 3.2s → <2s（图片懒加载+CDN）
- 创意一致: 落地页首屏与广告创意呼应
- 紧迫感: 添加"限时8折"+"仅剩XX件"
```

## 常见问题

### Q1: 没有海外广告库访问权限怎么办?
A: 国内用户可使用巨量引擎创意中心(creative.oceanengine.com)、腾讯广告创意库、百度营销创意中心、小红书蒲公英平台等国内替代方案。Skill已内置中外广告库对照表,可自动切换至国内源。如需分析海外竞品,可让用户提供广告截图/链接手动输入。

### Q2: 分析需要多少广告样本才可靠?
A: 建议每个竞品至少5-10条活跃广告样本,涵盖图片/视频/轮播多种格式。样本少于3条时,结论置信度会显著下降,报告中会明确标注"样本不足,仅供参考"。对于多平台竞品,每个平台至少3条样本。

### Q3: 如何处理竞品的A/B测试变体?
A: 同一产品的多个变体视为一个"创意测试组",而非独立广告。分析时聚焦变体间的差异点(钩子/CTA/视觉),识别测试假设(如:测试问题型vs数据型钩子),总结竞品的测试方向而非逐条分析。

### Q4: 广告库数据有时效性吗?
A: 是。广告库通常仅展示活跃广告与最近6-12个月的历史。建议每次分析标注数据采集日期,长期跟踪的竞品建议每季度更新一次情报报告,存档至 `output/{competitor}/archive/` 目录。

## 已知限制

- 广告库仅展示广告素材与基础投放信息,无法获取实际投放预算、ROI、转化率等效果数据
- 受众定位信息在多数广告库中已脱敏,受众推断基于素材内容,存在主观性
- 部分竞品可能未在广告库公开(如:仅投放私域/达人种草),导致情报不完整
- 创意分析结论依赖LLM理解能力,对非主流语言/小众品类的分析准确度可能下降
- 海外广告库在国内访问受限,需配合国内替代方案使用
- 不适用于政治广告、公益广告等非商业广告分析
