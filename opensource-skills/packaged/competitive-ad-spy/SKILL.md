# 竞品广告侦察兵

从广告库提取并分析竞品广告。不仅看竞品投了什么广告,更分析为什么这样投,为你的投放提供情报。

## 使用场景

| 场景 | 触发条件 | 说明 |
|:-----|:---------|:-----|
| 竞品调研 | 了解竞品广告策略 | 全面广告情报收集 |
| 创意灵感 | 广告创意枯竭 | 从竞品广告找方向 |
| 差异化定位 | 找差异化卖点 | 对比竞品找空白 |
| 投放优化 | 自有广告效果不佳 | 对标竞品找改进点 |
| 新市场进入 | 进入新市场前 | 了解当地广告生态 |

## 工作流

### 1. 竞品识别

1. **直接竞品**:同品类/同价位/同受众
2. **间接竞品**:解决同一问题的不同方案
3. **广告库搜索**:
   - Meta Ad Library(meta.com/ads/library)
   - TikTok Creative Center
   - Google Ads Transparency Center
4. **筛选条件**:活跃广告/投放时长/平台

### 2. 广告数据提取

1. **广告素材**:
   - 图片/视频/轮播
   - 广告文案(标题/正文/CTA)
   - 落地页 URL
2. **投放信息**:
   - 投放平台(Facebook/Instagram/TikTok/Google)
   - 投放时长(开始日期-现在)
   - 投放状态(活跃/暂停)
   - 多版本变体
3. **受众信号**(如可获取):
   - 定位地区
   - 年龄/性别范围

### 3. 创意分析

1. **视觉分析**:
   - 视觉风格(真人/动画/产品/文字)
   - 色彩主调
   - 构图模式(前后对比/使用场景/UGC)
   - 视频结构(钩子/展示/证明/CTA)
2. **文案分析**:
   - 钩子类型(问题/数据/故事/反常识)
   - 核心卖点(价格/质量/便利/社交)
   - 情感诉求(恐惧/渴望/归属/成就)
   - CTA 用词
3. **受众推断**:
   - 基于素材推断目标受众
   - 痛点假设
   - 语言风格匹配

### 4. 策略解读

1. **信息策略**:
   - 主打什么卖点(USP)
   - 忽略什么(可能是弱点)
   - 如何定位自己(vs 竞品/品类)
2. **创意方向**:
   - UGC(用户生成内容)为主?
   - 专业制作为主?
   - 创意变体数量(测试力度)
3. **投放策略**:
   - 多平台 vs 单平台
   - 长期投放 vs 短期测试
   - 广告变体频率

### 5. 情报报告

1. **竞品广告概览**:数量/平台/时长
2. **创意模式总结**:常见模式与套路
3. **卖点矩阵**:各竞品主打卖点对比
4. **差异化机会**:竞品未覆盖的空白
5. **行动建议**:基于情报的投放建议

## 分析维度

| 维度 | 分析内容 |
|:-----|:---------|
| 钩子 | 前3秒/首句如何抓注意力 |
| 卖点 | 主打什么价值主张 |
| 证据 | 如何证明产品有效 |
| 情感 | 调动什么情绪 |
| CTA | 如何引导行动 |
| 受众 | 面向谁说话 |
| 频率 | 创意更新节奏 |
| 平台 | 哪个平台投入最多 |

## 输出规范

- 情报报告:`output/{competitor}/intel-report.md`

- 素材存档:`output/{competitor}/assets/`
