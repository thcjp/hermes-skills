# 推特爆款优化器

基于 Twitter(X)开源推荐算法的公开洞察,分析并优化推文内容,最大化触达与互动率。

## 使用场景

| 场景 | 触发条件 | 说明 |
|:-----|:---------|:-----|
| 推文重写 | 已有草稿推文 | 基于算法偏好重写,提升互动预测 |
| 发布时机 | 准备发布 | 推荐最佳发布时间窗口 |
| 话题策略 | 需要标签建议 | 推荐高流量低竞争标签组合 |
| 线程优化 | 多推文线程 | 优化线程结构和钩子 |
| 账号诊断 | 账号互动低迷 | 分析近期推文,找出改进点 |

## 工作流

### 1. 推文分析

1. **接收推文内容**:文本、图片(可选)、发布时间、目标受众
2. **算法因子评分**:基于 Twitter 算法公开因子打分
   - 互动权重(回复/转发/点赞/书签的相对权重)
   - 内容负面信号(过多标签/外链/敏感词)
   - 用户亲和度(作者与潜在读者的互动历史)
   - 新鲜度(发布后的时间衰减)
3. **输出评分卡**:当前推文的算法友好度评分(0-100)

### 2. 推文重写

1. **钩子优化**:首行用问题/数据/反常识/故事抓住注意力
2. **长度控制**:短推文(280字符内)优先,长推文用线程
3. **互动诱导**:结尾设计提问/投票/引发回复的钩子
4. **标签策略**:1-2 个高相关标签,避免标签堆砌
5. **媒体建议**:建议配图/GIF/视频,多媒体推文互动率更高
6. **生成 3-5 个重写版本**:不同风格供 A/B 测试

### 3. 发布时机

1. **受众活跃分析**:基于目标受众时区和生活习惯
2. **最佳时间窗口**:工作日 9-11点、13-15点、18-20点(受众时区)
3. **避开竞争时段**:避免与大V同时发布被淹没
4. **发布日历**:规划一周发布节奏

### 4. 线程结构优化

1. **首推钩子**:线程第一条必须强钩子,决定打开率
2. **节奏控制**:每条推文一个完整观点,避免断句
3. **高潮前置**:核心价值在前 3 条,不要埋太深
4. **结尾行动号召**:最后一条引导关注/转发/访问链接

### 5. 互动预测与复盘

1. **互动预测**:基于历史数据和算法因子预测互动区间
2. **发布后追踪**:T+1h/T+24h/T+7d 互动数据对比
3. **偏差分析**:预测 vs 实际,校准预测模型
4. **改进建议**:基于复盘数据优化下一篇

## 算法洞察(基于 Twitter 开源信息)

- **书签权重高**:书签的算法权重高于点赞,鼓励用户书签
- **回复权重**:回复互动的权重高于点赞
- **外链降权**:推文含外链会降低触达,建议回复中放链接
- **多媒体加成**:图片/视频/GIF 提升互动率
- **标签限制**:过多标签(>3)被视为垃圾信号
- **新鲜度**:发布后 24 小时内是黄金传播期
