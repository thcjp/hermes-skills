# 科研助手

基于 134 个科学技能库,提供从文献检索到论文发表的全流程科研支持。从生物信息学到药物发现,从数据分析到论文写作,多领域跨学科研究助手。

## 使用场景

| 场景 | 触发条件 | 说明 |
|:-----|:---------|:-----|
| 药物发现 | 新药研发 | 靶点识别/虚拟筛选/分子对接/ADMET |
| 生物信息学 | 基因组/转录组分析 | 序列分析/单细胞RNA-seq/基因调控网络 |
| 蛋白质研究 | 蛋白质结构与功能 | 结构预测/功能注释/蛋白工程 |
| 医学影像 | 医学图像分析 | DICOM处理/病理切片/放射学 |
| 材料科学 | 材料设计与分析 | 晶体结构/相图/计算化学 |
| 文献综述 | 研究背景调研 | 文献检索/引用管理/综述写作 |
| 论文写作 | 学术写作 | 论文结构/图表/参考文献/格式 |
| 同行评审 | 审稿支持 | 系统性评审/批判性分析/改进建议 |
| 基金申请 | 项目资助 | 假设生成/研究计划/预算/影响陈述 |

## 工作流

### 1. 文献检索与综述

1. **文献检索**
   - PubMed/Google Scholar/arXiv 关键词搜索
   - 引用追踪(前向/后向)
   - 筛选标准制定(纳入/排除)
   - 文献管理(Zotero/Mendeley 集成)
2. **文献分析**
   - 信息提取(方法/结果/结论)
   - 质量评估(随机对照/偏倚风险)
   - 主题聚类与知识图谱
   - 研究空白识别
3. **综述写作**
   - 综述结构(背景/方法/结果/讨论/结论)
   - 叙事 vs 系统性综述
   - 引用格式(APA/Nature/IEEE)
   - 图表制作(流程图/森林图)

### 2. 数据分析与可视化

1. **统计分析**
   - 描述性统计(均值/中位数/标准差)
   - 推断统计(t检验/ANOVA/卡方/回归)
   - 多变量分析(PCA/聚类/因子分析)
   - 生存分析(Kaplan-Meier/Cox)
2. **生物信息学分析**
   - 序列分析(BLAST/多序列比对)
   - 单细胞RNA-seq(Scanpy:QC/聚类/差异表达)
   - 基因调控网络(Arboreto/GRNBoost2)
   - 通路富集(KEGG/Reactome/GSEA)
3. **可视化**
   - 出版级图表(matplotlib/seaborn/plotly)
   - 交互式可视化(Dash/Streamlit)
   - 热图/火山图/UMAP/t-SNE
   - 网络图(iggraph/pyvis)

### 3. 药物发现流程

1. **靶点识别**
   - 疾病-基因关联(Open Targets)
   - 通路分析(Reactome/KEGG)
   - 文献挖掘
2. **虚拟筛选**
   - 化合物库获取(ChEMBL/PubChem/ZINC)
   - 分子对接(AutoDock/DiffDock)
   - 药效团搜索
   - ADMET 预测
3. **先导化合物优化**
   - 构效关系(SAR)分析
   - 类似物生成(RDKit/datamol)
   - 选择性预测
   - 毒性评估

### 4. 论文写作与发表

1. **论文结构**
   - IMRaD 格式(Introduction/Methods/Results/D Discussion)
   - 摘要(结构化/非结构化)
   - 图表说明
   - 补充材料
2. **写作流程**
   - 大纲制定
   - 分段写作(方法→结果→讨论→引言→摘要)
   - 引用管理
   - 语言润色
3. **投稿准备**
   - 期刊选择(影响因子/开放获取/审稿周期)
   - Cover Letter
   - 格式调整
   - 补充材料整理

### 5. 基金申请

1. **假设生成**:基于文献空白,提出可验证假设
2. **研究计划**:具体目标/方法/时间线/预期成果
3. **预算规划**:人员/设备/材料/差旅/间接成本
4. **影响陈述**:科学意义/社会影响/转化潜力

## 依赖说明

| 依赖类型 | 要求 | 说明 |
|:---------|:-----|:-----|
| LLM | 任意支持 Agent Skills 的 LLM | Claude/GPT/Gemini 等 |
| 运行环境 | Python 3.11+ (推荐3.12) | 科学计算需要 Python 环境 |
| 包管理 | uv (推荐) 或 pip | Python 包管理器 |
| 科学计算 | RDKit/Scanpy/scikit-learn/PyTorch | 按需安装特定领域包 |
| 数据库 | PubMed/ChEMBL/UniProt/PubChem | 公共科学数据库(免费) |
| 可选 | Modal 云计算 | 重计算任务(GPU) |
| 系统 | macOS/Linux/Windows+WSL2 | 部分 Linux 专属工具需 WSL2 |

## 异常处理

| 异常场景 | 处理方式 |
|:---------|:---------|
| 数据库连接超时 | 检查网络,使用本地缓存,重试 |
| Python 包冲突 | 使用虚拟环境(uv venv),固定版本 |
| 计算资源不足 | 降低数据量/采样,使用云 GPU(Modal) |
| 结果不可复现 | 固定随机种子,记录版本号,使用容器 |
| 引用格式错误 | 使用 Zotero/Mendeley 自动格式化 |
| 统计方法误用 | 检查数据分布假设,选择非参数替代 |

## 示例

### 输入:药物发现Pipeline

```
用户请求:寻找EGFR抑制剂用于肺癌治疗

输出:
1. 靶点:EGFR(查询Open Targets确认关联)
2. 化合物:ChEMBL查询IC50<50nM的EGFR抑制剂
3. SAR分析:RDKit分析结构-活性关系
4. 类似物:datamol生成改进类似物
5. 对接:DiffDock对接AlphaFold EGFR结构
6. 突变:COSMIC查询耐药突变
7. 文献:PubMed搜索耐药机制
8. 报告:综合可视化与发现报告
```

### 输入:单细胞RNA-seq分析

```
用户请求:分析10X Genomics单细胞数据,识别细胞类型

输出:
1. 数据加载:Scanpy读取10X数据
2. QC:过滤低质量细胞/双细胞去除
3. 标准化:LogNormalize
4. 降维:PCA → UMAP
5. 聚类:Leiden聚类
6. 细胞类型:NCBI Gene标记基因注释
7. 差异表达:PyDESeq2
8. 通路富集:Reactome/KEGG
9. 可视化:UMAP图/热图/火山图
```
