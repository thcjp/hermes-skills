# 画布艺术设计器

用代码在 PNG/PDF 画布上创建原创视觉艺术。坚持设计哲学:克制构图、图像主导层级、连贯内容结构、有品味的动效(静态作品中体现为视觉节奏)。

## 使用场景

| 场景 | 触发条件 | 说明 |
|:-----|:---------|:-----|
| 海报设计 | 活动海报/宣传海报 | 原创构图,非模板套用 |
| 艺术作品 | 抽象/具象艺术 | 生成可打印的艺术 PNG |
| 信息图 | 数据/流程可视化 | 静态信息图设计 |
| 品牌物料 | 名片/信纸/宣传单 | 统一视觉风格输出 |
| 社交配图 | 文章封面/社交分享图 | 多尺寸适配 |

## 工作流

### 1. 设计意图采集

1. **接收主题与目的**:设计主题、目标用途、目标受众、情感基调
2. **确定画布规格**:尺寸(如 A3/A4/1080x1080)、格式(PNG/PDF)、DPI
3. **风格定向**:参考风格(极简/复古/未来/手绘/几何)、配色倾向
4. **输出设计简报**:主题+规格+风格+情感,供确认

### 2. 构图与视觉层级

1. **确立视觉焦点**:确定画面主视觉元素(图像/图形/文字)
2. **布局网格**:基于三分法/黄金分割/对角线构图建立网格
3. **层级编排**:主视觉 > 次要元素 > 背景纹理 > 文字说明
4. **留白控制**:克制的留白,避免画面拥挤

### 3. 色彩与排版

1. **配色方案**:主色+辅色+强调色,限制在 3-5 色以内
2. **色彩情绪**:根据情感基调选择暖色/冷色/对比色/类比色
3. **字体选择**:标题字体+正文字体,最多 2 种字体
4. **字号层级**:标题/副标题/正文/注释,明确对比

### 4. 渲染输出

1. **代码生成**:用 Canvas API/SVG/HTML2Canvas 生成设计
2. **多格式输出**:PNG(位图)、PDF(矢量可打印)、SVG(可缩放)
3. **多尺寸适配**:同一设计输出不同尺寸版本
4. **打印就绪**:PDF 输出包含出血线和裁切标记

## 设计哲学(必须遵守)

1. **原创构图**:不套用模板,每个设计基于主题独立构图
2. **图像主导**:视觉重心由图像/图形承担,文字辅助
3. **克制色彩**:限制色板,避免彩虹色滥用
4. **有意义的内容结构**:每个元素都有存在理由,非装饰性堆砌
5. **拒绝通用 AI 美学**:避免紫色渐变、毛玻璃滥用、emoji 滥用

## 输出规范

- 设计源码保存到 `output/{design-name}/`
- PNG 预览:`output/{design-name}/preview.png`
- PDF 打印版:`output/{design-name}/print.pdf`
- 设计说明:`output/{design-name}/design-notes.md`(配色/字体/构图理由)
