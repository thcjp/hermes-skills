# 电子表格魔法师

全面处理 Excel 电子表格。从数据清洗到公式计算,从图表生成到透视表分析,覆盖电子表格全流程。

## 使用场景

| 场景 | 触发条件 | 说明 |
|:-----|:---------|:-----|
| 数据清洗 | 杂乱数据需整理 | 去重/分列/格式化 |
| 公式计算 | 复杂计算需求 | 财务/统计/工程公式 |
| 图表生成 | 数据可视化 | 柱状/折线/饼图/组合图 |
| 透视分析 | 多维数据汇总 | 透视表+切片器 |
| 多表合并 | 多个工作表整合 | 合并/关联/汇总 |
| 报表生成 | 定期报表 | 模板化自动生成 |

## 工作流

### 1. 数据读取与探查

1. **读取 Excel 文件**:.xlsx/.xls/.csv
2. **多工作表处理**:识别所有 Sheet,读取目标表
3. **数据探查**:
   - 行数/列数/数据类型
   - 缺失值统计
   - 唯一值/重复值检测
   - 数值范围/分布
4. **输出数据概览**:供用户确认处理方向

### 2. 数据清洗

1. **缺失值处理**:
   - 删除:缺失过多(>50%)
   - 填充:均值/中位数/众数/前后值
   - 标记:保留但标记
2. **重复值处理**:识别并删除/标记
3. **格式标准化**:
   - 日期格式统一(YYYY-MM-DD)
   - 数字格式(千分位/小数位)
   - 文本清理(去空格/统一大小写)
4. **数据类型转换**:文本转数字/日期/布尔

### 3. 公式与计算

1. **常用公式**:
   - 统计:SUM/AVERAGE/COUNT/MAX/MIN/MEDIAN
   - 查找:VLOOKUP/HLOOKUP/INDEX-MATCH/XLOOKUP
   - 逻辑:IF/IFS/AND/OR/SWITCH
   - 文本:CONCAT/LEFT/RIGHT/MID/SUBSTITUTE
   - 日期:DATE/DATEDIF/NETWORKDAYS/EOMONTH
2. **数组公式**:批量计算,避免辅助列
3. **条件聚合**:SUMIFS/COUNTIFS/AVERAGEIFS
4. **公式审计**:追踪依赖关系,检查错误

### 4. 图表生成

1. **图表类型选择**:
   - 趋势:折线图/面积图
   - 对比:柱状图/条形图
   - 占比:饼图/环形图
   - 关系:散点图/气泡图
   - 多维:组合图/雷达图
2. **图表美化**:
   - 标题与坐标轴标签
   - 数据标签与图例
   - 配色与主题一致
   - 关键数据高亮
3. **动态图表**:基于下拉/切片器的交互式图表

### 5. 透视表与多维分析

1. **创建透视表**:行/列/值/过滤器四区域
2. **分组与汇总**:按日期/数值区间分组
3. **计算字段**:透视表内自定义计算
4. **切片器**:交互式过滤
5. **透视表图**:基于透视表的图表

### 6. 输出与交付

1. **生成 .xlsx 文件**:保留公式与格式
2. **多工作表组织**:原始数据/清洗后/分析/图表分表
3. **条件格式**:数据条/色阶/图标集
4. **打印设置**:页面布局/打印区域/重复标题

## 输出规范

- Excel 文件:`output/{workbook-name}/workbook.xlsx`
- 数据概览:`output/{workbook-name}/data-overview.md`
- 清洗报告:`output/{workbook-name}/cleaning-report.md`
- 图表导出:`output/{workbook-name}/charts/`(PNG)
- 分析摘要:`output/{workbook-name}/analysis-summary.md`
