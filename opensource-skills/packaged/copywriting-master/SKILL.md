# 营销文案大师

基于经典文案框架与模板,生成以转化为导向的营销文案。不只写文字,更关注转化效果。

## 使用场景

| 场景 | 触发条件 | 说明 |
|:-----|:---------|:-----|
| 广告标题 | 投放广告 | 高点击率标题 |
| 产品描述 | 电商详情页 | 卖点提炼+转化文案 |
| 落地页 | 广告落地页 | 完整落地页文案 |
| 邮件营销 | EDM 邮件 | 主题+正文+CTA |
| 社交文案 | 社交媒体推广 | 平台适配文案 |
| 销售信 | 长文销售 | 说服型长文案 |

## 工作流

### 1. 需求分析

1. **产品/服务理解**:功能/卖点/差异化/定价
2. **目标受众画像**:人口统计/痛点/欲望/语言风格
3. **转化目标**:点击/注册/购买/分享
4. **投放渠道**:搜索引擎/社交/邮件/落地页
5. **品牌调性**:正式/活泼/专业/亲切

### 2. 框架选择

#### AIDA(注意力-兴趣-欲望-行动)
- Attention:标题抓住注意力
- Interest:引发兴趣(痛点/利益)
- Desire:激发欲望(场景/证据)
- Action:明确行动号召
- 适用:广告/落地页/销售信

#### PAS(问题-激化-解决)
- Problem:提出问题
- Agitate:激化痛点(后果/感受)
- Solve:给出解决方案
- 适用:痛点驱动的产品

#### FAB(特性-优势-利益)
- Feature:产品特性
- Advantage:相比的优势
- Benefit:用户的利益
- 适用:产品描述/技术产品

#### BAB(Before-After-Bridge)
- Before:现状(有问题)
- After:理想状态(无问题)
- Bridge:桥梁(产品)
- 适用:转化型文案

### 3. 文案生成

1. **标题生成**(5-10个候选):
   - 数字型:5个方法/3个秘密
   - 问题型:还在为X发愁?
   - 利益型:如何实现Y
   - 好奇型:你不知道的Z
   - 对比型:A vs B
2. **正文撰写**:
   - 钩子(首句抓住注意力)
   - 痛点(引发共鸣)
   - 方案(产品/服务介绍)
   - 证据(案例/数据/背书)
   - CTA(明确行动号召)
3. **多版本生成**:不同框架/不同角度/不同语气

### 4. 渠道适配

| 渠道 | 长度 | 风格 | 要点 |
|:-----|:-----|:-----|:-----|
| 搜索广告 | 标题30字/描述90字 | 精准 | 关键词+利益点 |
| 社交(微博) | 140字 | 活泼 | 钩子+话题 |
| 社交(小红书) | 500-1000字 | 种草 | 体验+关键词 |
| 邮件主题 | 50字内 | 吸引 | 好奇+利益 |
| 落地页 | 不限 | 说服 | 完整转化漏斗 |
| 产品详情 | 300-800字 | 专业 | FAB+证据 |

### 5. 优化与 A/B 测试

1. **文案评分**:可读性/说服力/转化潜力
2. **A/B 测试建议**:哪些元素值得测试
3. **关键词嵌入**:SEO 友好(如落地页)
4. **CTA 优化**:动词+利益+紧迫感

## 文案原则

1. **用户视角**:写"你"不写"我们",关注用户利益
2. **具体胜于抽象**:用数字/案例,不用空泛形容词
3. **一个文案一个核心信息**:不贪多,聚焦
4. **CTA 明确**:告诉用户下一步做什么
5. **简洁有力**:删减冗余,每字都有价值

## 输出规范

- 文案集:`output/{campaign-name}/copy.md`(多版本)
- 标题候选:`output/{campaign-name}/headlines.md`
- 落地页文案:`output/{campaign-name}/landing-page.md`
- 邮件文案:`output/{campaign-name}/email.md`
- A/B 测试建议:`output/{campaign-name}/ab-test-plan.md`
