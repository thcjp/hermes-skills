# PDF工具箱Pro

全面处理 PDF 文档。从内容提取到文档生成,从合并拆分到表单填写,覆盖 PDF 全流程。

## 使用场景

| 场景 | 触发条件 | 说明 |
|:-----|:---------|:-----|
| 内容提取 | 从 PDF 取文本/表格 | OCR + 结构化提取 |
| 文档合并 | 多个 PDF 合一 | 按顺序合并+目录 |
| 文档拆分 | 一个 PDF 拆多 | 按页/按书签拆分 |
| 表单填写 | PDF 表单自动化 | 数据填充表单字段 |
| PDF 生成 | 从内容生成 PDF | HTML/Markdown → PDF |
| 注释标记 | 审阅 PDF | 高亮/批注/签名 |

## 工作流

### 1. PDF 读取与解析

1. **读取 PDF 文件**:支持加密 PDF(需密码)
2. **元数据提取**:标题/作者/主题/关键词/创建时间
3. **页面信息**:页数/页面尺寸/方向
4. **书签/目录**:提取文档结构

### 2. 文本与表格提取

1. **文本提取**:
   - 纯文本 PDF:直接提取,保留段落
   - 扫描 PDF:OCR 识别(多语言支持)
   - 布局保留:多栏/表格/图文混排
2. **表格提取**:
   - 识别表格结构(行/列/合并单元格)
   - 输出为 CSV/Excel/Markdown 表格
   - 处理跨页表格
3. **图片提取**:提取嵌入图片,保留分辨率

### 3. 合并与拆分

1. **合并**:
   - 多个 PDF 按顺序合并
   - 添加分隔页/目录
   - 统一页面大小
2. **拆分**:
   - 按页码范围拆分(如 1-5, 6-10)
   - 按书签/章节拆分
   - 每页拆为单独 PDF
3. **页面操作**:
   - 删除指定页
   - 重排序页面
   - 旋转页面(90/180/270)
   - 提取指定页

### 4. 表单处理

1. **表单字段识别**:文本框/复选框/单选/下拉/签名域
2. **数据填充**:
   - 从 JSON/Excel 读取数据
   - 映射字段名到表单域
   - 批量填充多个表单
3. **表单扁平化**:填充后锁定,不可再编辑

### 5. 注释与标记

1. **高亮**:选中文本高亮(多色)
2. **批注**:添加便签批注
3. **画线标记**:下划线/删除线/波浪线
4. **形状标注**:矩形/椭圆/箭头
5. **签名**:手写签名图片嵌入
6. **水印**:文字/图片水印(透明度/位置)

### 6. PDF 生成

1. **从 HTML 生成**:HTML/CSS → PDF(支持打印样式)
2. **从 Markdown 生成**:Markdown → PDF(含目录/页码)
3. **从零构建**:程序化生成(文本/图片/表格/图形)
4. **页面设计**:页眉页脚/页码/水印/边距

## 输出规范

- 提取文本:`output/{pdf-name}/text.txt`
- 提取表格:`output/{pdf-name}/tables/`(CSV/Excel)
- 提取图片:`output/{pdf-name}/images/`
- 合并结果:`output/{merge-name}/merged.pdf`
- 拆分结果:`output/{split-name}/pages/`
- 填充表单:`output/{form-name}/filled.pdf`
- 元数据:`output/{pdf-name}/metadata.json`
