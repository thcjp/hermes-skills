# 品牌识别创造器

创建完整的品牌识别系统。从色彩到字体，从语调到视觉规范，确保品牌在所有触点保持一致。

## 核心能力

- **品牌定位**：使命/愿景/价值观/个性/目标受众/竞争定位/品牌承诺
- **色彩体系**：主色辅色功能色中性色 + 60-30-10 比例原则 + HEX/RGB/CMYK/Pantone 多格式输出
- **字体规范**：字体家族/字号层级/字重/行高字距/Web 字体配置（woff2 + font-display: swap）
- **语调与文案风格**：四维度语调定位（正式-亲切/严肃-幽默/专业-大众/保守-前卫）+ 文案原则 + 禁忌词清单
- **视觉规范**：Logo 规则（清晰空间/最小尺寸/禁用用法）+ 图标风格 + 插画风格 + 布局网格 + 应用示例

## 适用场景

| 场景 | 输入 | 输出 |
|:-----|:-----|:-----|
| 新品牌建立 | 品牌名称 + 行业 + 目标受众 | 完整品牌手册（定位+色彩+字体+语调+视觉+应用） |
| 品牌升级 | 现有品牌问题 + 升级方向 | 重新定义的品牌识别系统 + 迁移指南 |
| 多渠道统一 | 各渠道现有素材 + 统一需求 | 统一品牌应用规范 + 各渠道适配指南 |
| 品牌手册 | 品牌基本信息 | 可分享的 Markdown 品牌手册 + 色彩 JSON + 字体规范 |
| 子品牌创建 | 主品牌规范 + 子品牌定位 | 子品牌识别系统 + 与主品牌的关系定义 |

**不适用于**：
- Logo 矢量图形设计（需专业设计师使用 Illustrator/Figma）
- 摄影与插画原创内容（属摄影师/插画师职责）
- 包装物理打样（属生产制造环节）
- 品牌法律注册与商标申请（属法律服务）

## 使用流程

### Step 1: 品牌定位
- 品牌内核：使命（为什么存在）/ 愿景（要去哪里）/ 价值观（相信什么）/ 个性（如果品牌是人）
- 目标受众：人口统计 / 心理特征 / 文化背景
- 竞争定位：差异化定位图
- 品牌承诺：一句话承诺

### Step 2: 色彩体系
- 主色（Primary）：1 个主色，代表品牌核心，HEX/RGB/CMYK/Pantone 多格式
- 辅色（Secondary）：1-2 个辅色，与主色和谐搭配
- 功能色：成功（绿）/ 警告（黄）/ 错误（红）/ 信息（蓝）
- 中性色：文字色/背景色/边框色/禁用色，灰度阶梯 5-7 级
- 色彩比例：60-30-10 原则（主色 60% / 辅色 30% / 强调色 10%）

### Step 3: 字体规范
- 字体家族：标题字体（有个性，识别度高）+ 正文字体（高可读性，中性）+ 代码字体（等宽）
- 字号层级：H1 48-64px / H2 32-40px / H3 24-28px / 正文 16-18px / 小字 12-14px
- 字重：Regular(400) / Medium(500) / SemiBold(600) / Bold(700)
- 行高与字距：正文行高 1.5-1.6，标题行高 1.2-1.3
- Web 字体：提供 woff2，设置 `font-display: swap`

### Step 4: 语调与文案风格
- 品牌语调四维度定位（正式-亲切/严肃-幽默/专业-大众/保守-前卫）
- 文案原则：用"你"不用"您"（或反之）/ 主动语态优于被动 / 正面表述优于否定
- 禁忌词清单：避免使用的词汇/表达

### Step 5: 视觉规范
- Logo 规则：主 Logo/副 Logo/图标版 + 清晰空间 + 最小尺寸 + 不可接受的用法
- 图标风格：线性/填充/双色，圆角/直角
- 插画风格：扁平/手绘/3D/几何
- 图片风格：摄影/插画/混合，色调/构图
- 布局网格：栏数/间距/断点

### Step 6: 应用示例与输出
- 应用示例：名片/信头信封/PPT 模板/社交媒体头像封面/网站 Header/Footer/邮件签名
- 输出文件：
  - 品牌手册：`output/{brand-name}/brand-guidelines.md`
  - 色彩规范：`output/{brand-name}/colors.json`
  - 字体规范：`output/{brand-name}/typography.md`
  - 语调指南：`output/{brand-name}/tone-of-voice.md`
  - Logo 使用：`output/{brand-name}/logo-usage.md`
  - 应用示例：`output/{brand-name}/examples/`

### 示例 2：语调指南

**输入**：
```
品牌：CloudPulse
需要语调指南，用于文案团队参考。
```

**输出**：
```markdown
# CloudPulse 语调指南

## 语调四维度定位
| 维度 | 左 (0) | 定位 | 右 (10) |
|:-----|:-------|:-----|:--------|
| 正式 ←→ 亲切 | 商务正式 | 7 | 轻松亲切 |
| 严肃 ←→ 幽默 | 严谨克制 | 8 | 活泼幽默 |
| 专业 ←→ 大众 | 行业术语 | 3 | 大白话 |
| 保守 ←→ 前卫 | 稳重传统 | 6 | 创新前卫 |

定位说明：偏正式、偏严肃、偏专业（目标受众是技术人员，接受术语），适度前卫。

## 文案原则
1. 用"你"不用"您"（平等的技术伙伴关系）
2. 主动语态优于被动（"系统检测到异常" 而非 "异常被系统检测到"）
3. 正面表述优于否定（"保持稳定" 而非 "不要崩溃"）
4. 数据优于形容词（"99.9% 可用性" 而非 "超高性能"）
5. 一句话能说清的不用两句

## 禁忌词清单
- 避免："颠覆性"、"革命性"、"宇宙级"（过度营销词）
- 避免："简单易用"（不具体，改用"3 分钟接入"）
- 避免："业界领先"（无依据的宣称）
- 避免： emoji 滥用（官方文档最多 1 个/段落）
- 避免：感叹号连用（"!!!"）
```

## 错误处理

| 错误场景 | 原因 | 处理方式 |
|:---------|:-----|:---------|
| 色彩对比度不足 | 主色与背景色色差过小 | 使用 WCAG AA 标准（对比度 ≥ 4.5:1），自动校验并调整 |
| 字体不可商用 | 使用了付费字体未授权 | 优先选择开源字体（Inter/Roboto/Source Sans），标注授权要求 |
| 品牌定位模糊 | 用户提供的品牌信息不足 | 通过提问补充：目标受众/竞争对手/差异化/品牌个性 |
| 色彩格式缺失 | 仅提供 HEX，缺 CMYK/Pantone | 自动从 HEX 转换 RGB/CMYK，Pantone 建议人工匹配 |
| 应用示例尺寸错误 | 未按标准尺寸输出 | 参考标准尺寸表（名片 90x54mm/社交头像 512x512px 等） |
| 语调定位矛盾 | 四维度定位出现冲突 | 指出矛盾点，请求优先级排序，以目标受众偏好为准 |

## 常见问题

### Q1: 生成的色彩规范如何导入设计工具？
A: `colors.json` 文件可直接导入 Figma（通过 Tokens Studio 插件）/ Sketch / Adobe XD。也可转为 CSS 变量或 Tailwind 配置，供前端直接使用。

### Q2: 没有专业设计师能否使用本 Skill？
A: 可以。本 Skill 输出的是品牌规范文档（色彩值/字体/语调/规则），非 Logo 矢量图。Logo 图形建议使用 AI 生图工具或委托设计师，本 Skill 提供 Logo 使用规则（清晰空间/最小尺寸/禁用用法）。

### Q3: 色彩对比度如何保证无障碍？
A: 所有文字与背景色组合需满足 WCAG AA 标准（普通文字对比度 ≥ 4.5:1，大文字 ≥ 3:1）。建议使用在线工具（WebAIM Contrast Checker）校验，本 Skill 在输出时会标注对比度数值。

### Q4: 品牌手册如何分享给团队？
A: 输出的 Markdown 文件可托管在 Notion/Confluence/GitHub Wiki。`colors.json` 供设计师导入设计工具，`typography.md` 供前端开发者参考，`tone-of-voice.md` 供文案团队使用。

## 已知限制

- 本 Skill 输出品牌规范文档，不生成 Logo 矢量图形（需设计师或 AI 生图工具）
- 色彩的 Pantone 匹配为近似值，精确匹配需参考 Pantone 色卡实物
- 字体推荐基于通用可用性，特殊行业（如奢侈品）可能需要定制字体
- 性能取决于底层 LLM 能力，复杂品牌策略可能需要品牌顾问人工审查
- 不涵盖品牌法律事务（商标注册、版权保护需咨询律师）

## 安全

- **API Key 零暴露**：本 Skill 不涉及任何外部 API 调用，无需配置密钥
- **字体授权合规**：推荐开源字体（OFL/Apache 协议），标注付费字体的授权要求
- **数据本地化**：所有品牌文档保存在本地 `output/` 目录，不上传外部服务
- **商标冲突提醒**：品牌命名时提醒用户检索商标数据库，避免侵权风险
