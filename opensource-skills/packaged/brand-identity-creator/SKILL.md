# 品牌识别创造器

创建完整的品牌识别系统。从色彩到字体,从语调到视觉规范,确保品牌在所有触点保持一致。

## 使用场景

| 场景 | 触发条件 | 说明 |
|:-----|:---------|:-----|
| 新品牌建立 | 创建新品牌 | 从零构建品牌识别系统 |
| 品牌升级 | 品牌形象老化 | 重新定义品牌视觉与语调 |
| 多渠道统一 | 各渠道风格不一 | 统一品牌应用规范 |
| 品牌手册 | 需要品牌指南 | 生成可分享的品牌手册 |
| 子品牌创建 | 品牌延伸 | 在主品牌下创建子品牌 |

## 工作流

### 1. 品牌定位

1. **品牌内核**:
   - 使命(为什么存在)
   - 愿景(要去哪里)
   - 价值观(相信什么)
   - 个性(如果品牌是人,是什么性格)
2. **目标受众**:人口统计/心理特征/文化背景
3. **竞争定位**:差异化定位图
4. **品牌承诺**:一句话承诺

### 2. 色彩体系

1. **主色(Primary)**:
   - 1个主色,代表品牌核心
   - HEX/RGB/CMYK/Pantone 多格式
   - 使用场景(Logo/标题/按钮)
2. **辅色(Secondary)**:
   - 1-2个辅色,补充主色
   - 与主色和谐搭配
3. **功能色**:
   - 成功(绿)/警告(黄)/错误(红)/信息(蓝)
4. **中性色**:
   - 文字色/背景色/边框色/禁用色
   - 灰度阶梯(5-7级)
5. **色彩比例**:60-30-10 原则(主色60%/辅色30%/强调色10%)

### 3. 字体规范

1. **字体家族**:
   - 标题字体:有个性,识别度高
   - 正文字体:高可读性,中性
   - 代码字体:等宽(如需要)
2. **字号层级**:
   - H1: 48-64px / H2: 32-40px / H3: 24-28px
   - 正文: 16-18px / 小字: 12-14px
3. **字重**:Regular(400)/Medium(500)/SemiBold(600)/Bold(700)
4. **行高与字距**:正文行高 1.5-1.6,标题行高 1.2-1.3
5. **Web 字体**:提供 woff2,设置 font-display: swap

### 4. 语调与文案风格

1. **品牌语调维度**:
   | 维度 | 左 | 右 |
   |:-----|:---|:---|
   | 正式 ←→ 亲切 | 商务正式 | 轻松亲切 |
   | 严肃 ←→ 幽默 | 严谨克制 | 活泼幽默 |
   | 专业 ←→ 大众 | 行业术语 | 大白话 |
   | 保守 ←→ 前卫 | 稳重传统 | 创新前卫 |
2. **文案原则**:
   - 用"你"不用"您"(或反之,视品牌)
   - 主动语态优于被动
   - 正面表述优于否定
3. **禁忌词清单**:避免使用的词汇/表达

### 5. 视觉规范

1. **Logo 规范**:
   - 主 Logo/副 Logo/图标版
   - 清晰空间(最小边距)
   - 最小尺寸
   - 不可接受的用法(拉伸/变色/旋转)
2. **图标风格**:线性/填充/双色,圆角/直角
3. **插画风格**:扁平/手绘/3D/几何
4. **图片风格**:摄影/插画/混合,色调/构图
5. **布局网格**:栏数/间距/断点

### 6. 应用示例

1. 名片设计
2. 信头/信封
3. PPT 模板
4. 社交媒体头像/封面
5. 网站 Header/Footer
6. 邮件签名

## 输出规范

- 品牌手册:`output/{brand-name}/brand-guidelines.md`
- 色彩规范:`output/{brand-name}/colors.json`(HEX/RGB/CMYK)
- 字体规范:`output/{brand-name}/typography.md`
- 语调指南:`output/{brand-name}/tone-of-voice.md`
- Logo 使用:`output/{brand-name}/logo-usage.md`
- 应用示例:`output/{brand-name}/examples/`
