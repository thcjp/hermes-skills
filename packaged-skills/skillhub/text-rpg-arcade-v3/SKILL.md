# Text Game Arcade Uni

## 核心能力

- 综合性 ASCII 文字游戏大厅
- 严格按规则渲染和裁判五子棋、围棋、井字棋、黑白棋、四子棋、海战棋、中国象棋、国际象棋、扫雷、数独、2048、推箱子、文字迷宫，以及接龙、猜词、猜数、Lights Out 等游戏
- ASCII/线框字符棋盘渲染与规则裁判
- 人机对战与双人轮流模式

### 对局流程

1. 用户说"下五子棋"或"来一局井字棋"激活游戏
2. 系统渲染初始棋盘并询问先后手
3. 用户落子，系统裁判并渲染更新后的棋盘
4. 人机模式下AI自动应对；双人模式等待下一玩家
5. 系统自动判定胜负/和棋，提示是否再来一局

## 难度设置

人机对战支持三档难度：

| 难度 | AI策略 | 适用场景 |
|:------|:------|:------|
| 简单 | 随机走法+基础规则 | 休闲体验、学习规则 |
| 中等 | 启发式评估+1-2步搜索 | 日常对弈 |
| 困难 | Minimax+Alpha-Beta剪枝+开局库 | 挑战极限 |

切换难度：在游戏开始前说"困难模式"或在对局中说"调整难度为中等"。

## 常见问题

### Q1: 如何开始使用Text Game Arcade Uni？
A: 直接用自然语言说出你想玩的游戏即可，例如"来一局五子棋"、"玩扫雷"、"下国际象棋"。系统会自动渲染棋盘并开始对局。无需安装或配置，所有游戏都在对话中通过ASCII字符渲染。

### Q2: 支持悔棋吗？
A: 棋类对弈支持悔棋功能。在对局中输入"悔棋"或"undo"即可撤销上一步操作。人机模式下悔棋会同时撤销玩家和AI的各一步。扫雷、数独、推箱子等解谜游戏不支持悔棋，建议谨慎操作。

### Q3: 围棋的提子规则如何判定？
A: 系统严格按围棋规则执行：每次落子后检查对方棋块是否有气，无气则提走。同时检查己方棋块是否有气（禁止自杀手，除非该手能提对方子）。劫争规则也会严格执行，禁止立即回提同一交叉点。

### Q4: 如何切换双人模式？
A: 在开始游戏时说"双人模式"即可。双人模式下系统不进行AI应对，而是等待两个玩家轮流输入。系统会显示当前轮到哪方落子。适用于两人共用一个对话窗口对弈的场景。

### Q5: 棋盘在手机上显示错位怎么办？
A: ASCII棋盘在等宽字体下显示最佳。如果手机上出现错位，建议：（1）将聊天字体设置为等宽字体；（2）横屏查看较大棋盘（如15x15五子棋）；（3）缩小棋盘尺寸，说"用9x9棋盘下围棋"。

| 错误场景(续)| 原因 | 处理方式 |
|:---------|---------:|:---------|
| LLM响应超时或无响应 | 网络延迟或模型负载过高 | ，请求；确认Agent平台LLM服务正常 |
| 输入内容格式不正确 | 用户输入不符合skill预期格式 | 检查输入是否符合skill使用说明中的格式要求，参考示例章节 |
