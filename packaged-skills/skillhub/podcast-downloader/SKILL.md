# Podcast Downloader

## 核心能力

- 从小宇宙(xiaoyuzhoufm
- com)下载播客音频和Show Notes
- 自动转换为MP3格式（兼容Sanag、小游等骨传导蓝牙耳机、水下游泳时离线播放）

```bash

```

| 参数名 | 类型 | 必填 | 说明 |

| content | string | 否 | 相关说明, 默认: 默认值 |
