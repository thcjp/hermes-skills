# 百度网盘 Free

百度网盘基础文件管理工具，所有操作限制在 `/apps/bdpan/` 目录内。本免费版仅支持查看、列表、上传与下载小文件。

## 触发规则

用户消息中包含**明确指向百度网盘**的关键词（`百度网盘`、`百度云盘`、`bdpan`、`baidu netdisk`、`baidu pan`、`baidu drive`、`pan.baidu.com`），且操作意图属于本免费版支持范围（查看/列表/上传/下载）时触发。

用户仅说"网盘""云盘"等泛指词时，需反问"您指的是百度网盘（pan.baidu.com）吗？"再决定。

**写操作**（上传、下载覆盖）必须先列出"将要执行的操作 + 影响范围"并取得用户显式确认（`y`）后再执行。

## 已知限制

1. **登录**：必须使用 `bash ${CLAUDE_SKILL_DIR}/scripts/login.sh`，禁止直接调用 `bdpan login`
2. **Token/配置**：禁止读取或输出 `~/.config/bdpan/config.json` 内容
3. **路径安全**：禁止路径穿越（`..`、`~`）、禁止访问 `/apps/bdpan/` 范围外的绝对路径
4. **文件大小**：本免费版仅支持 ≤50MB 的小文件下载，不支持 nohup 后台下载策略

## 前置检查

每次触发时按顺序执行：

1. **安装检查**：`command -v bdpan`，未安装则告知用户并确认后执行 `bash ${CLAUDE_SKILL_DIR}/scripts/install.sh`
2. **登录检查**：`bdpan whoami`，未登录则引导执行 `bash ${CLAUDE_SKILL_DIR}/scripts/login.sh`
3. **路径校验**：验证远端路径在 `/apps/bdpan/` 范围内

## 确认规则

| 风险等级 | 操作 | 策略 |
| --- | --- | --- |
| **高（必须确认）** | 上传/下载目标已存在同名文件 | 列出影响范围，等待用户确认 |
| **中（路径模糊时确认）** | upload、download | 路径明确直接执行，不明确则确认 |
| **低（直接执行）** | ls、whoami | 无需确认 |

## 基础操作

### 查看状态与列表

```bash
bdpan whoami                                          # 查看登录状态
bdpan ls [目录路径] [--json] [--order name|time|size] [--desc] [--folder]
```

### 上传

```bash
bdpan upload <本地路径> <远端路径>
```

**关键约束：** 单文件上传远端路径必须是文件名，禁止以 `/` 结尾。文件夹上传：`bdpan upload ./project/ project/`。

步骤：确认本地路径存在 → 确认远端路径 → `bdpan ls` 检查远端是否已存在 → 执行。

### 下载（仅小文件 ≤50MB）

```bash
bdpan download <远端路径> <本地路径>
```

步骤：`bdpan ls` 确认云端存在 → 用 `bdpan ls --json` 获取文件大小 → 确认 ≤50MB → 检查本地是否已存在 → 执行。Bash 工具 timeout 设为 300000（5 分钟）。

> 超过 50MB 的大文件下载需使用 nohup 后台策略，该功能仅在付费版中提供。

## 案例展示

### 案例 1：查看网盘文件并下载小文件

**用户请求**：帮我看下百度网盘里 docs 目录有哪些文件，然后把 report.pdf 下载到本地。

**执行流程**：

```bash
# 1. 前置检查
bdpan whoami

# 2. 列出 docs 目录（只读）
bdpan ls docs/

# 3. 确认文件存在与大小
bdpan ls --json docs/report.pdf

# 4. 确认 ≤50MB 后下载（写操作，需确认）
# 向用户展示：将下载 docs/report.pdf 到 ./report.pdf，确认后执行
bdpan download docs/report.pdf ./report.pdf
```

**输出说明**：先列出 docs 目录文件清单，确认 report.pdf 存在且为小文件后下载。若文件超过 50MB，提示该功能需升级付费版。

### 案例 2：上传文件到网盘备份

**用户请求**：把本地的 notes.md 上传到百度网盘的 backup 目录。

**执行流程**：

```bash
# 1. 前置检查
bdpan whoami

# 2. 确认本地文件存在
ls -l ./notes.md

# 3. 检查远端是否已存在同名文件
bdpan ls backup/notes.md

# 4. 上传（写操作，需确认）
# 向用户展示：将上传 ./notes.md 到 backup/notes.md，确认后执行
bdpan upload ./notes.md backup/notes.md
```

**输出说明**：上传前检查远端是否已有同名文件，确认后执行。展示时用中文名"我的应用数据/bdpan/backup/notes.md"。

## 错误处理

| 错误场景 | 原因 | 处理方式 |
|---------|------|---------|
| `bdpan: command not found` | 未安装 bdpan 工具 | 引导用户确认后执行 `bash ${CLAUDE_SKILL_DIR}/scripts/install.sh` |
| `未登录` / `whoami 失败` | Token 过期或未登录 | 引导执行 `bash ${CLAUDE_SKILL_DIR}/scripts/login.sh` 重新登录 |
| 路径穿越被拦截 | 路径含 `..`、`~` 或绝对路径越界 | 拒绝执行，提示操作仅限 `/apps/bdpan/` 范围内相对路径 |
| 远端文件不存在 | `bdpan ls` 返回空 | 检查路径拼写，用 `bdpan ls` 确认目录内容 |
| 文件超过 50MB | 本免费版不支持大文件后台下载 | 提示用户升级至付费版 baidu-netdisk-skills 技能 |
| 同名文件已存在 | 上传/下载目标路径已有同名文件 | 列出现有文件，询问覆盖还是重命名，确认后执行 |
| 直接调用 bdpan login 被拦截 | Agent 误用裸 `bdpan login` 命令 | 禁止该调用，必须通过 `scripts/login.sh` 执行登录 |

## 常见问题

### Q1：怎么查看网盘里有哪些文件？
A：先 `bdpan whoami` 确认登录状态，再 `bdpan ls [目录路径]` 列出文件。支持 `--order name|time|size` 排序、`--folder` 只看文件夹。展示时路径映射为中文名（`我的应用数据/bdpan/...`）。

### Q2：上传时远端路径怎么填？
A：单文件上传远端路径必须是文件名（如 `docs/notes.md`），禁止以 `/` 结尾。文件夹上传用 `bdpan upload ./project/ project/`。上传前用 `bdpan ls` 检查远端是否已存在同名文件。

### Q3：能下载大文件或从分享链接下载吗？
A：本免费版仅支持 ≤50MB 的小文件直接下载。超过 50MB 的大文件后台下载、分享链接转存下载、nohup 后台策略等高级功能需升级至付费版 baidu-netdisk-skills 技能。

### Q4：为什么不能访问 /apps/bdpan/ 以外的路径？
A：安全限制。所有操作限定在 `/apps/bdpan/` 范围内，禁止路径穿越（`..`、`~`）和越界绝对路径，防止误操作系统其他目录。

## 已知限制

- 所有操作限制在 `/apps/bdpan/` 目录范围内。
- 仅支持 ≤50MB 小文件下载，不支持大文件后台下载策略。
- 不支持转存、分享、搜索、移动、复制、重命名、删除等操作。
- 不支持 Agent 记忆备份与恢复功能。
