# Bilibili Helper Free

B站视频运营基础助手。提供标题生成、简介与标签推荐、口播脚本撰写，输出内容可直接用于B站投稿页面。

## 使用方式

通过自然语言向Agent描述需求，或使用以下命令格式驱动脚本执行：

```bash
bili.sh title "视频主题"
bili.sh desc "视频主题"
bili.sh script "视频主题" --length 5|10|15
bili.sh tags "视频主题"
bili.sh help
```

所有输出为纯文本，可直接复制到B站投稿页面使用。

## 适用时机

- 准备发布B站视频，需要标题、简介、标签等投稿素材
- 撰写知识区或生活区口播视频脚本
- 视频标签不够用，需要补充推荐标签

## 核心能力

### 视频标题生成

基于B站平台调性与热门标题公式，为给定主题生成5个可选标题方案。标题风格覆盖悬念式、数字式、对比式等常见类型，适配知识区、科技区、生活区等主流分区。生成时考虑B站搜索权重，规避标题党与平台限流词。

- 执行`视频标题生成`操作，处理输入数据并返回结果
- 验证执行结果，确认输出符合预期格式
- 参考`视频标题生成`相关配置参数进行设置
### 视频简介与标签

生成结构化视频简介，包含内容概述与互动引导语。标签推荐结合视频主题与分区热词，输出8至10个标签供选择，覆盖核心主题词与长尾搜索词。

- 执行`视频简介与标签`操作，处理输入数据并返回结果
- 验证执行结果，确认输出符合预期格式
- 参考`视频简介与标签`相关配置参数进行设置
### 口播脚本撰写
按指定时长（5分钟、10分钟、15分钟）生成基础口播脚本，包含开场钩子、内容主体与结尾互动引导。脚本适配知识科普、经验分享等常见口播类型，标注建议语速。5分钟脚本约1100字，10分钟约2200字，15分钟约3300字。

**输入**: 用户提供口播脚本撰写所需的指令和必要参数。
**处理**: 按照skill规范执行口播脚本撰写操作,遵循单一意图原则。

- 执行`口播脚本撰写`操作，处理输入数据并返回结果
- 验证执行结果，确认输出符合预期格式
- 参考`口播脚本撰写`相关配置参数进行设置

## 异常处理

### 标题生成结果包含敏感词

B站标题审核较严，生成结果偶发包含擦边词或平台限流词。处理方式：重新生成时在主题描述中补充"避开敏感词"约束，或手动替换敏感词后再次生成。若多次生成仍出现敏感词，建议调整视频主题表述方式。

### 标签推荐与视频分区不匹配

当视频主题跨分区时，标签推荐可能与目标分区不匹配。处理方式：在输入时明确指定目标分区（如"知识区-科技科普"），生成结果将优先匹配该分区的标签体系。若已生成不匹配标签，可手动筛选保留相关标签后重新生成。

### 口播脚本时长偏差过大

生成10分钟脚本实际字数可能偏离预期（按每分钟220字计算应为2200字左右）。处理方式：在输入时补充内容密度要求（如"详细展开"或"精简概括"），或在生成后手动增删段落调整时长。建议录制前通读脚本并计时。

### 视频主题过于小众无法生成有效标签

当视频主题属于极小众领域时，标签推荐可能仅包含泛化标签，搜索价值低。处理方式：在输入时补充主题的关联领域与大类目信息，帮助生成更广泛的关联标签。建议结合手动搜索B站同类视频的标签进行补充。

### 标签数量超出B站限制

B站投稿标签数量上限为10个，生成结果可能输出超过10个标签。处理方式：从生成的标签中按相关性排序选取前8至10个，优先保留核心主题词与高搜索量长尾词，剔除重复或泛化标签。

## 常见问题

### 生成的标题是否符合B站审核规范？

标题生成时已避开常见违规词与标题党套路，但B站审核规则会动态调整。建议投稿前在B站创作中心查看最新审核规范，对生成标题进行人工复核，特别注意敏感领域的表述限制。

### 口播脚本的时长如何控制？

脚本时长通过字数与建议语速控制，5分钟脚本约1100字、10分钟约2200字、15分钟约3300字（按每分钟220字计算）。实际录制时长受语速与停顿影响，建议录制前通读计时并预留浮动空间。

### 标签推荐的依据是什么？

标签推荐综合考虑视频主题相关度、B站搜索热度与分区标签体系。核心标签优先匹配主题关键词，长尾标签覆盖用户搜索习惯。标签效果受平台算法变动影响，建议定期更新。

### 能否针对特定分区生成内容？

支持针对知识区、科技区、生活区等主流分区生成差异化内容。在输入时指定分区名称，标题风格、简介结构与标签推荐将自动适配该分区的调性。若未指定分区，将根据主题内容自动推断。

## 已知限制

- 无法直接调用B站API发布视频，生成内容需手动复制到投稿页面
- 标题与标签的流量效果受B站推荐算法变动影响
- 口播脚本为文本输出，无法生成配音音频或字幕文件
- 不适用于直播切片、二创混剪等短视频内容的自动化生成
- 生成内容基于文本模型，无法分析视频画面内容
