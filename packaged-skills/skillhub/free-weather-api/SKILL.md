## 适用场景

**简单查询**:

* "北京天气怎么样？"
* "上海明天会下雨吗？"
* "查一下深圳最近三天的天气"

**详细查询**:

* "下周去杭州旅游，帮我看看天气情况"
* "周末广州的天气适合户外活动吗？"
* "成都未来一周的空气质量如何？"

**特殊场景**:

* "明天去爬山，需要带什么装备？"（结合天气给建议）
* "后天有重要会议要出差，需要注意什么？"（关注极端天气）

## 输入格式

| 输入 | 来源 | 说明 |
| --- | --- | --- |
| location | 用户指定 | 城市或地区名称，如"北京"、"上海浦东" |
| days | 用户指定（可选） | 预报天数，默认为当天，可选1-7天 |
| details | 用户指定（可选） | 是否需要详细信息（空气质量、紫外线等） |
