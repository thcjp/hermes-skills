# AnyGen 图表生成器（免费版）

通过 AnyGen CLI 调用 `www.anygen.io` 服务端绘图引擎，将文本描述转换为流程图、架构图等基础可视化图表。核心操作类型为 `smart_draw`，由 `anygen-workflow-generate` 依赖技能编排生成流程。本免费版支持基础图表类型与默认渲染配置。

## 认证配置

AnyGen CLI 提供 API Key 认证方式：

```bash
anygen auth login --api-key sk-xxxxxxxxxxxxxxxx
```

也可通过环境变量持久化配置：

```bash
export ANYGEN_API_KEY=sk-xxxxxxxxxxxxxxxx
```

当命令返回认证错误时，检查 API Key 是否正确设置，或重新执行登录命令后重试。

## 依赖技能安装

本技能依赖 `anygen-workflow-generate` 技能来编排 `smart_draw` 操作。首次使用前需安装：

```bash
anygen skill install --platform claude-code -y
```

`--platform` 参数根据当前 Agent 运行平台选择，常见值包括 `claude-code`、`cursor`、`codex`、`gemini-cli`。

## 适用场景

### 场景一：技术文档流程图配图

编写技术文档时，需要将业务流程或算法流程可视化为标准流程图。用户提供流程步骤的文字描述，本技能通过 smart_draw 操作生成包含判断分支、起止节点的流程图，直接嵌入文档。

### 场景二：系统架构图生成

设计系统或梳理现有架构时，需要将模块组成与依赖关系可视化为架构图。用户提供系统组成部分与调用关系的文字描述，本技能生成包含组件节点与连线的架构图，辅助团队理解系统结构。

### 场景三：业务流程梳理

产品或运营团队梳理业务流程时，需要将业务步骤可视化。用户提供业务环节描述，本技能生成标准流程图，便于团队评审与讨论流程合理性。

## 使用案例

### 案例一：用户注册流程图生成

用户描述用户注册流程包含填写信息、邮箱验证、手机号验证、创建账户四个步骤，其中邮箱验证与手机号验证失败需返回填写信息步骤重新提交。本技能调用 smart_draw 操作，生成包含起止节点、判断菱形、回退连线的标准流程图，每个节点附带简短说明，便于产品团队评审注册流程。

## 异常处理


### 认证失败（auth error）

命令返回认证错误时，说明 API Key 未设置或失效。处理方式：检查 `ANYGEN_API_KEY` 环境变量是否正确设置，或重新执行 `anygen auth login --api-key sk-xxx` 命令，认证成功后重新执行原命令。

### 依赖技能未安装

执行 smart_draw 操作时报错提示 `anygen-workflow-generate` 技能不存在。处理方式：根据当前 Agent 平台执行 `anygen skill install --platform <平台标识> -y` 完成安装，安装完成后检查网络连接和配置后重试图表生成请求。

### 网络连接超时

调用 `www.anygen.io` 服务端时出现连接超时或不可达。处理方式：检查本机网络连接与代理配置，确认能正常访问外网。受限网络环境下配置 HTTP 代理后检查网络连接和配置后重试。

### smart_draw 操作参数缺失

发起 smart_draw 请求时未提供足够的图表描述信息，导致服务端无法识别目标图表类型。处理方式：补充图表类型、参与方、关系描述等关键信息后重新发起请求。描述越具体，生成质量越高。

### 图表生成结果与预期不符

生成的图表与用户描述存在偏差。处理方式：在原描述基础上补充约束条件（如节点数量、布局方向），或明确指定图表类型后重新生成。复杂图表建议拆分为多个简单子图分别生成。

## 常见问题

### Q1：AnyGen CLI 支持哪些操作系统？

AnyGen CLI 支持 Windows、macOS、Linux 三大主流操作系统。安装方式参考 AnyGen 官方文档对应平台的说明。CLI 运行依赖 Node.js 运行时环境，安装前请确认 Node.js 版本满足要求。

### Q2：API Key 如何获取？

API Key 通过 AnyGen 官方平台注册账号后获取。登录后在账户设置或 API 管理页面创建 API Key，复制后通过 `anygen auth login --api-key` 命令或 `ANYGEN_API_KEY` 环境变量配置。注意保管 API Key，避免泄露。

### Q3：免费版支持哪些图表类型？

免费版支持流程图、系统架构图、时序图等基础图表类型。通过 smart_draw 操作由智能引擎自动识别图表类型并渲染。对于复杂的图表需求（如多层级嵌套、大量节点），建议简化描述或拆分生成。

### Q4：生成的图表可以编辑吗？

生成的图表以图片形式输出，不能直接在 AnyGen CLI 中编辑。若需调整，修改原始文本描述后重新执行 smart_draw 操作重新生成。建议保留原始描述文本便于后续迭代。

| 命令执行失败 | 运行环境不满足要求或权限不足 | 确认运行环境符合依赖说明中的要求；检查命令权限设置 |

## 已知限制

- 需要联网访问 `www.anygen.io` 服务端，纯离线环境无法使用
- 依赖 AnyGen CLI 与 `anygen-workflow-generate` 技能，两者缺一不可
- 图表生成质量取决于文本描述的清晰度，模糊描述会导致生成结果偏差
- 单次生成的图表节点数量存在上限，复杂图表需拆分生成
- 生成的图表为图片输出，不支持生成后直接编辑
- 服务端渲染耗时随图表复杂度增加
