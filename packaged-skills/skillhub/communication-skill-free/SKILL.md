# Comm LITE - 沟通辅助基础版

通过3步流程(GATHER-CRAFT-REFINE)帮助撰写得体回应消息,采用"承认-过渡-引导"单一结构组织回应内容,适用于日常文本沟通场景.

## 核心能力

### 3步基础工作流: GATHER → CRAFT → REFINE
**第1步 GATHER(收集)**: 提取对话背景信息
- 识别对方身份(客户/同事/上级/下级)
- 提取对方核心诉求或情绪信号
- 明确本次回应目标(解释/说服/安抚/拒绝)

**第2步 CRAFT(起草)**: 按"承认-过渡-引导"结构撰写回应
- **承认(Acknowledge)**: 复述对方关键点,表明已收到信息
- **过渡(Bridge)**: 用"同时/不过/经查实"等连接词转向自己的立场
- **引导(Guide)**: 给出明确下一步行动或具体建议

**第3步 REFINE(润色)**: 自检回应质量
- 检查语气是否与对方关系匹配(平级/上下级/客户)
- 删除可能引起误解的模糊表达
- 确认是否给出了可执行的下一步

> **升级提示**: 深度倾听(4层)、沟通心理学5原则、3种回应结构、语气校准矩阵、复杂场景处理等高级功能仅在 communication-skill 付费版中提供.
**输出**: 返回步基础工作流: GATHER → CRAFT → REFINE的处理结果,包含执行状态码、结果数据和执行日志.

## 使用流程

### 1. GATHER - 收集对话背景
1. 通读对方发来的原始消息
2. 用一句话总结对方画像: `[客户/同事/上级]希望[诉求],情绪[平静/不满/焦急]`
3. 标记消息中的关键短语(如"不行就退款"、"尽快回复"),这些是对方底线信号

### 2. CRAFT - 按"承认-过渡-引导"起草
按以下模板组织回应三段式:
```
[承认] 您提到/我理解您说的...(复述对方关键点,不带评判)
[过渡] 同时/不过/经查实...(转向自己的立场或客观情况)
[引导] 建议我们/接下来我会...(明确下一步可执行行动)
```

### 3. REFINE - 自检并润色
逐项检查清单:
4. [ ] 承认部分是否准确复述了对方诉求(无曲解、无添油加醋)
5. [ ] 过渡是否自然,无生硬转折或推卸感
6. [ ] 引导部分是否给出可执行的具体行动(非空话)
7. [ ] 整体语气是否与双方关系匹配(客户用"您",熟络同事可放松)

**结果验证**: 任务完成后,查看输出确认状态。成功时返回摘要和数据;失败时根据错误信息排查,参考恢复章节获取修复步骤.
## 案例展示

### 案例1: 客户投诉物流延迟

**场景**: 电商卖家收到客户投诉,物流已超3天未更新

**客户原始消息**:
> 我的订单都3天了还没动,你们到底能不能发货?不行就退款!

**GATHER(收集)**:
- 对方身份: 客户
- 核心诉求: 要么发货要么退款
- 情绪信号: 不满、焦急
- 底线信号: "不行就退款"

**CRAFT(起草)** - 按"承认-过渡-引导"结构:
```
[承认] 您好,我理解您的订单已3天未更新物流,这确实让人着急.
[过渡] 经查实,您的包裹因双十一爆仓滞留在中转站,我们已加急处理.
[引导] 我已为您备注优先发货,预计24小时内更新物流;若您不愿等待,我可立即为您办理全额退款,请您告知意向.
```

**REFINE(润色)**:
- 承认部分准确复述"3天未更新物流" ✓
- 过渡用"经查实"自然转向客观原因,无推卸感 ✓
- 引导给出两个可执行选项(等待/退款) ✓
- 语气平和专业,匹配商家-客户关系 ✓

**最终回应**:
> 您好,我理解您的订单已3天未更新物流,这确实让人着急。经查实,您的包裹因双十一爆仓滞留在中转站,我们已加急处理。我已为您备注优先发货,预计24小时内更新物流;若您不愿等待,我可立即为您办理全额退款,请您告知意向.
## 异常处理

| 错误场景 | 原因分析 | 处理方式 |
|:---:|:---:|:---:|
| 诉求误读 | GATHER阶段未识别对方真实意图 | 重新阅读原始消息,关注"不行就退款"等底线短语背后的核心诉求 |
| 语气不匹配 | 回应过于正式或过于随意 | 根据双方关系调整:客户用"您",平级同事可适当放松,上级保持敬语 |
| 转折生硬 | "承认"与"过渡"之间缺乏连接词 | 使用"同时/不过/经查实/基于此"等过渡词实现软转折 |
| 引导缺失 | 只解释原因未给下一步 | 补充具体可执行动作,如"我将在24小时内..." |
| 过度承诺 | 为安抚对方承诺无法兑现的内容 | REFINE阶段核查承诺可行性,只给出有把握的行动 |
| 共情不足 | 对方情绪激烈但回应过于理性 | 在"承认"部分增加共情表达(如"这确实让人失望"),复杂情绪场景请升级付费版 |

## 常见问题

### Q1: 何时该用"承认-过渡-引导"结构?
A: 适用于需要回应对方诉求的多数场景(投诉、请求、询问)。当对方纯粹分享信息无需你回应行动时,可省略"引导"部分,仅用"承认+过渡"表达共情即可.
### Q2: 对方情绪激动时,这个结构是否够用?
A: 免费版仅提供基础结构。情绪激烈时建议在"承认"部分增加更多共情表达(如"这确实让人失望")。若涉及深层情绪疏导或高冲突场景,请升级付费版获取深度倾听(4层)与复杂场景处理能力.
### Q3: 回应看起来像模板,如何避免生硬感?
A: 在REFINE阶段重点润色:将"我理解您说的"替换为对方原话关键词(如"3天未更新物流"),让承认部分更具针对性,避免套话.
### Q4: 免费版和付费版的核心区别?
A: 免费版(LITE)提供3步基础流程+1种回应结构。付费版(communication-skill)额外提供:
- 5步完整工作流(GATHER-LISTEN-CONSIDER-CRAFT-REFINE)
- 深度倾听4层(事实/情绪/需求/价值观)
- 沟通心理学5原则
- 3种回应结构(承认-过渡-引导 / 共情-探索-共识 / 事实-影响-方案)
- 语气校准矩阵
- 复杂场景处理(高冲突谈判、跨文化沟通、危机公关等)

## 已知限制

- **结构单一**: 仅支持"承认-过渡-引导"1种回应结构(付费版提供3种)
- **无深度倾听**: 不分析对方话语背后的情绪/需求/价值观层次(付费版提供4层深度倾听)
- **无心理学原则**: 不应用互惠、承诺一致等沟通心理学原则(付费版提供5原则)
- **无语气校准**: 仅做基础语气匹配,无跨场景语气矩阵(付费版提供语气校准矩阵)
- **复杂场景受限**: 不支持高冲突谈判、危机公关、跨文化沟通等复杂场景。升级付费版获取深度倾听(4层)、沟通心理学原则、3种回应结构、复杂场景处理等功能
- **纯文本场景**: 仅处理文本沟通,不涉及语音/视频沟通中的语调与肢体语言
