# 标题钩子工厂 Title Hook Factory

分析爆款标题特征,生成高点击率、高转化的内容标题。内置10+种钩子公式,支持多标题变体生成与AB测试预测。

## 使用场景

| 场景 | 触发条件 | 说明 |
|:-----|:---------|:-----|
| 标题生成 | 内容创作完成后 | 生成3-5个爆款标题选项 |
| 标题优化 | 现有标题点击率低 | 分析优化建议 |
| 对标分析 | 输入竞品标题 | 分析爆款特征并生成差异化标题 |
| AB测试预测 | 多标题候选 | 预测各标题点击率,推荐最优 |

## 钩子公式库

### 电商类标题公式

| 公式 | 示例 | 适用场景 |
|:-----|:-----|:---------|
| 数字+结果 | "3天涨粉1000,我做了这5件事" | 教程/攻略类 |
| 痛点+方案 | "流量差?这个方法让你爆单" | 问题解决类 |
| 对比+反差 | "月入5000和5万的差距在哪" | 对比分析类 |
| 平台+年份 | "2026年电商平台最新运营技巧" | 时效性内容 |
| 疑问+悬念 | "为什么你的内容没人看?" | 引发好奇类 |

### 内容矩阵类标题公式

| 公式 | 示例 | 适用场景 |
|:-----|:-----|:---------|
| 热点+观点 | "AI代写火了,但90%的人不知道这个技巧" | 热点追踪类 |
| 个人经历 | "我用AI月入3万的真实经历" | 故事类 |
| 清单合集 | "10个免费AI工具合集" | 资源分享类 |
| 避坑指南 | "电商运营最常见的5个坑" | 经验分享类 |

### 历史故事类标题公式

| 公式 | 示例 | 适用场景 |
|:-----|:-----|:---------|
| 人物+悬念 | "李白最失意的那天,写下了千古名篇" | 历史人物故事 |
| 诗词+故事 | "一首静夜思背后,藏着一个游子的心酸" | 诗词故事 |
| 朝代+揭秘 | "盛唐最传奇的友谊:李白与杜甫的相遇" | 历史揭秘 |
| 反转+人物 | "被贬谪的诗人,却写出了最豁达的诗" | 人物反转 |
| 数字+历史 | "5首诗读懂李白的传奇一生" | 历史科普 |

## 工作流

1. **分析输入内容** - 提取核心主题、目标人群、价值点
2. **匹配标题公式** - 根据内容类型选择最合适的钩子公式
3. **AI生成标题** - 使用LLM生成3-5个标题变体
4. **标题评分** - 从吸引力/搜索量/点击率预估三个维度评分
5. **AB测试预测** - 对比多标题变体,预测各变体点击率表现
6. **输出推荐标题** - 按综合评分排序输出最优标题

## 输入格式

```json
{
  "content_topic": "电商自动回复教程",
  "target_audience": "电商卖家/副业人群",
  "content_type": "教程",
  "platform": "ecommerce",
  "reference_titles": ["自动回复设置教程", "如何用AI提高回复效率"]
}
```

字段说明:
- `content_topic`: 内容主题
- `target_audience`: 目标人群
- `content_type`: 内容类型(教程/攻略/故事/清单等)
- `platform`: 目标平台(ecommerce电商/xiaohongshu小红书/weibo微博等)
- `reference_titles`: 对标参考标题(可选,用于竞品分析)

## 异常处理

| 异常 | 处理策略 | 降级方案 |
|:-----|:---------|:---------|
| AI模型不可用 | 使用模板库生成标题 | 返回预定义模板标题 |
| 输入内容过短 | 提示补充更多信息 | 基于关键词生成通用标题 |
| 平台规则变更 | 更新标题合规检查规则 | 跳过合规检查并标记风险 |

## 依赖说明

### 运行环境
- **Agent平台**: Claude Code / Cursor / Codex / Gemini CLI / Windsurf 等支持SKILL.md的任意Agent
- **操作系统**: Windows / macOS / Linux
- **运行时**: 需要Agent支持exec(命令行执行)能力

### 第三方依赖
| 依赖项 | 类型 | 是否必需 | 获取方式 |
|:-------|:-----|:---------|:---------|
| LLM API | API | 必需 | 任意LLM服务商,由Agent内置LLM提供 |

### API Key 配置
- **LLM_API_KEY**: 必需(通常由Agent内置) - 标题生成/评分/AB测试预测
- 配置方式: 在Agent的环境变量中设置

### 纯Markdown使用说明
AI模型不可用时降级为模板库生成标题。

只需将SKILL.md文件放入Agent的skills目录即可直接使用。
如果Skill中包含exec工具调用,需要Agent支持命令行执行能力。

### 可用性分类
- **分类**: MD+EXEC
- **说明**: 纯Markdown,但需要exec能力(命令行执行),用于文件读写和命令调用
