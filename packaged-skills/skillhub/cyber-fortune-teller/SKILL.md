# 赛博命理师 Cyber Fortune Teller

赛博命理师,基于命理排盘引擎支持8种命理体系,AI深度解读生成专业命理报告,支持报告交付与运势摘要生成。

## 使用场景

- 电商平台客户下单命理服务后自动生成命理报告
- 内容矩阵发布运势、命理相关内容引流
- 私域社群定期运势推送
- 多种命理体系交叉验证提升专业感

## 工作流

### 1. 接收算命请求

- 输入: birth_date(出生日期), birth_time(出生时辰), gender(性别), type(命理体系类型)
- 验证: 日期格式合法,时辰在12地支范围内,性别为男/女
- 异常: 参数缺失→提示补全;格式错误→给出正确格式示例

| type值 | 命理体系 | 排盘内容 | 定价建议 |
|:-------|:---------|:---------|:---------|
| bazi | 八字排盘 | 四柱+大运流年 | 9.9-29.9元 |
| ziwei | 紫微斗数 | 12宫+108星+四化 | 49.9-99元 |
| tarot | 塔罗占卜 | 抽牌+解读 | 9.9-19.9元 |
| meihua | 梅花易数 | 起卦+断卦 | 19.9-49.9元 |
| liuyao | 六爻 | 排卦+分析 | 19.9-49.9元 |
| qimen | 奇门遁甲 | 天地人神四盘 | 49.9-99元 |
| astrology | 西方占星 | 星盘+流运 | 9.9-29.9元 |
| almanac | 黄历查询 | 黄历宜忌 | 免费(引流) |

### 2. 调用命理排盘API生成基础排盘

- 八字: 排四柱+五行+十神+纳音,排大运流年
- 紫微: 排12宫+108星+四化,排运限
- 塔罗: 抽牌+解读
- 梅花: 起卦+断卦(体用生克+互卦+变卦)
- 六爻: 排卦+分析
- 奇门: 排盘(天盘地盘人盘神盘)
- 占星: 排星盘+流运
- 异常: 排盘引擎不可用→提示"命理引擎维护中,请稍后再试"(不降级,避免计算错误)

### 3. AI深度解读

- 将排盘数据+用户信息组合为LLM提示词
- 提示词模板按type选择,每种type有专属角色名+结构化输出格式+免责声明
- 输出: 800-1500字命理报告(含事业/感情/财运/健康四维度)
- 异常: LLM超时→返回原始排盘数据+简短解读(200字)

| type | 角色名 | 输出结构 |
|:-----|:-------|:---------|
| bazi | 玄微子 | 日主强弱+五行平衡+十神格局+大运走势+流年细断+四维度 |
| tarot | 月曦 | 逐牌解读+牌阵整体+综合建议(1核心+3行动+2注意+时间线) |
| meihua | 观梅老人 | 体用生克+互卦过程+变卦结果+应期+建议 |
| astrology | 星言 | 星盘核心+四维度运势+重点提醒 |
| ziwei | 天机子 | 命宫格局+四化飞星+关键宫位+大限流年 |

### 4. 敏感词审核

- 检测报告内容中的敏感词
- 替换封建迷信敏感词为心理学术语(例:"命中注定"→"性格倾向")
- 添加免责声明: "本报告仅供娱乐参考,不构成任何决策建议"
- 异常: 审核不通过→重新生成解读段落

### 5. 报告交付

1. 将命理报告保存为文件(PDF/文本),文件名格式: {order_id}_{type}_{timestamp}
2. 上传报告文件到文件存储服务指定目录
3. 生成安全分享链接(随机6位密码+3天有效期)
4. 将分享链接通过电商平台消息发送给买家
5. 事后验证发货结果(链接可达+消息已送达)
- 异常: 上传失败→记录待发送队列,下次重试; 发送失败→记录待发送队列; 验证失败→触发告警

### 6. 运势内容引流(可选)

1. 生成50字运势摘要: 从命理报告中提取核心运势关键词,生成精简摘要(含今日运势+幸运色/数字)
2. 发布运势摘要到社交平台(可配置)
3. 发布内容格式: "[星座/生肖]今日运势 | [50字摘要] | #运势 #今日运势"
4. 记录发布结果
- 异常: 发布不可用→跳过,不影响核心流程; 发布失败→记录日志,不重试

## 异常处理

| 异常场景 | 处理方式 | 错误码 |
|:---------|:---------|:-------|
| 参数缺失 | 提示补全所需参数 | MISSING_PARAMS |
| 排盘引擎不可用 | 提示维护中,不降级 | ENGINE_UNAVAILABLE |
| LLM超时 | 返回排盘数据+简短解读 | LLM_TIMEOUT |
| 敏感词审核不通过 | 重新生成解读段落 | SENSITIVE_CONTENT |
| 报告发送失败 | 记录待发送队列 | DELIVERY_FAILED |
| 社交发布不可用 | 跳过运势引流,不影响核心流程 | CONTENT_PUBLISH_SKIPPED |

## 依赖说明

### 运行环境
- **Agent平台**: Claude Code / Cursor / Codex / Gemini CLI / Windsurf 等支持SKILL.md的任意Agent
- **操作系统**: Windows / macOS / Linux
- **运行时**: 需要Agent支持exec(命令行执行)能力

### 第三方依赖
| 依赖项 | 类型 | 是否必需 | 获取方式 |
|:-------|:-----|:---------|:---------|
| LLM API | API | 必需 | 任意LLM服务商,由Agent内置LLM提供 |
| 命理排盘引擎 | API | 必需 | 八字/紫微/塔罗/梅花/六爻/奇门/占星/黄历 |
| 文件存储 | 文件系统 | 可选 | exec工具保存报告文件 |

### API Key 配置
- **LLM_API_KEY**: 必需(通常由Agent内置) - AI深度解读命理报告
- **FORTUNE_API_KEY**: 可选 - 命理排盘引擎(可使用开源排盘库)
- 配置方式: 在Agent的环境变量中设置

### 纯Markdown使用说明
命理排盘可使用开源排盘库(如lunar-python)替代API。LLM超时时返回原始排盘数据+简短解读。

只需将SKILL.md文件放入Agent的skills目录即可直接使用。
如果Skill中包含exec工具调用,需要Agent支持命令行执行能力。

### 可用性分类
- **分类**: MD+EXEC
- **说明**: 纯Markdown,但需要exec能力(命令行执行),用于文件读写和命令调用

## 示例

用户在电商平台下单"八字排盘"服务:

1. 接收订单→提取出生信息(1990-01-15,子时,男)
2. 调用命理排盘API排盘→己巳年丁丑月庚辰日丙子时(立春前用1989年柱)
3. 排大运→戊寅/己卯/庚辰/辛巳/壬午
4. AI解读→生成1200字四维度报告(事业/感情/财运/健康)
5. 敏感词审核→替换3处敏感词+添加免责声明
6. 保存报告文件→上传文件存储→生成安全分享链接(6位密码+3天有效期)
7. 通过电商平台消息发送分享链接给买家→验证发货结果
8. 生成50字运势摘要→社交平台发布引流

## 合规提醒

- 定位为"心理娱乐"服务,避免封建迷信宣传

- 禁止使用"必应""化解""破灾"等违规词
