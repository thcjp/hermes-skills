## 适用场景

### 场景一：品牌升级全案

企业进行品牌升级，需要完整的品牌体系构建。

```text
用户指令：我们公司要从传统形象升级为科技感品牌，请提供完整的品牌升级方案。
# ...
创意总监输出：
【品牌升级策略】
升级方向：传统可靠 → 科技创新
核心挑战：保持客户信任感的同时注入科技活力
# ...
【品牌定位更新】
- 品牌使命：用科技赋能传统行业
- 品牌个性：创新、专业、前瞻
- 品牌承诺：技术驱动，价值可信
# ...
【视觉系统升级】
- 标志：保留核心识别，简化线条，增加动态效果
- 色彩：从沉稳蓝调升级为科技蓝+活力橙
- 字体：从衬线体切换为无衬线几何体
- 图形：引入数据可视化风格辅助图形
# ...
【语调系统更新】
- 声音特征：自信、前瞻、平易近人
- 语言风格：专业但不晦涩，创新但不浮夸
- 禁用词：过时、传统、老牌（强调新锐感）
# ...
【迁移计划】
阶段1：核心视觉系统更新（1个月）
阶段2：数字渠道全面迁移（2个月）
阶段3：线下物料逐步替换（3个月）
```

## 常见问题

### Q1：品牌体系构建需要多长时间？

完整品牌体系构建通常需要4-8周，取决于品牌复杂度与项目范围。

### Q2：营销战役方案可以直接执行吗？

专业版输出完整的战役框架与执行计划，但具体创意制作需要创意团队执行。

### Q3：竞品分析的情报来源是什么？

竞品分析基于公开可获取的营销案例与市场信息，非内部机密数据。

### Q4：团队协作功能如何使用？

通过创意审批工作流管理方案版本，支持多角色协作与反馈循环。

### Q5：与免费版的评估模型兼容吗？

兼容。专业版在免费版4维度基础上扩展为8维度，免费版的评估结果可作为专业版的输入。
