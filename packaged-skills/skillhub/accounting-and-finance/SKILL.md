## 核心能力

### 一、估值建模 (14项)

#### 收益法估值
- **DCF折现现金流模型**: 构建自由现金流预测模型,通过WACC折现计算企业内在价值
  输入: 历史财务数据(3-5年) + 行业增长率假设 → 输出: 企业内在价值 + 敏感性分析表
- **DDM股利贴现模型**: 适用于成熟分红型企业的估值,包括Gordon增长模型和两阶段DDM
  输入: 历史股利数据 + 股利增长率 → 输出: 每股内在价值
- **FCFE/FCFF模型**: 股权自由现金流vs企业自由现金流,分别计算股权价值和企业价值
  输入: 财务报表数据 → 输出: 股权价值 + 企业价值 + 每股价值
- **剩余收益模型(RI)**: 基于账面价值和超额收益的估值方法,适用于重资产行业
  输入: 净资产 + ROE + 资本成本 → 输出: 内在价值

#### 市场法估值
- **可比公司分析(CCA)**: 选取同行业可比上市公司,基于P/E、P/B、EV/EBITDA等乘数估值
  输入: 目标公司财务数据 + 3-5家可比公司数据 → 输出: 估值区间 + 乘数对比表
- **可比交易分析(PTA)**: 参考同行业并购交易价格,基于交易乘数估值
  输入: 近3年同行业并购交易数据 → 输出: 交易乘数统计 + 估值区间
- **P/E乘数估值**: 市盈率估值,适用于盈利稳定的成熟企业
  输入: 净利润 + 行业平均P/E → 输出: 估值
- **EV/EBITDA乘数估值**: 企业价值/EBITDA,跨资本结构可比,适用于跨国比较
  输入: EBITDA + 行业平均EV/EBITDA → 输出: 企业价值
- **P/B乘数估值**: 市净率估值,适用于银行、保险等金融行业
  输入: 净资产 + 行业平均P/B → 输出: 估值

#### 特殊行业估值
- **银行估值模型**: 基于净利息收入(NIM)、拨备覆盖率、资本充足率等银行特有指标
  输入: 银行财务报表 → 输出: 银行内在价值 + 关键指标分析
- **保险公司估值**: 基于内含价值(EV)和新业务价值(VNB)的估值方法
  输入: 寿险精算数据 + 财报数据 → 输出: 内含价值 + 每股价值
- **房地产估值**: 基于NAV(净资产价值)和RNAV(重估净资产)的估值方法
  输入: 物业估值 + 负债数据 → 输出: NAV + RNAV
- **科技公司估值**: 结合用户增长、ARPU、LTV/CAC等互联网指标的估值方法
  输入: 用户数据 + 收入模型 → 输出: 科技公司估值 + 关键指标分析
- **分部估值(SOTP)**: 多业务线企业的分部加总估值法
  输入: 各业务线财务数据 → 输出: 分部估值 + 折价/溢价分析

### 二、财务分析 (26项)

#### 财务比率分析
- **偿债能力分析**: 流动比率、速动比率、现金比率、资产负债率、利息保障倍数
  输入: 资产负债表 → 输出: 偿债能力指标表 + 行业对比
- **营运能力分析**: 存货周转率、应收账款周转率、总资产周转率、固定资产周转率
  输入: 资产负债表 + 利润表 → 输出: 营运效率指标表
- **盈利能力分析**: 毛利率、营业利润率、净利率、ROE、ROA、ROIC
  输入: 利润表 + 资产负债表 → 输出: 盈利能力指标表 + 趋势分析
- **发展能力分析**: 营收增长率、净利润增长率、总资产增长率、股东权益增长率
  输入: 3年财务数据 → 输出: 成长性指标表 + CAGR计算

#### 综合分析框架
- **DuPont杜邦分析**: 将ROE分解为净利润率×总资产周转率×权益乘数,定位盈利驱动因素
  输入: 财务报表 → 输出: DuPont分解图 + 驱动因素分析
- **ROE-ROIC矩阵**: 区分经营绩效和财务杠杆对ROE的贡献
  输入: 财务报表 → 输出: ROIC + 利差分析(ROIC-WACC)
- **Z-Score破产预测**: Altman Z-Score模型,预测企业破产风险
  输入: 财务报表 → 输出: Z-Score值 + 风险等级(安全/灰色/危险)
- **盈利质量分析**: 经营现金流/净利润比率,识别盈余管理行为
  输入: 现金流量表 + 利润表 → 输出: 盈利质量评分 + 异常项识别

#### 现金流分析
- **现金流结构分析**: 经营/投资/筹资现金流占比,判断企业现金流健康度
  输入: 现金流量表 → 输出: 现金流结构图 + 健康度评分
- **自由现金流分析**: FCFF和FCFE计算,评估企业可分配现金
  输入: 财务报表 → 输出: FCFF + FCFE + 趋势分析
- **现金流量充足率**: 评估企业经营活动产生现金满足资本支出和偿债需求的能力
  输入: 现金流量表 → 输出: 现金流量充足率 + 预警信号

#### 资产与资本结构
- **资产负债结构分析**: 资产构成、负债构成、资本结构分析
  输入: 资产负债表 → 输出: 结构分析表 + 异常项提示
- **营运资本管理**: 营运资本周转分析,识别资金占用问题
  输入: 资产负债表 → 输出: 营运资本周期 + 优化建议
- **资本结构优化**: WACC计算、最优资本结构分析
  输入: 资本结构数据 → 输出: WACC + 最优债务比例建议

#### 报表处理
- **三大报表勾稽**: 验证资产负债表、利润表、现金流量表的勾稽关系
  输入: 三大报表 → 输出: 勾稽验证表 + 差异项提示
- **财务报表标准化**: 按统一格式重组非标准财报数据
  输入: 原始财报数据 → 输出: 标准化财报模板
- **非经常性损益调整**: 识别并调整一次性损益,计算调整后净利润
  输入: 利润表 + 附注 → 输出: 调整后净利润 + 剔除项明细
- **会计政策变更影响**: 分析会计政策变更对财务数据的影响
  输入: 变更前/后财报 → 输出: 影响差异表
- **合并报表分析**: 母子公司合并报表分析,识别少数股东权益和商誉影响
  输入: 合并报表 + 母公司报表 → 输出: 合并分析表

#### 行业专项分析
- **银行业专项分析**: 净息差、不良率、拨备覆盖率、资本充足率
  输入: 银行财报 → 输出: 银行业专项指标表
- **零售业专项分析**: 同店增长、坪效、库存周转、客单价
  输入: 零售企业财报 → 输出: 零售业专项指标表
- **制造业专项分析**: 产能利用率、单位成本、良品率、存货周转
  输入: 制造业财报 → 输出: 制造业专项指标表
- **ST/退市风险预警**: 基于财务指标评估A股ST和退市风险
  输入: 上市公司财报 → 输出: ST风险评分 + 退市风险等级

### 三、风险评估 (18项)

#### 财务舞弊识别
- **Beneish M-Score模型**: 8因子模型识别盈余操纵可能性
  输入: 财务报表 → 输出: M-Score值 + 操纵概率(阈值-1.78)
- **盈余管理识别**: 通过应计利润、非经常性损益识别盈余管理行为
  输入: 财务报表 → 输出: 盈余管理指标 + 可疑项列表
- **财务造假红旗预警**: 识别关联交易异常、存货异常、收入异常等造假信号
  输入: 财务报表 + 附注 → 输出: 红旗预警清单 + 风险等级
- **收入确认操纵识别**: 分析收入确认时点、渠道压货、循环交易等收入操纵手法
  输入: 利润表 + 收入附注 → 输出: 收入质量评分 + 可疑模式

#### 风险度量
- **流动性风险评估**: 短期偿债压力、现金流覆盖率、流动性缓冲分析
  输入: 资产负债表 + 现金流 → 输出: 流动性风险评分 + 预警信号
- **信用风险评估**: 基于财务指标的信用评级映射,参考KMV模型
  输入: 财务报表 → 输出: 信用评级 + 违约距离(DD)
- **市场风险敏感性分析**: 利率、汇率、商品价格变动对利润的影响测算
  输入: 财务数据 + 敏感性参数 → 输出: 敏感性分析表 + VaR估算
- **经营杠杆分析**: 固定成本vs变动成本结构,评估经营风险
  输入: 成本结构数据 → 输出: 经营杠杆系数 + 盈亏平衡点

#### 深度风险分析
- **关联交易分析**: 识别关联方交易占比、定价公允性、资金占用
  输入: 关联交易披露 → 输出: 关联交易风险评分 + 异常项
- **表外负债识别**: 经营租赁、担保、未决诉讼等表外项目分析
  输入: 财报附注 → 输出: 表外负债清单 + 调整后资产负债率
- **商誉减值风险**: 商誉占净资产比例、减值测试假设合理性分析
  输入: 合并报表 → 输出: 商誉减值风险评分
- **大额资产减值识别**: 存货跌价、应收坏账、固定资产减值分析
  输入: 资产明细 + 减值政策 → 输出: 减值风险评分 + 关注项

#### 审计与合规
- **审计报告分析**: 审计意见类型、关键审计事项、强调事项段分析
  输入: 审计报告 → 输出: 审计意见解读 + 关键事项清单
- **内控有效性评估**: 基于内控自评报告和审计内控报告评估内控质量
  输入: 内控报告 → 输出: 内控缺陷清单 + 有效性评分
- **会计估计合理性**: 折旧年限、坏账计提比例、保修准备金等会计估计合理性分析
  输入: 会计政策 + 行业对比 → 输出: 估计偏差评分 + 调整建议
- **信息披露质量评估**: 财报附注完整性、重大事项披露及时性评估
  输入: 财报 + 附注 → 输出: 信息披露质量评分 + 缺失项清单
- **持续经营能力评估**: 综合财务指标评估企业持续经营能力(参考审计准则)
  输入: 财务报表 → 输出: 持续经营评估 + 强调事项

2. 行业对标数据(3-5家可比公司)
3. 财报附注(关联交易、会计政策、重大事项)

### 2. 数据预处理与验证
4. 运行「三大报表勾稽」验证数据完整性

### 3. 选择分析框架
根据分析目标选择对应技能组合:

8. **分析场景** → 比率分析 + DuPont + 现金流分析 + 行业专项分析

### 4. 执行分析并交叉验证
10. 对同一目标使用多种方法交叉验证(如DCF+CCA+PTA)
11. 对比行业基准和历史趋势
12. 识别异常值并进行敏感性分析

### 5. 生成分析报告
输出结构化报告,包含:分析结论 + 关键指标表 + 风险提示 + 假设条件 + 数据来源

## 案例展示

### 案例1: IPO估值定价 (DCF + 可比公司)
**场景**: 某科技公司计划IPO,需要确定发行价格区间

**分析步骤**:
```
1. 数据准备: 获取公司近3年财报 + 5家可比上市公司数据
2. DCF估值:
   - 预测未来5年自由现金流(基于营收增长率15%假设)
   - 计算WACC = 无风险利率2.5% + Beta 1.2 × 风险溢价6% = 9.7%
   - 终值用Gordon模型,永续增长率3%
   → 企业价值 = 52.3亿元
# ...
3. 可比公司分析:
   | 可比公司 | P/E | EV/EBITDA | P/S |
|:------|------:|:------|:------|
   | 公司A    | 25.3 | 15.2 | 4.8 |
   | 公司B    | 28.7 | 17.5 | 5.2 |
   | 公司C    | 22.1 | 13.8 | 4.3 |
   | 平均值   | 25.4 | 15.5 | 4.8 |
   → 基于公司净利润2.1亿: P/E估值 = 2.1 × 25.4 = 53.3亿
   → 基于公司EBITDA 3.5亿: EV/EBITDA估值 = 3.5 × 15.5 = 54.3亿
# ...
4. 估值结论:
   | 方法 | 估值(亿元) |
|---:|:---|
   | DCF  | 52.3 |
   | P/E  | 53.3 |
   | EV/EBITDA | 54.3 |
   | 估值区间 | 52-54亿 |
   → 建议发行价区间: 52-54元/股(基于1亿股本)
```

### 案例2: 财务舞弊识别 (Beneish M-Score + 红旗预警)
**场景**: 投资者怀疑某上市公司存在财务造假,需要验证

## 异常处理

| 错误场景 | 原因分析 | 处理方式 |
|----|----|----|
| 财报数据缺失 | 年报未披露或数据不全 | 标注缺失项,使用行业平均值估算,在报告中明确假设条件 |
| 三大报表不勾稽 | 数据录入错误或非标准格式 | 运行「报表勾稽验证」定位差异项,手动调整后重新分析 |
| 可比公司不足 | 细分赛道上市公司少 | 扩大至上下游行业,或使用海外可比公司并调整估值乘数 |
| 会计政策差异 | 不同企业采用不同会计准则(如CAS vs IFRS) | 运行「会计政策变更影响」调整至统一基准后比较 |
| 负值指标无法计算 | 净利润为负导致P/E无意义 | 改用EV/EBITDA或P/S乘数,或使用前瞻P/E(基于预测利润) |
| 行业基准不可得 | 新兴行业缺乏历史数据 | 参考海外市场数据,或使用自下而上方法构建行业基准 |
| 附注信息不完整 | 关联交易等附注披露不充分 | 标注信息缺失风险,降级关联交易分析的置信度 |
| 预测假设过于乐观 | 管理层指引偏离行业趋势 | 进行多情景分析(乐观/中性/悲观),标注假设敏感性 |

## 常见问题

### Q1: 如何处理不同会计准则下的财务数据?
A: 本工具支持CAS(中国企业会计准则)和IFRS(国际财务报告准则)数据。主要差异处理方式:
- 开发支出资本化: CAS允许有条件资本化,IFRS同样允许,但判断标准略有不同
- 投资性房地产: CAS允许成本模式或公允价值模式,IFRS优先公允价值模式
- 处理方式: 运行「会计政策变更影响」技能将数据调整至统一准则后分析

### Q2: DCF模型中如何确定WACC?
A: WACC = E/(D+E) × Ke + D/(D+E) × Kd × (1-T),其中:
- Ke(股权成本) = 无风险利率 + Beta × 市场风险溢价 (CAPM模型)
- Kd(债务成本) = 企业实际借款利率或行业平均
- 无风险利率: 使用10年期国债收益率(中国约2.5-3%,美国约4-4.5%)
- Beta: 从彭博/Wind获取,或用回归分析法计算
- 市场风险溢价: 中国一般用6-8%,美国一般用5-6%

### Q3: 如何判断可比公司的可比性?
A: 从以下维度评估可比性:
- 行业相同或相近(申万二级行业分类)
- 市值规模相近(±50%范围内)
- 业务模式相似(如都是To B或To C)
- 成长性相近(营收增长率±20%)
- 地域市场重叠
- 如可比性不足,在估值时调整折扣率或溢价

### Q4: Beneish M-Score的阈值如何解读?
A: M-Score = -1.78是经验阈值:
- M-Score > -1.78: 盈余操纵的可能性较高(约76%的操纵公司超过此值)
- M-Score < -1.78: 盈余操纵的可能性较低
- 但M-Score只是辅助工具,不能单独作为舞弊结论,需结合红旗预警和现金流分析交叉验证

### Q5: 如何处理负现金流企业的估值?
A: 负现金流企业的DCF估值需要特殊处理:
- 使用收入乘数(P/S)或用户价值乘数替代利润乘数
- 预测期延长至7-10年,等待企业达到盈亏平衡
- 使用概率加权情景分析(乐观/中性/悲观)
- 参考VC估值方法,加入成功率折扣

## 已知限制

- **数据依赖**: 分析质量取决于输入数据的完整性和准确性,垃圾进垃圾出
- **历史外推**: 基于历史数据预测未来,无法预判黑天鹅事件和行业拐点
- **估值主观性**: DCF中的增长率、折现率等假设具有主观性,不同分析师可能得出不同结论
- **行业局限**: 对新兴行业(如Web3、AI)的估值缺乏成熟方法论,参考价值有限
- **时效性**: 财报数据有滞后性(季度报告1-3个月,年报3-4个月),分析结论可能已过时
- **非财务因素**: 无法量化管理层能力、企业文化、政策风险等非财务因素,需结合定性分析
- **市场有效假说**: 估值模型假设市场最终会回归内在价值,但短期市场可能持续非理性
