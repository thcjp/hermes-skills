# Email Free

基于SMTP协议的轻量邮件发送Skill,覆盖"配置-校验-投递"基础发信链路。

## 核心能力

- **主流供应商支持**:内置Gmail(smtp.gmail.com:587)与Outlook(smtp.office365.com:587)两套SMTP预设,自动启用TLS加密。
- **纯文本正文**:通过`--body`参数传入纯文本内容,自动按UTF-8编码,避免乱码。
- **单附件投递**:支持`--attachment`参数附加单个文件,受服务商25MB大小限制。
- **测试邮件校验**:`--test`标志发送最小化测试邮件,用于验证凭据与连通性。
- **凭据配置文件**:通过`email_config.json`集中管理SMTP服务器、端口、用户名、密码。

## 环境准备

创建`email_config.json`:

```json
{
  "smtp_server": "smtp.gmail.com",
  "smtp_port": 587,
  "username": "your-email@gmail.com",
  "password": "your-app-password",
  "sender_name": "个人助手",
  "use_tls": true
}
```

Gmail用户需在账号开启2FA后,从"应用密码"页面生成16位应用专用密码填入`password`字段。

## 异常处理


### AUTH_001 应用专用密码失效

- **现象**:`SMTPAuthenticationError: 535 Username and Password not accepted`。
- **原因**:Gmail账号未启用2FA,或应用专用密码过期。
- **处理**:重新生成应用专用密码,确认2FA开启后更新配置文件。

### CONN_002 SMTP端口被拦截

- **现象**:`SMTPConnectError: Connection refused`。
- **原因**:587端口被防火墙拦截。
- **处理**:切换至465端口并设置`use_ssl=true`、`use_tls=false`。

### ATTACH_003 附件超过25MB

- **现象**:`SMTPDataError: 552 Message size exceeds fixed limit`。
- **原因**:附件体积超过服务商硬限制。
- **处理**:压缩附件或拆分投递;超大文件改用云存储链接放入正文。

### FORMAT_004 收件人邮箱格式非法

- **现象**:`ValueError: Invalid recipient format`。
- **原因**:`--to`参数邮箱字符串缺失域名或含非法字符。
- **处理**:校验邮箱格式后剔除空项与重复项再执行ping命令测试网络连通性,检查防火墙和代理设置连接后重新执行命令。

### ENCODING_005 主题包含非ASCII字符乱码

- **现象**:收件方主题显示为`???`或未解码的编码串。
- **原因**:未对Subject做RFC 2047编码。
- **处理**:使用`email.header.Header(subject, 'utf-8').encode()`构造主题。

## FAQ

### Q1: 是否支持HTML富文本邮件?

免费版仅支持纯文本正文。HTML富文本、CSS内联样式、邮件模板等能力由付费版提供。

### Q2: 能否同时发送多个附件?

免费版仅支持单附件投递。多附件并行投递能力由付费版提供。

### Q3: 是否支持CC/BCC抄送?

免费版仅支持`--to`主收件人。CC/BCC收件人编排能力由付费版提供。

### Q4: 如何切换到QQ邮箱或自定义SMTP?

免费版仅内置Gmail与Outlook预设。QQ邮箱、Yahoo及自定义SMTP服务器适配由付费版提供。

## 错误处理


| 错误场景 | 原因 | 处理方式 |
|---------|------|---------|
| LLM响应超时或无响应 | 网络延迟或模型负载过高 | 执行ping命令测试网络连通性,检查防火墙和代理设置连接，执行ping命令测试网络连通性,检查防火墙和代理设置连接后重新执行命令请求；确认Agent平台LLM服务正常 |
| 输入内容格式不正确 | 用户输入不符合skill预期格式 | 检查输入是否符合skill使用说明中的格式要求，参考示例章节 |
| 执行结果与预期不符 | 指令描述不够明确或上下文不足 | 提供更详细的指令描述，补充必要的上下文信息 |
| 命令执行失败 | 运行环境不满足要求或权限不足 | 确认运行环境符合依赖说明中的要求；检查命令权限设置 |

## 已知限制

- 仅支持纯文本正文,不支持HTML富文本与邮件模板。
- 单邮件仅支持单个附件,且受25MB服务商硬限制。
- 仅支持`--to`主收件人,不支持CC/BCC抄送。
- 仅内置Gmail与Outlook两套SMTP预设,不支持自定义服务器。
- 凭据仅支持配置文件方式注入,不支持环境变量与密钥管理服务。
