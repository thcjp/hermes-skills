# Chief Creative Officer Free

首席创意官基础AI助手。提供创意方向规划、视觉风格指导与营销创意概念生成，为个人创作者与小型团队提供创意策划支持。


## 输入格式

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| input | string | 是 | 首席创意官助手-免费版处理的输入数据或指令 |
| options | object | 否 | 附加配置选项,如模式选择、格式偏好等 |
| callback_url | string | 否 | 异步处理完成后的回调通知URL |

## 使用方式

通过自然语言向Agent描述创意需求，Agent将根据需求类型输出对应的创意方案或方向文档：

```
"为[品牌/产品]制定创意方向，目标受众是[受众描述]，调性偏[风格描述]"
"策划一期[主题]营销campaign，预算[规模]"
"为[品牌]定义视觉风格方向"
```

输出内容为结构化文本，可用于团队讨论或创意参考。

- 策划营销campaign，需要创意概念

### 创意策略制定
基于品牌定位与目标受众，输出创意方向规划。包含创意核心概念、调性关键词与创意 pillars 拆解。支持从品牌价值主张出发推导创意策略，确保创意方向与品牌定位一致。输出文档可用于内部讨论或创意参考。

**输入**: 用户提供创意策略制定所需的指令和必要参数。
**处理**: 解析创意策略制定的输入参数,执行核心处理逻辑,返回结构化结果和执行状态。### 视觉风格指导
为品牌或campaign定义视觉风格方向，包含色彩方向、字体搭配建议与图形语言指引。输出风格关键词与视觉方向描述，帮助设计团队理解创意方向。支持基本的风格一致性描述，对比设计方向与目标风格的关键差异点。

**输入**: 用户提供视觉风格指导所需的指令和必要参数。
**处理**: 解析视觉风格指导的输入参数,执行核心处理逻辑,返回结构化结果和执行状态。### 营销创意策划

### 案例：新消费品牌创意方向制定

某新消费品牌主打面向年轻人的健康饮品，需要制定创意方向。输入品牌定位（健康、轻负担、日常陪伴）与目标受众（18至25岁城市年轻人）后，获取创意方向规划。核心创意概念定为"轻松喝好每一天"，拆解为三个创意pillars：成分透明、场景陪伴、情绪共鸣。视觉风格方向建议采用明亮色系搭配手绘插画，调性关键词为"轻松、真实、有温度"。品牌方依据该文档指导设计团队执行，首套品牌视觉上线后社交媒体讨论度高于行业新品平均水平。

## 异常处理

### 创意brief信息不完整

输入的品牌定位或目标受众信息过于简略时，生成的创意方向可能缺乏针对性。处理方式：在输入时补充品牌价值主张与产品核心卖点，提供足够上下文。若已生成泛化方案，可针对具体维度分别补充约束后重新生成。

### 品牌调性与创意方向冲突

当品牌调性描述与生成结果存在矛盾时，需重新校准。处理方式：在输入时明确调性边界与禁忌方向（如"避免使用网络流行语"），生成结果将规避冲突方向。若冲突已产生，手动调整调性关键词后重新生成。

### 创意方案超出预算约束

生成的方案涉及的执行复杂度可能超出实际预算。处理方式：在输入时明确预算规模与资源限制（如"仅支持2条短视频"），生成结果将适配预算范围。若已生成超出预算的方案，可手动缩减内容方向。

### 视觉风格描述缺乏具体性

视觉风格指导输出的描述可能过于抽象。处理方式：在输入时补充具体的参考方向（如品牌名或风格关键词），生成结果将增加可操作的视觉规范描述。建议搭配视觉情绪板使用。

### 多个创意方向难以取舍

当生成多个创意方向且各有优势时，决策困难。处理方式：在输入时补充评估优先级（如"优先考虑差异化"），生成结果将附带优先级标注。建议结合团队讨论辅助决策。

## 常见问题

### 创意策略如何与品牌定位对齐？

创意策略从品牌价值主张出发推导，确保创意 pillar 能追溯到品牌核心定位。输入时提供品牌定位描述，生成结果将体现定位与创意方向的关联。建议策略输出后与品牌团队进行对齐确认。

### 能否针对特定行业生成创意方向？

支持针对快消、科技、美妆、餐饮等主流行业生成差异化创意方向。在输入时指定行业名称，创意策略与视觉风格将适配该行业的调性特征。若行业较为小众，建议补充行业背景信息。

### 创意brief模板可以自定义吗？

创意brief模板支持按项目类型调整。在输入时指定项目类型（品牌形象、产品发布、节日营销），生成结果将包含对应字段的填写指引。建议团队建立标准brief模板按项目类型调用。

### 视觉风格指导包含哪些内容？

视觉风格指导包含色彩方向、字体搭配建议、图形语言指引与风格关键词。输出为文字描述，帮助设计团队理解创意方向。建议搭配视觉情绪板与参考案例使用，将文字描述转化为视觉参考。

## 已知限制

- 无法直接生成设计稿或视觉作品，输出为文字描述与方向指导
- 创意效果受执行团队能力影响，不保证落地效果
- 不适用于需要用户调研数据支撑的创意决策场景
- 创意方案为概念输出，无法分析设计稿的视觉细节
- 生成内容基于文本模型，无法评估色彩与构图等视觉层面质量
