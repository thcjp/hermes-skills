## 错误处理


| 错误场景 | 原因 | 处理方式 |
|---------|------|---------|
| "删除后笔记本上也没了" | 同步按设计工作，非错误 | 解释同步机制；若需保留，先复制到非同步目录再删除 |
| "iCloud满了但手机有空间" | 手机存储与iCloud存储是两个配额 | 引导到iCloud管理存储页面清理，而非删除手机本地文件 |
| 照片在多个服务重复 | 多个服务同时备份相机卷 | 选一个主备份服务，关闭其余的自动备份 |
| 同步长时间未完成 | 网络慢或文件过大 | 
| 共享链接被滥用 | 链接过期未撤销 | 季度审查共享链接，撤销旧的；新链接设过期时间 |
| 2FA未启用 | 账户安全基础缺失 | 立即在所有云账户启用2FA（短信/认证器/硬件密钥） |
| 备份"假成功" | 备份进程静默失败 | 每月检查备份状态，不要假设它在工作 |
| 跨平台同步冲突 | 混合设备用非原生服务 | 选Dropbox等跨平台一致的服务；或按主设备选原生服务 |
| 付费多个服务 | 重复付费 | 选一个主服务，取消其余；按设备组合决定主服务 |
| 敏感文档明文存储 | 安全习惯缺失 | 先加密再上传，或不用云；启用2FA |
