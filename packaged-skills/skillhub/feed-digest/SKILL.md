## 输入格式

| 参数名 | 类型 | 必填 | 说明 |
|---:|---:|---:|---:|
| content | string | 否 | feed-digest处理的内容输入 |,  |
| mode | string | 否 | 处理模式, 可选: json/text/markdown,  |
