# 赛博命理师 Cyber Fortune Teller

赛博命理师,基于命理排盘引擎支持8种命理体系,AI深度解读生成专业命理报告,支持报告交付与运势摘要生成.

|---|---|---|

## 核心能力

1. **8种命理体系全覆盖**:八字排盘(四柱+大运流年)/紫微斗数(12宫+108星+四化)/塔罗占卜(抽牌+解读)/梅花易数(起卦+断卦)/六爻(排卦+分析)/奇门遁甲(天地人神四盘)/西方占星(星盘+流运)/黄历查询(宜忌),定价9.9-99元全价位覆盖
2. **AI深度解读生成报告**:排盘数据+用户信息组合为LLM提示词,每种type有专属角色名+结构化输出格式+免责声明,输出800-1500字命理报告(含事业/感情/财运/健康四维度)
3. **敏感词审核与合规**:检测报告内容中的封建迷信敏感词,自动替换为心理学术语(例:"命中注定"→"性格倾向"),添加免责声明,定位为"心理娱乐"服务确保合规
4. **安全报告交付**:报告保存为文件→上传文件存储→生成安全分享链接(随机6位密码+3天有效期)→通过电商平台消息发送→事后验证发货结果,全链路闭环
5. **运势内容引流**:生成50字运势摘要(含今日运势+幸运色/数字),发布到社交平台引流,格式"[星座/生肖]今日运势 | [50字摘要] | #运势 #今日运势"

## 使用流程

### Step 1: 接收算命请求
- 解析输入:birth_date(出生日期)、birth_time(出生时辰)、gender(性别)、type(命理体系类型)
- 验证:日期格式合法,时辰在12地支范围内,性别为男/女
- 校验失败处理:参数缺失→提示补全;格式错误→给出正确格式示例

| type值 | 命理体系 | 排盘内容 | 定价建议 |
|---:|---:|---:|---:|
| bazi | 八字排盘 | 四柱+大运流年 | 9.9-29.9元 |
| ziwei | 紫微斗数 | 12宫+108星+四化 | 49.9-99元 |
| tarot | 塔罗占卜 | 抽牌+解读 | 9.9-19.9元 |
| meihua | 梅花易数 | 起卦+断卦 | 19.9-49.9元 |
| liuyao | 六爻 | 排卦+分析 | 19.9-49.9元 |
| qimen | 奇门遁甲 | 天地人神四盘 | 49.9-99元 |
| astrology | 西方占星 | 星盘+流运 | 9.9-29.9元 |
| almanac | 黄历查询 | 黄历宜忌 | 免费(引流) |

### Step 2: 调用命理排盘API生成基础排盘
- 八字:排四柱+五行+十神+纳音,排大运流年
- 紫微:排12宫+108星+四化,排运限
- 塔罗:抽牌+解读
- 梅花:起卦+断卦(体用生克+互卦+变卦)
- 六爻:排卦+分析
- 奇门:排盘(天盘地盘人盘神盘)
- 占星:排星盘+流运
- 异常:排盘引擎不可用→提示"命理引擎维护中,请稍后再试"(不降级,避免计算错误)

### Step 3: AI深度解读
- 将排盘数据+用户信息组合为LLM提示词
- 提示词模板按type选择,每种type有专属角色名+结构化输出格式+免责声明
- 输出:800-1500字命理报告(含事业/感情/财运/健康四维度)
- 异常:LLM超时→返回原始排盘数据+简短解读(200字)

| type | 角色名 | 输出结构 |
|:---:|:---:|:---:|
| bazi | 玄微子 | 日主强弱+五行平衡+十神格局+大运走势+流年细断+四维度 |
| tarot | 月曦 | 逐牌解读+牌阵整体+综合建议(1核心+3行动+2注意+时间线) |
| meihua | 观梅老人 | 体用生克+互卦过程+变卦结果+应期+建议 |
| astrology | 星言 | 星盘核心+四维度运势+重点提醒 |
| ziwei | 天机子 | 命宫格局+四化飞星+关键宫位+大限流年 |

### Step 4: 敏感词审核
- 检测报告内容中的敏感词
- 替换封建迷信敏感词为心理学术语(例:"命中注定"→"性格倾向")
- 添加免责声明:"本报告仅供娱乐参考,不构成任何决策建议"
- 异常:审核不通过→重新生成解读段落

### Step 5: 报告交付
1. 将命理报告保存为文件(PDF/文本),文件名格式:{order_id}_{type}_{timestamp}
2. 上传报告文件到文件存储服务指定目录
3. 生成安全分享链接(随机6位密码+3天有效期)
4. 将分享链接通过电商平台消息发送给买家
5. 事后验证发货结果(链接可达+消息已送达)
- 异常:上传失败→记录待发送队列,下次重试;发送失败→记录待发送队列;验证失败→触发告警

### Step 6: 运势内容引流(可选)
1. 生成50字运势摘要:从命理报告中提取核心运势关键词,生成精简摘要(含今日运势+幸运色/数字)
2. 发布运势摘要到社交平台(可配置)
3. 发布内容格式:"[星座/生肖]今日运势 | [50字摘要] | #运势 #今日运势"
4. 记录发布结果
- 异常:发布不可用→跳过,不影响核心流程;发布失败→记录日志,不重试

## 异常处理

| 异常场景 | 原因 | 处理方式 | 错误码 |
|:------|------:|:------|:------|
| 参数缺失 | birth_date/birth_time/gender/type未提供 | 提示补全所需参数 | MISSING_PARAMS |
| 排盘引擎不可用 | 命理API服务宕机或Key无效 | 提示维护中,不降级(避免计算错误) | ENGINE_UNAVAILABLE |
| LLM超时 | LLM响应超过超时阈值 | 返回排盘数据+简短解读(200字) | LLM_TIMEOUT |
| 敏感词审核不通过 | 报告内容命中违禁词 | 重新生成解读段落 | SENSITIVE_CONTENT |
| 报告发送失败 | 网盘上传失败或消息发送失败 | 记录待发送队列,稍后 | DELIVERY_FAILED |
| 社交发布不可用 | 社交平台API不可用 | 跳过运势引流,不影响核心流程 | CONTENT_PUBLISH_SKIPPED |

## 合规提醒

- 定位为"心理娱乐"服务,避免封建迷信宣传
- 所有报告必须附带免责声明+AI生成声明
- 禁止使用"必应""化解""破灾"等违规词
- 电商商品描述使用"性格分析""趋势参考"等替代话术
- AI生成声明:"本报告由AI辅助生成,仅供参考娱乐"

只需将SKILL.md文件放入Agent的skills目录即可直接使用.
如果Skill中包含exec工具调用,需要Agent支持命令行执行能力.
### 可用性分类
- **分类**: MD+EXEC
- **说明**: 纯Markdown,但需要exec能力(命令行执行),用于文件读写和命令调用

## 常见问题

### Q1: 命理排盘引擎不可用时怎么办?
A: 命理排盘引擎不可用时返回ENGINE_UNAVAILABLE错误,提示"命理引擎维护中,请稍后再试",不降级以避免计算错误。建议:1)使用开源排盘库lunar-python替代API(本地运行无需API Key);2)检查FORTUNE_API_KEY配置是否正确;3)等待排盘引擎服务恢复后重试.
### Q2: 报告如何确保合规不违规?
A: 本Skill有三重合规保障:1)敏感词审核自动替换封建迷信敏感词为心理学术语(如"命中注定"→"性格倾向");2)所有报告强制添加免责声明"本报告由AI辅助生成,仅供娱乐参考,不构成任何决策建议";3)定位为"心理娱乐"服务,禁止使用"必应""化解""破灾"等违规词,电商商品描述使用"性格分析""趋势参考"等替代话术.
### Q3: 报告交付的安全分享链接有效期多久?
A: 安全分享链接有效期为3天,配有随机6位密码保护。交付流程:1)报告保存为PDF/文本文件;2)上传文件存储;3)生成安全分享链接(6位密码+3天有效期);4)通过电商平台消息发送链接+密码给买家;5)事后验证链接可达+消息已送达。发送失败会记录待发送队列支持重试.
### Q4: 8种命理体系分别适合什么定价?
A: 定价建议按深度分级:引流款9.9-19.9元(塔罗/黄历免费/占星/八字基础);标准款19.9-49.9元(梅花/六爻/八字详细);深度款49.9-99元(紫微/奇门/多体系交叉验证)。建议用黄历免费引流→塔罗/八字9.9元转化→紫微/奇门99元深度变现的漏斗模型.

## 已知限制

1. **定位心理娱乐非封建迷信**:所有报告必须附带免责声明,禁止"必应""化解""破灾"等违规词,定位为"心理娱乐"服务,不可作为真实命运预测依据
2. **排盘引擎不可用时不降级**:命理排盘需要精确计算,引擎不可用时返回ENGINE_UNAVAILABLE而非降级,避免错误计算导致误导(可使用开源lunar-python库替代)
3. **LLM超时仅返回200字简短解读**:LLM超时时降级返回原始排盘数据+200字简短解读,无法生成完整800-1500字四维度报告
4. **报告有效期3天**:安全分享链接有效期仅3天,超期后买家需联系重新生成,建议买家及时下载保存
5. **不支持实时咨询对话**:本Skill为报告生成模式,不支持一对一实时命理咨询对话,需用户先提供出生信息再生成报告
